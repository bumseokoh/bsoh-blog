概要

写真: 韓国観光公社 観光写真ギャラリー (全南広州統合特別市 和順郡) · 公共ヌリ

無等山（ムドゥンサン）は、光州広域市と全羅南道和順郡にまたがる山で、湖南地方を代表する国立公園です。山頂付近は軍事通信施設があるため立ち入りが制限されていますが、一般の登山客なら西石台（ソソクデ）や立石台（イプソクデ）といった柱状節理のエリアまで登ることができます。

園内の登山道の総延長（各区間の合計）は13.55km。森林庁のデータによれば、登録されている区間の最高難易度は「初級」となっています。ですが、実際に『100名山』の実測GPXデータで見ると、山頂付近まで登る代表的なコースは片道5〜9km、累積標高差も450〜780mにまで及びます。データ上の「初級」という言葉をそのまま鵜呑みにすると、結構な体力勝負になるかもしれません（笑）。とはいえ、日帰り登山として十分に完走できるコースです。

山の地形と風景

写真: 韓国観光公社 観光写真ギャラリー (全南光州統合特別市 和順郡) · 公共ヌリ

無等山（ムドゥンサン）は、なだらかな稜線の上にダイナミックな岩峰や柱状節理がそびえる、実にユニークな地形をしています。山頂付近の西石台（ソスッデ）と立石台（イプソクデ）は、まるで屏風のように並ぶ柱状節理が見事で、学術的・景観的な価値から国立地質公園にも指定されているほどです。

主要な峰々としては、万延峰（628m）、ヌエ峰（1084m）、ラクダ峰（938m）、安養山（876m）、万延山（687m）、元暁峰（583m）、中頭嶺（617m、地点により613mとも）などが挙げられます。今回のコースの実測GPX値では、最高地点は1120mでした。

本来の最高峰は天王峰（約1187m）なのですが、通信施設があるため立ち入りが制限されています。そのため、登山客の多くは西石台付近を実質的な山頂として楽しむことになりますね。また、ガレ場（石が積み重なった場所）である地工のルダーや、奎峰庵周辺の岩場も、この山ならではの特徴的な地形です。

登山の様子

写真：韓国観光公社 観光写真ギャラリー（全南光州統合特別市 和順郡） · 公共누리

登り始めは、ヒノキや広葉樹の森に囲まれた土の道や緩やかな稜線が続くので、そこまで足取りも重くありません。ところが、高度が上がるにつれてガレ場や岩場が混じり始め、一気に体感的な難易度が跳ね上がってきます。中峰・長仏才（チャンブルジェ）付近では、ススキの群生と開けた稜線の眺めから季節感を存分に味わえますし、西石台（ソセクデ）・立石台（イプセクデ）付近では、目の前に柱状節理の岩峰が広がる圧巻の景色に出会えます。秋ならススキと紅葉、冬なら凍てついた稜線や美しい霧氷が、この山の魅力的な見どころとしてよく名前が挙がります。

コース 1 — 元暁寺（ウォニョサ）入口バス停 → 元暁分所 山岳救助隊

📏 6.75km ⛰️ 734m 獲得標高 ⏱️ 2:55 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 元暁寺駐車場 → 無等山休憩所 → 元暁寺入口バス停 → 元暁寺駐車場トイレ → 松道山荘 → 無等山行こう（ムデサンガジャ） → 分かれ道 → 元暁寺分岐トイレ → 元暁分所案内所 → 元暁分所山岳救助隊 → 元暁分所水飲み場 → 元暁地区食堂街 → ヒノキの森 → コマクジェ → 急勾配区間

コースの特徴・おすすめ: 最初はヒノキの森や食堂街を通る、緩やかな土の道が続きます。ところが、後半のコマクジェを越えると一変して急な登り坂が待ち受けています。ヒノキの森での穏やかな散策気分を味わいつつ、最後はしっかり本格的な登山も楽しみたい！という欲張りな方にぴったりのコースです。

特徴・おすすめの人: 奎峰庵や地空のガレ場といった岩場の地形を通り抜けながら、静かな山寺とダイナミックな岩の世界を同時に楽しめるコースです。距離もそこそこありますし、標高差もしっかりあるので、なかなかの体力勝負になります。「静かなお寺の雰囲気も味わいたいけれど、本格的な岩場歩きも楽しみたい！」という欲張りな方（あるいは私のように、ついつい無茶なコースを選んでしまう方）にはぴったりです。

難易度・路面・注意点

写真: 韓国観光公社 観光写真ギャラリー (全南広州統合特別市 華旬郡) · 公共ヌリ

行政区画に基づく登録区間（登山コース名ではなく、管内の総延長による区分）は以下の通りです。

区間（行政区画基準）総延長難易度路面
無渡里（ムドリ）区間5.87km初級粘土質
無渡里谷峠〜無渡里切谷区間5.4km初級粘土質
土交里（トギョリ）区間2.28km初級粘土質

林野庁のデータ上では難易度は「初級」となっていますが、これはあくまで低地部分の距離を基準にしたもの。実際に山頂まで登るGPXの実測コースとなると、累積標高差と距離の関係で、難易度は一気に最高ランクの ★★★★★ に跳ね上がります。 路面は粘土質が多いので、雨が降るとかなり滑りやすくなります。コースの要所でも、「落石注意（コース4・5）」、「急斜面の落石リスク（コース5）」、「凍結区間・無等山の氷岩（コース4）」、「落雷多発地域（コース6）」といった、実際の警告表示が確認されています。急斜面や登山道の欠損箇所（コース6）もあるため、雨上がりなどは路面のコンディションを必ず確認するようにしましょう。 コース内の水場は1カ所（元暁分所 飲水場）確認できました。避難小屋や、個別の危険エリア・展望スポットのPOIは登録されていないため、現地の看板や案内所の情報を参考に進むのが良さそうです。

距離と所要時間

写真: 韓国観光公社 観光写真ギャラリー (全南広州統合特別市 和順郡) · 公共누리

登り累計239分、下り累計170分（全区間を合算した数値であり、単一コースの所要時間ではありません）というデータを基準に計算すると、平均的な登りのペースは1kmあたり約18分になります。山頂へ向かうコースの多くは片道5〜9km、標高差が450〜780mほどあるため、このペースを当てはめると片道はおよそ2時間〜3時間20分ほどが一般的な目安です。

往復となると6〜8時間前後かかることもあるため、早めの出発をおすすめします。とはいえ、山小屋などの避難所はなく、総延長も13.55kmとそれほど長くはないため、日帰り登山でも十分に歩ききれる範囲ですよ。

アクセス（公共交通機関・駐車場）

写真：韓国観光公社 観光写真ギャラリー（全南光州統合特別市 和順郡） · 公共ヌリ

光州の市内中心部では、地下鉄よりも市バスを利用するのが一般的です。 元暁寺（ウォニョサ）方面へ向かうなら「無等山国立公園（元暁寺）」停留所、証心寺（チュンシムサ）方面へは「証心寺地区」行きのバスを利用するのが分かりやすいルートです。また、須万里（スマンリ）方面へ向かう場合は「野菊村入口 須万里3区」停留所で下車してください。 バスの路線や運行時間はその時々によって変わることもあるので、出発前に必ずKakaoMapやNaver Mapでリアルタイムのルートを確認しておくのが賢明です。 なお、公的なデータ上では駐車場は「0件」となっています（道園村や須万里に地元の方向けの小さなスペースはありますが、かなり限られています）。そのため、週末や観光シーズンには公共交通機関を利用するのがおすすめですよ。

シーズン・天気・装備

写真: 韓国観光公社 観光写真ギャラリー (全南光州統合特別市 和順郡) · 公共누리

春はツツジ、秋はススキや紅葉がとにかく見事で、登山には春と秋がベストシーズンと言えます。 夏場は落雷が発生しやすい場所もあるので、午後に雷の予報が出ている時は無理をせず、早めに下山するのが安全ですね。 冬は凍結箇所や凍った岩場も実際に目にするので、アイゼンやゲイターなどの防寒・滑り止め装備は欠かせません。 また、地面が粘土質なので雨上がりはかなり滑りやすくなります。しっかりとした登山靴とトレッキングポールを用意しておくと安心です。それから、季節を問わず水分補給用の水と行動食は、常に余裕を持って準備しておきましょう。

旅のヒント

写真: 韓国観光公社 観光写真ギャラリー (全南光州統合特別市 和順郡) · 公共누리

無等山（Mudeungsan）の代表的なスポットといえば、西石台（Seosokdae）、立石台（Ipseokdae）、中峰（Jungbong）などが挙げられます。国立公園内なので、入山許可や入場料などは特にかかりませんが、山頂付近の一部は通信施設があるため立ち入りが制限されている場所もあります。

トイレは、元暁寺の駐車場や分岐点、道園探訪支援センター、水万里野菊村の駐車場、堂山樹の周辺など、いくつかの場所に用意されています。飲み水に関しては、元暁分所の水飲み場（1箇所）がありますが、全体的に水場はそれほど多くありません。念のため、自分でも十分な量の水を持参することをおすすめします。

また、韓国の登山道はGoogleマップだと情報が正確でないことが多いため、現地では「KakaoMap」か「Naver Map」のアプリを使うのが賢明です。

道中では、落石注意や凍結区間、落雷の恐れがある場所などの案内看板が実際に設置されています。必ず表示に従って行動してくださいね。なお、避難所はないため、「日没前には必ず下山する」という計画を立てておくことが極めて重要です。
