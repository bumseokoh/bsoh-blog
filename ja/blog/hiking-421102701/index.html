概要

写真: 韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三川洞）· 公共누리

三岳山（サマクサン）は、江原特別自治道の春川市西面・新東面一帯に位置し、北漢江と衣岩湖を抱えるようにそびえる山です。山頂部は龍華峰（654m）を中心に、いくつもの峰が連なって稜線をつくっています。京春線の江村駅や衣岩ダムから近く、首都圏から公共交通機関でアクセスしやすい山として広く知られていて、全体の難易度はデータ上「中級」に分類されています。短めのコースを選べば日帰りの往復登山でも十分にこなせますし、長い稜線コースをつないでいけば、まる一日をかけての本格的な山行にもなります。

山容と地形

写真: 韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三千洞）· 公共ヌリ

三岳山は花崗岩の岩峰や奇岩絶壁があちこちに顔を出す山で、なだらかな土の山というより、岩の稜線と急斜面が入り混じった地形です。林野庁の登山道空間情報に登録されている主な峰としては、三岳左峰（383m）、登仙峰（632m）、青雲峰（536m）、龍華峰（654m）、そして山頂部（767m）と表記された地点などが確認できます。ただ、一部の峰は表記の仕方によって少しずつ違う標高値で重複登録されていて（たとえば龍華峰が654m・675mといった具合に併記されている）、実際に登るときは現地の道標の表記を基準にしたほうが安全です。江村の方から眺めると、稜線がノコギリの歯のように尖って連なっている姿がひときわ目立ち、義岩湖や北漢江の側には急な斜面が続いていて、視界がぱっと開ける地点も多めです。

登山の雰囲気

写真: 韓国観光公社 観光写真ギャラリー (江原特別自治道 春川市 三川洞) · 公共ヌリ

登り口のあたりは、たいてい広葉樹と松が入り混じった森の道で、なだらかに始まります。ただ、尾根に近づくにつれて花崗岩の岩稜やロープ・階段の区間がちょくちょく現れて、体感の難易度はぐっと上がっていきます。山頂部や尾根のあちこちからは、北漢江や衣岩湖、江村一帯の集落を見下ろす眺めが開けます。春は新緑とツツジ、秋は紅葉が尾根沿いに続いて、季節感がはっきりしているのが持ち味です。夏は谷筋の湿気と蒸し暑さ、冬は岩場の凍結が体感の難易度を大きく左右する、そんな山だと言えます。

経由: 방동2리入口（방동保健診療所）→ 駐車場 → 방동2리 → 방동2리村会館 → 村会館駐車場 → 休憩所 → 分岐点 → 墓地 → 登山道の様子 → 上水源保護区域および休養林の案内 → 春川スカイキャンプ場 → 양지말 → 방동리終点

特徴・おすすめの人: 序盤は集落の道と休憩所を抜けながら、じわじわと高度を上げていく区間です。上水源保護区域を通るぶん、森が比較的よく保たれている印象でした。長距離・長時間のコースなので、体力にも時間にも余裕のある、山歩き経験者向きといえます。

特徴・おすすめの人: 四つの峰を次々と越えていく稜線縦走のコースで、道標に「滑落注意」「転落注意」の表示がはっきり出ているあたり、このあたり一帯で一番気を引き締めたい区間と言えそうです。いくつもの峰からの眺めを一度にまとめて味わいたい、山慣れした方向き。正直、初心者にはあまりおすすめできません。

難易度・路面・注意点

写真：韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三川洞）・公共ヌリ

山林庁の登山道空間情報を見るかぎり、三岳山エリアで登録されている区間の最高難易度は「中級」で、路面の材質については特に登録がありませんでした。エリア内の空間情報の集計上、危険地域・展望ポイント・水飲み場・避難所・登山口/下山口の指定地点はいずれも0か所となっていて、現地の案内板を当てにするより、あらかじめ地図で確認しておくほうが安心です。とはいえ、個別コースのGPXの標識を見ていくと、危険地域（コース5）、展望点（コース2・コース5）、滑落注意・転落注意の表示（コース6）などが実際に出てきますので、こうした区間では標識をよく見ておきたいところです。岩稜・岩場の区間は雨だと滑りやすく、冬場は凍結もしやすいので、こういう場所ではチェーンスパイクやアイゼンを使うのがおすすめです。

距離と所要時間

写真: 韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三千洞）· 公共ヌリ

林野庁の登山道空間情報によると、この地域の登山道の総延長は23.92km（複数区間の合計）で、上りは累計634分、下りは444分と記録されています。これを距離で割ると、上りの平均ペースはおよそ1kmあたり27分。ただしこれは地域内の全区間をすべて歩いた場合の累計ペースであって、単一コースの所要時間ではない点に注意が必要です。100名山のGPX実測をもとにすると、山頂を含むコースは片道11〜34km、片道でおよそ4時間〜10時間ほど。往復でもたいていは日帰りが可能ですが、いちばん長いコース（駐車場→小さなロウソク峰、34.27km）や縦走型のコース（ホンジョク終点→モンドク山山頂）となると、早朝の出発と十分な余裕時間の確保が欠かせません。避難小屋は特に確認できないので、日没前の下山を基準に日程を組むのがおすすめです。

行き方（公共交通・駐車）

写真: 韓国観光公社 観光写真ギャラリー (江原特別自治道 春川市 三川洞) · 公共ヌリ

首都圏からなら、京春線のITX-青春か、あるいは無窮花号で江村駅まで行って、そこから江村の市内バスか徒歩で登山口方面へ向かう、というのが一般的なアクセス方法として知られています。登山口の近くにある実際のバス停としては、江村入口、鳳徳寺入口の二つが確認できるので、この二つを基準にすれば、だいたいの徒歩でのアクセス距離を見当づけられると思います。方洞里・西上里・サリ峠といった外れの集落にある登山口は、公共交通の便がかなり少ないので、現実的には自家用車でしょうか。ただ、管内のデータ上、正式な駐車場は0か所という集計になっているので、路肩に停めるか、集落近くの空き地を頼るしかないかもしれません。実際のルートは、出発前に必ずカカオマップ(KakaoMap)かネイバー地図(Naver Map)で、公共交通の時刻表と徒歩区間をもう一度確かめておくのがおすすめです。

シーズン・天気・装備

写真: 韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三川洞）· 公共ヌリ

登山のベストシーズンは、新緑が芽吹く春（4～5月）と、紅葉が色づく秋（9月末～11月初め）ですね。夏場は渓谷区間の湿気と蒸し暑さ、それに梅雨時の落石やスリップに気をつけないといけません。冬は岩稜区間が凍結するので、アイゼンとスパッツはほぼ必須と考えておいたほうがいいでしょう。基本の装備としては、登山靴、着替え、たっぷりの飲み水（園内に水飲み場は事実上ないようなので、最低でも1～1.5L以上は持っていくのがおすすめです）、ヘッドランプ、手袋、トレッキングポールあたりを用意しておくと安心です。

外国人旅行者向けのヒント

写真: 韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三川洞）· 公共ヌリ

稜線の主な峰は、Yonghwabong（龍華峰、654m）、Deungseonbong（登仙峰、632m）、Cheongunbong（青雲峰、536m）、Samakjwabong（三岳左峰、383m）とローマ字表記できます。特別な入山許可や入場料は確認できませんでしたが、国立公園ではなく地方が管理する山なので、季節ごとの入山規制（コース2の「入山統制案内」の標識を参照）があるかどうかは、現地で改めて確認しておくと安心です。トイレと水飲み場は、西上2里の終点や、ホンジョク終点の案内所周辺など、一部の登山口・下山口でしか見当たらず、稜線区間には水の補給施設がありません。出発前に十分な水を用意しておきましょう。Googleマップは韓国国内の登山道情報が正確でないことが多いので、カカオマップやNAVERマップのアプリをあらかじめ入れておき、リアルタイムのルートや公共交通の情報を確認するのがおすすめです。稜線の一部には危険地域・落下注意・スリップ注意の標識があるので、標識のハングルを事前に翻訳アプリなどで確認しておき、単独より同行者との山行を計画したほうが安全です。

周辺のほかの山

  • 花崗岩の岩稜がそびえる、ソウルの岩山・道峰山（トボンサン） — 直線距離で約58km
  • 都心のすぐそば、花崗岩の稜線が見せる眺め・ソウル北漢山（プカンサン） — 直線距離で約65km
  • 鋸の歯のような八峰稜線と連珠台の岩峰・ソウル冠岳山（クァナクサン） — 直線距離で約78km
