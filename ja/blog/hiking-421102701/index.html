概要

사진: 한국관광공사 관광사진 갤러리 (강원특별자치도 춘천시 삼천동) · 공공누리

春川（チュンチョン）の삼악산(Samaksan)は、北漢江と義岩湖（의암호）を抱え込むようにそびえ立つ山です。場所は江原特別自治道の春川市、西面と新東面あたりに位置しています。

標高654mの龍華峰（용화봉）を中心に、いくつもの峰が美しい稜線を形作っています。京春線の江村（강촌）駅や義岩ダムからもほど近く、首都圏から公共交通機関を使ってアクセスしやすい山として広く知られています。

難易度はデータ上では「中級」といったところでしょうか。短いコースを選べばサクッと往復できますが、稜線を繋ぐ長距離コースを選ぶと、丸一日がかりの本格的な山歩いを楽しむことになります。

山の地形と景色

写真：韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三千洞） ・ 公共ヌリ

三岳山（サマクサン）は、至る所に花崗岩の岩峰や奇岩が顔を出す山です。なだらかな土の山というよりは、ゴツゴツとした岩の稜線と急斜面が入り混じったような地形をしています。

林野庁の登山道情報に登録されている主な峰としては、三岳左峰（383m）、登仙峰（632m）、青雲峰（536m）、龍華峰（654m）、そして山頂部（767m）などが挙げられます。ただ、情報の登録方法によって高度が微妙に異なっていることもあり（例えば龍華峰が654mだったり675mだったりと）、実際の山歩きでは、現地の案内板を基準にするのが一番確実で安全ですね。

江村（カンチョン）方面から眺めると、まるで鋸の歯のように切り立った稜線がとても印象的です。一方で、義岩湖（ウィアムホ）や北漢江側には急な斜面が続いていて、視界がパッと開ける絶景ポイントもたくさんあります。

登山の雰囲気

写真：韓国観光公社 写真ギャラリー（江原特別自治道 春川市 三川洞） · 公共ヌリ

登り始めは、広葉樹と松が混ざり合う緩やかな林道から始まります。しかし、稜線に近づくにつれて花崗岩の岩場やロープ、階段などが頻繁に現れるため、だんだんと体感的な難易度が上がっていきます。

山頂や稜線のあちこちからは、北漢江（プックァンガン）や医岩湖（ウィアムホ）、そして江村（カンチョン）周辺の街並みを一望できる素晴らしい眺望が広がります。春には新緑とツツジ、秋には紅葉が稜線を彩り、四季の移ろいをはっきりと感じられるのが魅力です。

夏場は谷間の湿度と暑さが、冬場は岩場の凍結が、それぞれ体感的な難易度を大きく左右する……そんな、季節によって表情が変わる山といえるでしょう。

コース1 — 方洞2里入口（方洞保健診療所）→ 頂上

📏 17.67km ⛰️ 807m 獲得標高 ⏱️ 5:46 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由地: 方洞2里入口（方洞保健診療所）→ 駐車場 → 方洞2里 → 方洞2里集会所 → 集会所駐車場 → 休憩所 → 分かれ道 → 墓地 → 登山道の様子 → 水源保護区および休息森林地案内 → 春川スカイキャンプ場 → ヤンジマル → 方洞里終点

アクセス: 方洞2里は春川の郊外にある集落です。公共交通機関については、春川市内のバスが方洞里方面へ走っていますが、本数がかなり少なめ。事前にリアルタイムの運行状況をチェックしておくのが賢明です。車で行く場合は、方洞保健診療所のあたりに駐車スペースがありますが、いわゆる「大きな駐車場」があるわけではないので注意してくださいね。

特徴・おすすめの人: 序盤は集落の道や休憩所を通りながら、少しずつ高度を上げていく区間です。水源保護区を通るルートなので、森も比較的きれいな状態で保たれています。ただ、かなりの長距離・長時間コースになるため、体力と時間に余裕のある経験者向けの山行になりそうです。

コース 2 — ソサン2里終点 → 山頂

📏 12.64km ⛰️ 729m 獲得標高 ⏱️ 4:23 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由地: ソサン2里終点 → 休憩所 → トイレ → 駐車場 → 分かれ道 → 低炭素グリーン成長の森 → 林道の始まり → 登山道の様子 → 渓谷上流 → 展望スポット → 枯れ木 → 採種林環境改善事業区間 → 開発制限区域の標識石 → サバンダム（砂防ダム） → 入山規制案内

登り口へのアクセス: ソサン2里の終点にはトイレと駐車場が完備されているようで、他の登り口に比べると出発時の利便性はかなり高いです。ただ、公共交通機関は村バスの路線に頼ることになるので、事前にしっかり調べておく必要がありますね。

特徴・おすすめの人: 渓谷の上流に沿って登っていき、展望スポットを通過しながら、枯れ木や採種林の整備区間など、自然と森林管理が共存する様子を眺められるコースです。序盤に設備が整っているので、「まずはトイレと駐車場を確保してから、落ち着いて歩き始めたい」という方には、比較的スムーズなスタートが切れるおすすめのルートです。

コース3 — 駐車場 → 小촛대봉

📏 34.27km ⛰️ 817m 獲得標高 ⏱️ 9:56 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由地: 駐車場 → 徳頭園1里2班 → 分かれ道 → 山岳気象観測設備 → 石派嶺（ソッパリョン）峠の頂上 → 休憩所 → 国土地点番号 → 展望ポイント → 道しるべ → 自然景観 → 墓地 → 小촛대봉 → カラマツの群落 → 登山道の様子 → モミの木群落

アクセス: 徳頭園里方面の駐車場から出発します。この区間は山林庁の登山道データにある「徳頭園里区間（13.3km、難易度：初級〜中級）」と重なるエリア。道自体は比較的よく整備されているので、アプローチはスムーズですよ。

特徴・おすすめの人: 全コースの中で最も距離がある34.27km級のロングコースです。石派嶺の峠や、カラマツ・モミが織りなす美しい林を通り抜けるので、森の豊かな生態系を存分に味わえます。丸一日時間を確保できるベテランの方や、あるいは無理せず区間を分けて歩きたいという方にもぴったりです。

コース4 — サリジェ入口 → ゲグァンサン山頂

📏 12.14km ⛰️ 760m 獲得標高 ⏱️ 4:18 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由地: サリジェ入口 → サリジェ村 → サリジェ終点 → 分岐点 → 国土地点番号 → 道標 → イノシシ出現注意 → ミズバクダル林 → コナラ群落 → ゲグァンサン山頂 → 休憩所 → モリペンション

アクセス: サリジェ入口は集落の道路から入る形になります。近隣にモリペンションなどの宿泊施設もあるので、車でのアクセスは比較的スムーズです。公共交通機関はあまり便利ではないので、自家用車で行くのが現実的な選択肢ですね。

特徴・おすすめ: ミズバクダルやコナラの美しい広葉樹林の中を歩くのがこのコースの醍醐味です。イノシシの目撃情報もあるような場所ですので、単独行よりは誰かと一緒に行くのが安心かもしれません。緑豊かな森の道が多いので、夏の暑い時期でも木陰で涼しく歩きたい方にはぴったりです。

コース5 — チャンジョルゴン入口 → 山頂

📏 12.41km ⛰️ 793m 獲得標高 ⏱️ 4:25 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由地: パンドン1里 → 保護樹 → 春川博士村 → チャンジョルゴン入口 → 危険区域 → 登山道入口 → 登山道の様子 → 分かれ道 → 墓地 → カシノナガキ群落 → 展望スポット → 林野庁無線中継施設 → 国家地点番号 → 登山道の様子

アクセス: 「春川博士村」の愛称で親しまれるパンドン1里を抜け、チャンジョルゴン入口へと進むルートです。保護樹のある村の入り口付近は歩きやすいですが、公共交通機関でのアクセスは限られているため、事前の確認が必要です。

特徴・おすすめの対象: 標識に「危険区域」の表示がある区間もあり、決して油断はできません。カシノナガキの群落や絶好の展望スポットを楽しみながら進めますが、安全標識のある地点では細心の注意が必要です。林野庁の無線中継施設などを通るため、整備された林道のような側面も持ち合わせています。緩やかな序盤に対し、中盤以降は程よい緊張感のある山歩きが楽しめるコース。変化を求める方にぴったりです。

コース 6 — 紅積終点 → 夢徳山頂

📏 25.9km ⛰️ 798m 獲得標高 ⏱️ 7:48 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 紅積終点 → 分岐点 → 国土地点番号 → 案内所 → 道標 → 夢徳山頂 → 加徳山頂 → 休憩所 → 北培山頂 → 展望ポイント → 滑り注意 → 滑落注意 → 保納山頂 → 駐車場 → トイレ

ルート概要: 紅積終点からスタートし、夢徳山・加徳山・北培山・保納山の4つの峰を繋いで歩く縦走コースです。終点には案内所があるので、出発時にコース情報を確認できるのがありがたいですね。なお、ゴール側の駐車場やトイレなどの設備は比較的整っています。

特徴・おすすめの方: 4つの山頂を次々と越えていく、本格的な稜線縦走コースです。道中の標識には「滑り注意」や「滑落注意」の表示もしっかりあり、このコースの中でも特に気を引き締めなければならない区間といえます。次々と変わる絶景を一度に満喫したいという熟練の登山愛好家にはたまらないコースですが、初心者の方には少しばかりハードすぎるかもしれません。

難易度・路面状況・注意点

사진: 한국관광공사 관광사진 갤러리 (강원특별자치도 춘천시 삼천동) · 공공누리

山林庁の登山道データによると、三岳山（삼악산）内の登録区間における最高難易度は「中」とのこと。なお、路面の材質についての記載はありません。 また、現地の空間情報データ上では、危険地帯や展望ポイント、水飲み場、避難所などはすべて「0件」と集計されています。看板を過信せず、事前に地図でしっかりルートを確認しておく必要がありそうです。

とはいえ、実際のコースのGPXデータを確認してみると、危険箇所（コース5）や展望スポット（コース2・5）、滑落注意の標識（コース6）などが実際に点在しています。こうした区間では、現地の案内表示を注意深く確認しながら進むのが良さそうです。 岩が露出している箇所は、雨天時は滑りやすく、冬場は凍結もしやすいため、チェーンアイゼンなどの準備をしておくことをおすすめします。

距離と所要時間

写真：韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三川洞） ・ 公共ヌリ

林野庁の登山道データによると、エリア内の登山道の総延長は23.92km（複数区間の合算）となっており、累積値として登り634分・下り444分と記録されています。これを距離で割ると、平均的な登りのペースは約27分/kmとなりますが……。これ、勘違いしやすいポイントなのですが、あくまで全区間を歩いた場合の合算値であって、単一のコースにかかる時間ではない点には注意が必要です。

「百名山」のGPX実測データを参考にすると、山頂付近へのコースは片道11〜34km、時間は片道で約4〜10時間ほどかかる計算になります。往復しても基本的には日帰りで行ける範囲ですが、最も長いコース（駐車場から小さな燭台峰へ向かう34.27kmのルート）や、縦走型のコース（洪積終点から夢徳山の山頂まで）などは、早朝の出発とたっぷりの余裕を持ったスケジュール管理が欠かせません。

山小屋などの避難所は特に確認できていないため、日没前には確実に下山を完了できるよう、余裕を持って計画を立てるのが賢明です。

アクセス（公共交通機関・駐車場）

写真：韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三千洞） ・ 公共누리

首都圏から向かうなら、軽春線（キョングルソン）のITX-青春や無窮花号で江村（カンチョン）駅まで行き、そこから地元のバスや徒歩で登山口方面へ向かうのが一般的なルートです。登山口の近くには「江村入口」や「奉徳寺入口」といったバス停があるので、これらを一つの目安としておくと、歩く距離のイメージが湧きやすいでしょう。

ただ、訪東里（パンドンリ）や西上里（ソサンリ）、サリ財（サリジェ）といった少し離れた集落方面へ向かうとなると、公共交通機関は本数がかなり限られているため、正直なところ車での移動が現実的です。ただ、あいにく現地のデータ上では正式な駐車場が0件となっているため、路肩や集落近くの空き地などを利用することになるかもしれません。

実際に現地へ向かう際は、念のためカカオマップやNAVERマップで最新の時刻表や徒歩ルートをしっかり再確認してから出発するようにしてくださいね。

シーズン・天気・装備

写真：韓国観光公社 観光写真ギャラリー（江原特別自治道 春川市 三川洞） ・ 公共ヌリ

山歩きを楽しむなら、新緑が美しい春（4〜5月）か、紅葉の鮮やかな秋（9月末〜11月初旬）がベストです。夏場は谷間の湿気と暑さ、それに梅雨時期の落石や滑りやすさに注意が必要。冬は岩場が凍結することもあるので、アイゼンやスパッツの準備は欠かせません。

基本装備としては、登山靴、着替え、そして十分な水分を準備しましょう。現地には水飲み場がほとんどないので、最低でも1〜1.5L以上は持参することをおすすめします。あわせて、ヘッドライト、手袋、トレッキングポールなども揃えておくと安心ですよ。

写真: 韓国観光公社 観光写真ギャラリー (江原特別自治道 春川市 三川洞) · 公共누리

主な山頂は、龍華峰(Yonghwabong, 654m)、登仙峰(Deungseonbong, 632m)、青雲峰(Cheongunbong, 536m)、三岳座峰(Samakjwabong, 383m)といった名前になります。 入山許可や入場料は特にかからないようですが、ここは国立公園ではなく地方自治体が管理している山です。季節によって入山規制がかかることもあるので（コース2の「入山規制案内」の看板などを参考に）、現地の状況をしっかり確認してから進むのが安全です。

トイレや水飲み場は、徐上2里（ソサン2リ）の終点や洪積（ホンジェク）終点の案内所付近など、登り口・下り口の一部にしかありません。尾根沿いには水場がないので、出発前に十分な量の水を持っておくようにしましょう。

また、韓国の登山道ではGoogleマップだと情報が正確でないことがよくあります。あらかじめ「カカオマップ」や「NAVERマップ」をインストールしておいて、リアルタイムのルートや公共交通機関の情報が確認できるようにしておくのがコツです。 尾根の一部には、危険地帯や滑落注意、足元が滑りやすい場所を示す看板が出ています。韓国語の指示を翻訳アプリなどで事前に確認したり、無理に単独で行くよりも、誰かと一緒に歩く計画を立てたりするのが一番ですよ。
