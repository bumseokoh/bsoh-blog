概要

写真：韓国観光公社 観光写真ギャラリー（釜山広域市 金井区 金城洞）· 公共ヌリ

金井山（クムジョンサン／Geumjeongsan）は、釜山広域市の金井区・北区・東莱区と慶尚南道梁山市にまたがる山で、主峰である姑堂峰（コダンボン／Godangbong）を中心に、稜線に沿って金井山城が続いていることで広く知られています。区域内に登録された登山道の総延長は複数のコースを合算した値で4.75kmですが、これは単一コースの距離ではなく、行政区域内の登山道の空間情報を集計した数字です。データ上、登録区間の難易度は「やさしい」に分類されていますが、以下でご紹介する山頂周辺の実測コースは距離も標高差もそれなりにあるので、体感の難易度は決して低くありません。登山道はよく整備されていて、公共交通機関でのアクセスも比較的しやすいため、日帰り登山で十分に楽しめる山です。

山なみと地形

写真: 韓国観光公社 観光写真ギャラリー（釜山広域市 金井区 金城洞）· 公共ヌリ

金井山は、花崗岩の岩盤があちこちで顔を出しつつも、全体としては土の道や尾根がよく発達した山として知られていて、土山（육산）と岩山（암산）の両方の性格をあわせ持つ山として紹介されることが多いですね。頂上にあたる姑堂峰（コダンボン）一帯は岩の峰といった風情で、尾根沿いに将軍峰・上鶏峰・玻璃峰など、いくつもの峰が連なっているというのが一般的な説明です。その尾根そのものが金井山城の城壁と重なっているため、城壁づたいに歩く区間が多いのも、この山ならではの大きな特徴です。実測データでは、頂上の最高標高は833mと確認できます。

登山の感じ

写真: 韓国観光公社 観光写真ギャラリー（釜山広域市 金井区 金城洞）· 公共누리

全体的に森の道が占める割合が高めなので、夏場でも木陰が続く区間が多いと言われていて、稜線まで登ると釜山の市街地や洛東江、梁山方面の眺めが開ける地点が現れると紹介されています。頂上部の姑堂峰あたりは岩稜地形なので、手をついて登らないといけない区間があったりして、土の山の区間とはまた違った緊張感を味わえます。春は城郭沿いの道に新緑が、秋はススキと紅葉が入り混じる山として、よく名前が挙がります。

コース1 — 分かれ道 → 高堂峰展望台

📏 9.88km ⛰️ 603m 獲得標高 ⏱️ 3:29 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 分かれ道 → 高堂の泉（고당샘） → 高堂峰展望台 → 高堂峰 登山口へのアクセス: 尾根の途中にある分かれ道から始まる区間で、ふつうは金井山城の北門か山城村（산성마을）の方面から入り、この地点まで登ってきてそのまま続く、という形で紹介されています。専用の駐車場は見当たらないので、山城村近くの路肩に停めるか、公共交通機関を使うのが無難だと思います。 特徴・おすすめの人: 途中の高堂の泉で飲み水を補給できるポイントがあると紹介されているコースで、尾根歩きと展望台からの眺めをどちらも楽しみたい方にぴったりです。距離が長めなので、体力配分には気をつけたいところです。

コース2 — 梵魚寺（ボモサ）駐車場 → 高堂峰（コダンボン）

📏 7.75km ⛰️ 628m 獲得標高 ⏱️ 2:59 ➡️ 片道 難易度 ★★★★☆ ⤓ GPX

経由: 梵魚寺駐車場 → 金井山城（クムジョンサンソン） → 梵魚寺 禅文化教育館 → 警備室 → 分岐11 → 幢竿支柱（タンガンジジュ） → 駐車場（80台収容） → 分岐14 → 分岐15 → トイレ → 分岐18 → 分岐19 → 分岐20 → 案内板 → 分岐22 登山口へのアクセス: 釜山都市鉄道1号線の梵魚寺駅から村バスや路線バスに乗り換えて梵魚寺の入口まで向かうのが一般的なアクセス方法とされています。梵魚寺の境内近くに大型駐車場（80台規模）があるので、マイカーでも比較的行きやすいほうです。 特徴・おすすめの方: 千年の古刹・梵魚寺を過ぎ、山城と溶け合う森の道を登っていく金井山の代表的なコースです。途中にトイレがあるので、初めての方や家族連れの登山客にも比較的無理のないコースになっています。お寺と山城の遺構をあわせて巡ってみたい方におすすめです。

コース3 — ホポ新町 → 山頂

📏 7.91km ⛰️ 738m 獲得標高 ⏱️ 3:12 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: ホポ新町 → 救助目標 金井山1-1 → ヨンドク農園 → 救助目標 金井山1-2 → 道標 → 開発制限区域案内板 → 分岐点7 → 救助目標 金井山1-3〜1-7 → 分岐点13、15 → 救助目標 東面林道6-28 登山口へのアクセス: 釜山都市鉄道2号線ホポ駅近くの集落から入っていくコースとして知られていて、大きな駐車場は見当たらないので、公共交通機関で行くのがおすすめです。 特徴・おすすめの人: 純標高差が738mと、山頂へ向かうコースのなかでもかなり急な部類に入るので、体力の消耗は大きめ。集落の道からだんだん林道・山道へと変わっていく構成が特徴です。ある程度登山に慣れた方向きですね。

コース4 — 分かれ道5 → 展望台

📏 6.13km ⛰️ 715m 獲得標高 ⏱️ 2:43 ➡️ 片道 難易度 ★★★★☆ ⤓ GPX

経由: 分かれ道5 → ホポ新町内 → ヤンサンヌリ道 → 金湖寺 → イ・スセン墓 → 東屋 → ベンチ・運動施設 → 分かれ道12 → ホポ農場 → 救助標識 金井2-1 → 車両通行止めパイプ → 道標・案内板 → ベンチ・救助標識 → 分かれ道18 → ベンチ・道標 登り口へのアクセス: コース3と同じくホポ方面からアクセスします。ヤンサンヌリ道とつながる区間を通るので、ヤンサン側の入口も合わせて参考にするといいと思います。 特徴・おすすめの人: 東屋やベンチ、運動施設など、ひと息つける場所があちこちにあって、途中の休憩をこまめに取りながら歩ける、なだらかな流れのコースとして紹介されています。ヤンサンヌリ道とつなげて歩いてみたい登山者にぴったりです。

コース5 — 青蓮庵の脇道 → 頂上

📏 10.14km ⛰️ 746m 獲得標高 ⏱️ 3:47 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 青蓮庵の脇道 → 塔 → 電気柵 → 立入禁止区間 → 分岐5 → 内院庵の案内板 → 分岐7・8・9 → 立入禁止の立て札 → ヒノキ林 → 鉄フェンス → 分岐14・15 → 湿地保護区域 登山口へのアクセス: 青蓮庵という寺のそばから始まるコースで、だいたい北区華明洞方面の進入路とつながっているようです。公共交通機関を使うなら、近くの路線バスを事前に確認してからのほうが安心です。 特徴・おすすめの人: ヒノキ林と湿地保護区域を通り抜けるのがこのコースの見どころで、一部に立入禁止の表示がある区間もあるので、標識は必ず守ってください。自然の生態系に興味のある登山者にとっては、なかなか面白いコースだと思います。

コース6 — タバン三差路 → タバン峰

📏 10.54km ⛰️ 726m 獲得標高 ⏱️ 3:51 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: タバン三差路 → タバン川案内図 → ケソク村停留所 → 標識石（ケソク村）→ 分岐5 → 極東飯店 → 分岐7 → テジョンアパート右手の道 → 分岐9 → 道標 → 鉄塔 → ベンチ → 分岐13 → イノシシ注意区間 → 山火事跡地 登山口へのアクセス: 梁山（ヤンサン）市タバン洞のケソク村方面から入っていくコースで、経由地に村バスの停留所が含まれているので、公共交通機関でのアクセスも比較的しやすそうです。 特徴・おすすめの人: 村の道から山火事跡地まで続く、ちょっと変わった構成が特徴です。野生動物（イノシシ）の出没案内も出ているので、単独よりは誰かと一緒に歩くのがおすすめですね。

コース7 — 錦山ヒューマンシア停留所 → 山頂

登山口へのアクセス: マンション団地のすぐ近くにあるバス停からスタートするコースです。山頂へ向かうコースの中でも距離が12.61kmと最も長く、丸一日がかりの山行に近い内容です。 特徴・おすすめの方: 距離も所要時間も最長のコースなので、体力に自信のある登山者向きです。往復するなら一泊まではいかないにしても、早朝のうちに出発しておくのがおすすめです。

コース8 — 地蔵庵入口 → 山頂

登り口へのアクセス: 地蔵庵というお寺のあたりから入っていくコースで、正味の標高差は475mと、山頂を目指すコースのなかでは比較的なだらかなほうです。 特徴・おすすめの方: 距離も標高差も比較的小さめなので、山頂まで登るコースのなかでは、入門者の方が挑戦するのにわりと負担が少ないコースと言えそうです。

距離と所要時間

写真：韓国観光公社 観光写真ギャラリー（釜山広域市 金井区 金城洞）・公共누리

山頂付近のコースは短いもので約7km、長いもので12.6kmほど。たいていは片道3〜4時間前後で計算されているので、往復しても日帰りで十分に行って帰ってこられます。管内の登録区間ぜんぶを合わせた累計だと、上りが82分・下りが59分と集計されているんですが、これは特定のコース1本の所要時間ではなく、管内の全区間を足し合わせた数字です。ここから換算すると、上りの平均ペースは約17分/kmになります。実際に山頂付近のコースで体力を配分するときには、このペースを目安にするといいと思います。

行き方（公共交通・駐車場）

写真: 韓国観光公社 観光写真ギャラリー（釜山広域市 金井区 金城洞）· 公共누리

釜山都市鉄道1号線の梵魚寺（ポモサ）駅から梵魚寺方面のバスかコミュニティバスに乗り換えれば、コース2（梵魚寺駐車場 → 姑堂峰）の登り口までたどり着けます。釜山都市鉄道2号線の湖浦（ホポ）駅の近くからは、コース3・4（湖浦セドンネ方面）へつながっているそうです。乗り換えの正確な路線やバス停の位置は、出発前にKakaoMapやNaver Mapでリアルタイムのルートを確認しておくのがおすすめです。公共交通データには、登山道の近くのバス停として新剣端中央駅交差点（トンネル前）、ディエトルザヒル（高山トンネル）、パラゴンボタニックパーク（高山トンネル）、プルジオザベニュー裏門、ディエトルザヒル・イウム高校などが登録されているので参考にはなりますが、実際に利用する前に地図アプリでそのバス停と登り口の間の距離を改めて確認しておいてください。駐車場は梵魚寺の近く（約80台分）を除けば、これといった大きな施設は見当たらないので、できれば公共交通機関の利用をおすすめします。

시즌·날씨·장비

사진: 한국관광공사 관광사진 갤러리 (부산광역시 금정구 금성동) · 공공누리

春と秋が山歩きには無難なシーズンとされていて、夏場は湿気とうだるような暑さ、それに予告なしの夕立に備えておく必要があります。冬場は山頂付近の岩稜帯が凍結する恐れがあるので、チェーンアイゼンと防寒装備は欠かせません。基本装備としては、登山靴、着替え、たっぷりの飲み水（水飲み場がほとんどないので、最低でも1〜1.5Lは持っていくのがおすすめです）、ヘッドランプ、雨具か防水ジャケットを用意しておきましょう。

外国人旅行者向けのヒント

写真: 韓国観光公社 観光写真ギャラリー（釜山広域市 金井区 金城洞）· 公共누리

山頂は Godangbong（고당봉）、山の名前は Geumjeongsan（금정산）と表記します。入山許可や入場料はとくに必要ないとされていますが、梵魚寺（범어사）などお寺の境内では別途の文化財拝観料がかかる場合があるので、現地の案内を確認してください。トイレはコース2（梵魚寺駐車場 → 고당봉）の途中に確認できますが、水飲み場は公共データ上、登録された地点がないため、飲み水は必ず持参してください。グーグルマップは韓国国内の登山道情報が正確でないことが多いので、KakaoMap や Naver Map のアプリをあらかじめインストールして、実際のルートや公共交通機関を確認しておくと安心です。山頂付近の岩稜帯ではスリップ事故に気をつけ、日が暮れてから下山が遅くならないよう、出発時間には余裕をもって計画を立ててください。
