概要

写真: 韓国観光公社 観光写真ギャラリー（釜山広域市金井区金城洞） · 公共누리

金情山（クムジョンサン）は、釜山の金井区・北区・東莱区、そして慶尚南道の梁山（ヤンサン）市にまたがる山です。主峰である高堂峰（コダンボン）を中心に、稜線に沿って続く金精山城があることで広く知られています。

さて、登録されている登山道の総延長は4.75kmとなっていますが、これはあくまで複数のコースを合算した数値。単一のルートの距離ではありません。データ上は「難易度：低」と分類されていますが、これからご紹介する山頂付近のコースは距離も標高差もしっかりあるので、実際に歩いてみると決して「楽勝」とはいかないはずです。

道はよく整備されていますし、公共交通機関でのアクセスも比較的スムーズなので、日帰りハイキングにはちょうどいい、そんな山です。

山の地形と特徴

写真: 韓国観光公社 観光写真ギャラリー (釜山広域市金井区金城洞) · 公共누리

金井山は、あちこちに花崗岩の岩肌が顔をのぞかせつつも、全体的には土の道や稜線がよく発達した山です。「岩山」としての表情と「土山」としての表情、その両方を一度に味わえるのが面白いところですね。

山頂の高堂峰（コダンボン）一帯は岩の峰が連なるような地形になっていて、稜線に沿って将軍峰、上渓峰、パリ峰といったさまざまな峰々が続いています。また、稜線そのものが金井山城の城壁と重なっているため、城壁に沿って歩ける区間が多いのもこの山の大きな特徴です。ちなみに、実測データによる最高標高は833mとのこと。

登山の雰囲気

写真: 韓国観光公社 観光写真ギャラリー (釜山広域市 金井区 金城洞) · 公共누리

全体的に森の中を歩く道が多く、夏でも木陰が続く区間が多いのが魅力です。稜線に上がると、釜山市街地や洛東江（ナクトンガン）、そして梁山（ヤンサン）方面まで見渡せる絶好のビューポイントが次々と現れます。 山頂の高堂峰（コダォンボン）付近は岩場になっているので、時には手を使う場面もあり、それまでの緩やかな道とはまた違った程よい緊張感がありますね。 春は城壁沿いの美しい新緑、秋はススキと紅葉のコントラスト……そんな四季折々の表情を楽しめる山として、よく名前が挙がるスポットです。

コース1 — 分かれ道 → 高堂峰展望台

📏 9.88km ⛰️ 603m 獲得標高 ⏱️ 3:29 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 分かれ道 → 高堂の井戸（コダンセム） → 高堂峰展望台 → 高堂峰 アクセス: 尾根の途中にある分かれ道から始まる区間です。だいたい金井山城（クムジョンサンソン）の北門や山城村方面から登ってきて、この地点に辿り着くルートとして紹介されることが多いですね。専用の駐車場は見当たらないので、山城村付近の路肩に停めるか、公共交通機関を利用するのが無難そうです。 特徴・おすすめ: 高堂の井戸で水分補給ができるのがポイント。尾根歩きと展望台からの絶景をどちらも欲張りに楽しみたいハイカーにはぴったりです。ただ、距離がそれなりにあるので、体力を温存しながら無理のないペースで進みましょう。

コース 2 — 梵魚寺（ポモサ）駐車場 → 高堂峰

📏 7.75km ⛰️ 628m 獲得標高 ⏱️ 2:59 ➡️ 片道 難易度 ★★★★☆ ⤓ GPX

経由: 梵魚寺駐車場 → 金井山城跡 → 梵魚寺 禅文化教育館 → 警備室 → 分かれ道11 → 塔身柱 → 駐車場（80台収容） → 分かれ道14 → 分かれ道15 → トイレ → 分かれ道18 → 分かれ道19 → 分かれ道20 → 案内板 → 分かれ道22

アクセス: 釜山都市鉄道1号線の梵魚寺駅（ポモサ駅）から、村バスや市バスに乗り換えて梵魚寺の入り口まで向かうのが一般的なルートです。なお、境内付近に80台ほど停められる大型駐車場があるので、車でのアクセスもかなりスムーズですよ。

特徴・おすすめ: 千年の歴史を誇る古刹、梵魚寺を抜け、山城の面影を残す森の中を登っていく金井山の代表的なコースです。途中にトイレもあるので、初めての方やご家族連れでも比較的歩きやすいのが嬉しいポイント。お寺の雰囲気と山城の遺構を一度に楽しみたい方に、ぜひおすすめしたいルートです。

コース 3 — ホポセドンネ → 頂上

📏 7.91km ⛰️ 738m 獲得標高 ⏱️ 3:12 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: ホポセドンネ → 枯れ木 金情山1-1 → 龍徳農園 → 枯れ木 金情山1-2 → 道標 → 開発制限区域案内板 → 分かれ道 7 → 枯れ木 金情山1-3~1-7 → 分かれ道 13, 15 → 枯れ木 東面林道 6-28 アクセス: 釜山地下鉄2号線のホポ駅近くの集落から入るコースです。大きな駐車場などは見当たらないので、公共交通機関を利用するのが一番スムーズかと思います。 特徴・おすすめの人: 獲得標高が738mもあり、頂上へ向かうコースの中ではかなりの急勾配。それなりに体力を削られるので、正直ちょっとした覚悟が必要かもしれません……。集落の道から徐々に林道や山道へと切り替わっていく構成が特徴ですが、その分登りがいがあります。ある程度、山歩きの経験がある方に向いているコースです。

コース 4 — 分かれ道5から展望台へ

📏 6.13km ⛰️ 715m 獲得標高 ⏱️ 2:43 ➡️ 片道 難易度 ★★★★☆ ⤓ GPX

経由: 分かれ道5 → ホポ・セドンネ → ヤンサン・ヌリギル → 金虎寺 → 李水生墓 → あずまや → ベンチ・運動施設 → 分かれ道12 → ホポ農場 → 構造木 金静2-1 → 車両通行止めパイプ → 道標・案内板 → ベンチ・構造木 → 分かれ道18 → ベンチ・道標

アクセス: コース3と同様に、ホポ方面から向かうルートです。ヤンサン・ヌリギルへと繋がる区間を通りますので、ヤンサン側の入り口も併せて目印にしておくとスムーズですよ。

特徴・おすすめ: あずまややベンチ、運動施設など、ちょっと一休みできる場所が点在しています。こまめに休憩を挟みながら進める緩やかなコースなので、のんびり歩きたい時に最適です。ヤンサン・ヌリギルと一緒に散策を楽しみたいハイカーの方にもぴったりですよ。

コース 5 — 青蓮庵（チョリョンアム）脇道 → 山頂

📏 10.14km ⛰️ 746m 獲得標高 ⏱️ 3:47 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 青蓮庵脇道 → 塔 → 電気柵 → 立入禁止区間 → 分岐5 → 内院庵看板 → 分岐7・8・9 → 立入禁止看板 → ヒノキの群生地帯 → 鉄柵 → 分岐14・15 → 湿地保護区 アクセス: お寺の青蓮庵付近からスタートするコースで、基本的には北区華明洞（ファミョンドン）方面への入り口へとつながっています。公共交通機関を利用する場合は、あらかじめバスの路線をしっかり確認しておくと安心ですよ（道に迷って体力を消耗するのは勘弁ですからね）。 特徴・おすすめ: ヒノキの群生地帯や湿地保護区を通り抜けるのがこのコースの醍醐味です。ただし、一部に立入禁止エリアがあるので看板の指示には必ず従ってくださいね。自然の生態系を間近で感じたいという、ちょっとマニアックな登山好きの方には面白いルートだと思います。

コース 6 — 多房三叉路 → 多房峰

📏 10.54km ⛰️ 726m 獲得標高 ⏱️ 3:51 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 多房三叉路 → 多房川案内図 → 桂石村停留所 → 標識（桂石村） → 分岐点5 → 極東飯店 → 分岐点7 → 大正アパート右道 → 分岐点9 → 道標 → 送電鉄塔 → 休憩台 → 分岐点13 → イノシシ注意区間 → 山火事跡地

アクセス: ヤンサン市多房洞の桂石村方面から入るコースです。停留所が経由地に含まれているので、バスを使ったアクセスも結構スムーズに行けそうです。

特徴・おすすめ: 集落の道から山火事跡地へと続く、ちょっと趣のある（？）構成になっています。イノシシが出るという注意書きもあるので、お一人様よりは誰かと一緒に行くのが安心かもしれませんね。

コース 7 — クムサン・ヒューマンシア停留所 → 山頂

登り口へのアクセス: アパートのすぐ近くにあるバス停からスタートするコースです。山頂を目指すルートの中では12.61kmと一番長く、もはや丸一日がかりの山行になりそうな距離感です。 特徴・おすすめ: 最長距離・最長時間となるタフなコースなので、体力に自信のある方向け。往復を考えると、宿泊まではいかなくとも早朝の出発が賢明でしょう。

コース8 — 地蔵庵入口 → 山頂

アプローチ: お寺の「地蔵庵」付近から登り始めるコースです。累積標高差は475mほどで、山頂を目指すルートとしては比較的緩やかな登りになっています。 特徴・おすすめ: 距離も標高差もそこまで大きくないので、山頂を目指すコースの中では初心者の方でも挑戦しやすく、負担の少ないルートといえるでしょう。

なお、上の表は複数のコースを合算した総延長ですので、単一のコース距離ではありません。路面は主に土ですが、高堂峰（コダンボン）の山頂付近などは花崗岩が露出した岩場になっています。雨の日や冬の凍結時には滑りやすくなるので、くれぐれも足元には注意してくださいね。

距離と所要時間

写真: 韓国観光公社 観光写真ギャラリー (釜山広域市 金井区 金城洞) · 公共누리

山頂へ向かうコースは、短いもので約7km、長いものだと12.6kmに及びます。片道でおよそ3〜4時間ほどかかる計算になるので、往復しても一日で十分に歩ききれる距離ですね。

ちなみに、登録されている全区間を対象とした累積所要時間は「登り82分・下り59分」と集計されています。これは特定のコース単体の時間ではなく、全区間を合算した数値ですが、ここから算出される平均的な登りのペースは約17分/kmとなります。 実際の登山で、無理なく体力を温存するための目安として参考にしてみてください。

アクセス（公共交通機関・駐車場）

写真：韓国観光公社 観光写真ギャラリー（釜山広域市金井区金城洞）· 公共ヌリ

地下鉄1号線の梵臥寺（ポモスァ）駅から、梵臥寺方面行きのバスや村バスに乗り換えると、コース2（梵臥寺駐車場 → 高堂峰）の登山口に辿り着けます。また、地下鉄2号線の浩浦（ホポ）駅付近からは、コース3・4（浩浦セドンネ方面）へと繋がっています。正確な乗り換え路線や停留所の位置については、出発前にKakaoMapやNaver Mapでリアルタイムの経路をチェックしておくのが一番ですね。

公共交通機関のデータには、登山道近くの停留所として「シンコムダンチュンアンヨクサグリ（トンネル前）」「ディエトル・ザ・ヒル（コサン・トンネル）」「パラゴン・ボタニカルパーク（コサン・トンネル）」「プルジオ・ザ・ベニュー裏門」「ディエトル・ザ・ヒル・イウム校」などが登録されていますが、これらはあくまで目安。実際に利用する際は、地図アプリを使って停留所から登山口までの距離を改めて確認するようにしてください。駐車場については、梵臥寺付近（約80台規模）を除いて大規模な施設は見当たりませんので、なるべく公共交通機関を利用するのが良さそうです。

シーズン・天候・装備

写真: 韓国観光公社 観光写真ギャラリー (釜山広域市衿田区金城洞) ・ 公共누리

登山には春と秋がほどよく、一番のベストシーズンと言えますね。夏場は湿度や暑さ、それに突然の夕立への備えが欠かせません。逆に冬は、山頂付近の岩場が凍結する恐れがあるので、チェーンアイゼンや防寒装備は必須です。

基本の装備としては、登山靴と着替え、あとは十分な飲み物を用意しましょう。水場がほとんどないので、最低でも1〜1.5L以上は持参することを強くおすすめします。あわせてヘッドライトやレインウェア（防水ジャケット）も、忘れずに準備しておいてくださいね。

旅人へのちょっとしたアドバイス

사진: 한국관광공사 관광사진 갤러리 (부산광역시 금정구 금성동) · 공공누리

山全体の名前は金井山（クムジョンサン）、頂上は高堂峰（コダンボン）といいます。 入山許可や入場料などは特に必要ないようですが、法漁寺（ポムサ）などの寺院エリアに入ると別途拝観料がかかる場合もあるので、現地の案内をよく確認してくださいね。

トイレはコース2（法漁寺駐車場〜高堂峰）の区間の中ほどにありますが、水飲み場は見当たらないので、必ず飲み物を持参するようにしましょう。 また、韓国ではGoogleマップだと登山道の情報が正確でないこともよくあります。KakaoMapやNaver Mapをあらかじめインストールしておき、実際のルートや公共交通機関を確認しておくと安心ですよ。

山頂付近の岩場は滑りやすいので、転倒には十分注意してください。日が暮れてから暗い中を下山することにならないよう、時間に余裕を持って出発する計画を立てるのがコツです。
