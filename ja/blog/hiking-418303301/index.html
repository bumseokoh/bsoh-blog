有明山（유명산、ユミョンサン）は京畿道加平郡と楊平郡の境界にまたがる山で、頂上の標高は資料によって869m・883m・884mなどと微妙に表記が揺れているものの、登山路の実測値でいくと最高地点は889mになるようです。お隣の魚飛山・小九尼山・中美山とは尾根続きになっていて、同じ山塊の一部を成しています。全体としては土の道が多い「肉山」寄りの性格が強いのですが、尾根筋には岩場も混じっていて、ただの土山と侮れないところもあります。難易度は登山路データ上「やさしい」区分も含まれていて初心者でも挑戦できるコースがある一方、登山口によっては片道4時間を超える長丁場・高難度コースも同居しているので、選ぶ入口次第でまったく違う山に化けます。日帰り登山で十分こなせる山ですし、夏場は渓谷の避暑地としても名の知れた存在です。

有明山は花崗岩系の土山（육산）を基本骨格としながら、頂上稜線一帯には露出した岩稜・岩峰区間も混じっている山です。管内の峰データを見ると、有明山の頂上のほかに於飛山（849m/846mの2地点で登録）、小九尼山（798m/812mの2地点で登録）、中美山（849m）が稜線でつながっていて、有明山一峰だけでなく複数の峰を結ぶ山稜全体を縦走するように歩けるのが特徴です。谷筋は幅が広く水量も豊富なほうで、夏場の水遊びスポットとして知られる有明山渓谷がその代表格。山の麓には有明山自然休養林があり、キャンプ場や林道が登山道と重なる区間もあります。頂上部は視界の開けた稜線型の峰で、切り立った岩峰がそびえる尖峰型の山ではありませんが、あちこちにある岩場区間が山行に変化を添えてくれます。

登山の感覚

写真: 韓国観光公社 観光写真ギャラリー（慶尚北道蔚珍郡）・公共누리

登山口から頂上まで、だいたい林道→渓谷道→尾根道の順に雰囲気が変わっていくコースが多いです。入口はまず広葉樹と針葉樹が入り混じったなだらかな林道から始まり、木陰が濃くて湿気を感じやすいのですが、渓谷沿いを登っていく区間では水音を聞きながら歩く清々しさがあります。尾根に近づくにつれて傾斜がきつくなり、岩がむき出しになった区間も出てきて体感的な難易度が上がっていき、頂上付近では木々の間から周囲の山並みが開けて見えるポイントもいくつかあります。春には新緑とヤマツツジ、秋には紅葉が渓谷の水と溶け合って色合いが際立つ季節感があり、夏は渓谷コースならではの涼しさと、湿った林道のむわっとした暑さが同時に味わえる山行になります。

선어치고개 → 정상

経由地：仙魚峙峠 → 分岐点 → 所九尼山（ソグニサン） → 有明山方向 → 有明山 → 渓谷コース

登山口へのアクセス：仙魚峙峠は加平郡（カピョングン）と楊平郡（ヤンピョングン）の境界にある峠で、このあたりは公共交通の便があまり良くないので、マイカーやタクシーで向かうのが一般的です。ちゃんとした駐車場があるというより路肩スペースに停めるケースが多いので、あまり期待せず、事前に地図アプリで進入路の幅を確認しておくと安心です。

特徴・おすすめの人：所九尼山を先に経由してから有明山へ取り付く稜線コースで、距離が短く高低差も比較的なだらかなので、旅徳コースの中では一番気軽に歩けるコースだと思います。稜線を歩いたあと下山は渓谷コースに抜ける形になっているので、登りと下りで雰囲気ががらっと変わるのも面白いところ。短時間で頂上稜線を味わってみたい初心者や、時間があまり取れない登山者にちょうどいいコースです。

コース2 — ボクドン三叉路 → 頂上

経由地：ボクドン三叉路 → パゴダキャンプ場 → 分岐点 → デブサン → 林道 → パラグライダー滑空場 → 高圧線 → 滑空場2 → ユミョンサン

登山口へのアクセス：ボクドン三叉路は公共データにも登録済みのバス停で、公共交通機関だけで登山口まで行けるのがありがたいポイントです。ただ、専用駐車場のデータは見当たらないので、車で行くなら道端の空きスペースを見つけて停めることになります。週末は混みがちなので、できれば公共交通機関を使うのが無難かなと思います。

特徴・おすすめの人：デブサンを越えて林道からパラグライダー滑空場の区間を抜け、頂上に取り付くコースです。累積標高差がなかなかのもので、序盤からずっと登り基調が続く、体力をごりごり削られるタイプのコースだと覚悟しておいてください。とはいえ滑空場の区間では視界が一気に開けて、景色を楽しめるポイントも出てきます。体力と経験にそれなりの自信がある山好きの方には、ぜひ挑戦してほしい定番コースです。

第3コース — 第2チケット売り場入口 → 頂上

経由: ハンファコンドサムゴリ → トンマッコル → パラグライダーパーク → パラグライディング → 分岐点 → 楊平青山 → ユミョン山林道 → 伐採作業地 → 登山路 → パラグライディング1 → パラグライディング2 → 四兄弟松 → 婚姻木 → 第2チケット売り場入口 → 第2売り場

登山口へのアクセス: ハンファコンドサムゴリにはバス停が登録されているので、公共交通機関でも行けるポイントです。チケット売り場のある正式な登山口ではあるものの、公共データ上は駐車場が別途確認できていません。車で行く場合は、近くの商店や宿泊施設の駐車スペースを事前にチェックしておいたほうが安心です。

特徴・おすすめの人: トンマッコルとパラグライダーパークを抜けて、四兄弟松や婚姻木といった名前の付いたスポットを巡っていく、一番距離の長いコースです。片道だけでも半日近くかかります。いろいろな区間をまんべんなく歩く総合的な山行がしたい人や、体力を使うロングトレッキングを目標にしている人向けです。

特徴・おすすめしたい人: 名前のとおり渓谷沿いを登っていくコースで、水の音を聞きながら歩ける区間が多く、特に夏場の登山で人気があります。距離が短く、しかも施設の整った登山口からスタートできるので、渓谷トレッキングと頂上登頂を一度に味わいたいという人にはぴったりのコースだと思います。

コース6 — 有名山終点 → 頂上

特徴・おすすめしたい人: 野花のせせらぎからスタートして於飛山を越え、有名山へとつながる尾根縦走型のコースで、眺望休憩所という名の展望ポイントを通りながら、於飛山と有名山という二つの峰を一度に味わえます。距離も高低差もそれなりにあるので体力は要りますが、公共交通機関だけでスタートとゴールを済ませられるのが強みで、縦走登山を計画している人には一番現実的な選択肢だと思います。

距離と所要時間

写真：韓国観光公社 観光写真ギャラリー（京畿道加平郡長水路156-20）・公共누리

管内の登山道の総延長は、複数コースを合算した19.51kmです。これは単一コースの距離ではなく、佳日里区間・龍泉里区間を含む管内全登山道の累積距離である点に注意してください。データ上の上り373分・下り260分という数値も同様に、管内全区間を積み上げたものなので、個別コースの所要時間と勘違いしないようにしましょう。この累積値から算出される平均登坂ペースはおよそ19分/kmで、実測GPXベースのコース別推定所要時間（例：ソノチコゲ峠コース約1時間23分、ポクトン三叉路コース約2時間44分）と併せて参考にすると、自分の体力に合ったコース選びに役立ちます。ユミョン山は日帰り登山で十分踏破できる山で、短めのコースなら半日、第2切符売り場入口や유명산終点のように稜線を長くつなぐコースだと丸一日かかることもあるので、出発時刻には余裕を持たせておくのがおすすめです。1泊が必要になるほどの規模の山ではありませんが、複数コースをつなげて縦走する場合は、トータルの所要時間を多めに見積もっておいたほうが安心です。

シーズン・天気・装備

写真：韓国観光公社観光写真ギャラリー（京畿道加平郡雪岳面長寿路79番길63-18）・公共누리

春と秋は気温も眺望も安定していて一番歩きやすい時期で、新緑や紅葉を渓谷とセットで楽しめます。夏は渓谷コースの人気がぐっと高まるシーズンですが、湿度が高いうえに雨が降ると渓谷の水量が一気に増えることがあるので、水遊びと登山を一緒に計画するときは天候をこまめにチェックしておいたほうがいいです。冬は稜線や岩稜区間の凍結リスクが高まる時期で、避難小屋や危険区域の表示がない山だということも踏まえて、アイゼン・スパッツなど冬装備は必ず用意しておくことをおすすめします。共通の必須装備としては、登山靴、予備の水（水場の設備がないため全量持参が必須です）、防水ジャケット、トレッキングポール、ヘッドランプ（長距離コースを歩く場合）などが挙げられます。

ユミョンサン(有名山)はローマ字表記だと「Yumyeongsan」です。主な登山口となる「福東三叉路(ボクドンサムゴリ)」「有名山終点(ユミョンサンジョンジョム、終点バス停)」「ハンファコンド三叉路」あたりの地名は、バスに乗る前に頭に入れておくと現地でかなり楽になります。

入山許可や入場料は基本的にかかりませんが、有名山自然休養林のエリアを使う場合は休養林側の入場・施設利用料が別途発生することがあるので、現地か公式サイトで事前に確認しておいたほうが安心です。トイレはコース上のいくつかの地点(下山口のトイレ、登山道や渓谷入口付近のトイレなど)にはありますが、コース全体をカバーしているわけではないので、登山口で済ませておくのが無難です。飲み水の補給スポットは公共データ上登録がないので、水は必ず多めに持っていってください。

地図アプリについては、Googleマップは韓国の登山道情報だとどうも精度が落ちる印象があるので、現地で広く使われている「カカオマップ」か「ネイバー地図」を入れておいて、実際の登山道ルートや公共交通の乗り換え情報を確認するのをおすすめします。安全面では、稜線の一部で電波が入りにくい区間があるので単独より同行者がいる山行が望ましく、万一の際は韓国の統合通報番号「119」が使えます。
