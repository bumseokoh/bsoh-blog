雲岳山(운악산、ウナクサン)は、京畿道抱川市華岳面と加平郡下面の境にそびえる山です。頂上の標高は資料によって935m前後とされていますが、実測GPXによる最高地点は948mに達します。昔から華岳山・冠岳山・紺岳山・松岳山と並んで「京畿五嶽(オアク)」の一つに数えられ、険しい岩稜と奇岩怪石が織りなす風景から「京畿の小金剛」という異名でも呼ばれています。全体的に短く急な岩稜区間が多く、体力と注意力が求められる山ですが、コースの大半は日帰りで往復・縦走できる規模なので、日帰り登山として十分楽しめる山です。

山容と地形

運악산は、なだらかな土の山というよりも、岩肌がその山容を支配する「岩の山」といった趣です。ひとつのずんぐりした峰で構成されているのではなく、西峰・東峰(頂上)・毘盧峰など複数の岩峰が稜線沿いに連なる構造をしていて、公共データに登録されている主な峰だけでも運岳山西峰(962m)、東峰(934m)、運岳山毘盧峰(937.5m、別称929m)、西峰(918m)、アギ峰(769m)、峨嵯山(513m)、円通山(590m)などバリエーション豊かです。なかでも頂上部の東峰一帯と西峰を結ぶ岩稜区間が運岳山登山のハイライトで、眉毛岩・屏風岩・立石岩といった名前の付いた岩々が稜線に沿ってずらりと並んでいます。南斜面には新羅時代の創建と伝わる懸燈寺があり、渓谷と寺院、岩峰が一体となった地形的特徴を見せています。

登山で感じたこと

写真:韓国観光公社観光写真ギャラリー(京畿道加平郡明智山路137-49)・公共누리

雲岳山は登り始めから急な階段と岩場が次々に現れて、序盤から「これはなかなか手強いぞ」と感じさせてくれる山です。稜線に出ると視界がぱっと開けて、抱川・加平一帯の山並みが一望でき、なかでも屏風岩展望台や西峰付近では、切り立った岩稜の上を歩くヒヤヒヤ感がたまりません。渓谷コース(懸燈寺方面)は夏場も木陰と水音のおかげでひんやりと涼しく感じられる一方、稜線コースは直射日光をまともに浴びる区間が多く、季節によって体感がかなり変わってきます。春はツツジと新緑、秋は紅葉が岩稜と重なり合って写真映えするスポットとしても知られていて、冬は岩稜に薄氷が張ることが多いため、山行の空気もぐっと引き締まる、そんな山です。

特徴・おすすめのタイプ: 円通山を経て西峰へと続くこのコースは、5コースの中でも一番距離が長く、稜線区間もたっぷり取れているのが特徴です。短くて急な岩稜をサクッと登るタイプより、緩急織り交ぜたロングな縦走を楽しみたい人向けと言えます。途中、イノシシの痕跡や滑りやすい注意ポイントが実測の経由地として確認されているくらいなので、ある程度山道に慣れた登山者におすすめしたいコースです。

コース2 — 玄燈寺入口 → 屏風岩展望台

経由地: 雲岳山下板里 → 公衆トイレ → 玄燈寺入口 → 分岐点 → 유경冷麺 → 玄燈寺観光案内所 → 簡易トイレ → 眉毛岩 → 道標前の休憩所 → 屏風岩展望台 → 雲岳山毘盧峰(937.5m) → 山頂下の休憩所 → 玄燈寺の湧き水 → 玄燈寺トイレ → 玄燈寺簡易トイレ

登山口へのアクセス: 下板里区間にまたがる玄燈寺入口は、雲岳山の各コースの中でも公共交通機関でいちばん行きやすい部類に入ります。抱川や加平方面のバスが玄燈寺入口付近の停留所を経由してくれるので、そこからはお寺の参道沿いに歩いていく形です。玄燈寺観光案内所の近くに駐車スペースもあるらしいので、車で来る人もそこそこいるみたいです。

特徴・おすすめしたい人: トイレと湧き水がコースのあちこち（公衆トイレ、簡易トイレ、玄燈寺の湧き水・トイレなど）にマークされていて、五つのコースの中では一番設備が充実している区間だと思います。眉毛岩と屏風岩展望台を通り過ぎながら、雲岳山らしい岩稜の眺めを短い距離（6.24km）でギュッと味わえるので、初めて雲岳山に来る人や、写真目的で登る人には特におすすめです。ただ、距離のわりに累積標高差（742m）が五コース中いちばん大きいので、序盤からけっこう急な傾斜に感じるかもしれません。正直なところ、なめてかかると最初の登りで息が上がります。

こんにちは、山道好き40代のブロガー、雨男です。今回はコース3、運岳山休憩所からアギ峰までのルートをご紹介します。

特徴・おすすめの人: このコースは「山崩れ危険区域」「危険区域」という地点が経由地としてはっきり記されている通り、雨上がり直後や凍結期にはとくに気をつけたいコースです。その代わり、貴木峰・恋人山・明智山・カルボン山稜線の展望ポイントに加えて、水原山・雲岳山の展望地点まで次々と現れるので、稜線の景色をじっくり楽しみたい人にはたまらないコースだと思います。オミナエシ、ニラ、ヒロハママコナといった山野草の群生地も経由地として記録されているので、季節の花を眺めながら歩くのにもぴったりです。

難易度・路面・注意点

写真：韓国観光公社観光写真ギャラリー（京畿道抱川市華岳面峰火路201-65）・公共누리

管内の登山道の空間情報上、雲岳山の最高難易度等級は「中間」に分類されています。ただ、実測GPXに基づく頂上圏コース5本は、ほとんどが★★★★☆以上の高い難易度を示していて、体感的にはこの等級表記よりもきつく感じられるはずです。コースごとの詳しい路面情報までは別途登録されていないのですが、西峰・屏風岩・眉岩(ヌンソッパウィ)区間の実測経由地名からもわかる通り、岩稜がこの山行の主な地形をなしています。とくにコース4（アチ山→頂上）には「山崩れ危険地域」と「危険地域」の2地点が登録されていて、雲岳山全体に登録されている危険地域2カ所（危険案内・危険警告各1件）がこのコースに集中している形です。雨上がり直後は岩が滑りやすくなりますし、冬場は日陰側の岩稜で凍結がよく起きるので、この区間を通るときはとくにペースを落として、三点支持を意識しながら進むのがおすすめです。一方、展望ポイントとして個別に登録されている地点はないものの、実測の経由地情報からは屏風岩展望台、稜線からの眺め、水源山・雲岳山を望むポイントなど、見どころとなる展望スポットが複数確認できます。

距離と所要時間

사진: 韓国観光公社 観光写真ギャラリー（京畿道抱川市禾峴面禾東路548-47）· 公共누리

雲岳山は管内の登山道の総延長が12.89km（複数区間を合算した数値）で、上り258分・下り180分という数値もやはり管内全区間を積み上げたもので、特定の一コースにかかる時間ではありません。この累積値を距離で割ると、上りの平均ペースは1kmあたり約20分という計算になります。平地の徒歩よりもかなり遅い速度ですが、これは岩稜や急斜面の多い雲岳山の地形をよく物語っているとも言えます。実際に代表的なコースの片道所要時間は、GPX実測に基づくと短いもので約2時間48分（ヒョンドゥンサ入口・登山口系ルート）、長いものでは3時間42分（登山口→アギボン）とかなり幅があり、往復となると5〜7時間前後を見ておくのが一般的です。管内に登録された避難小屋は確認できないため、5つのコースはいずれも日帰り往復で十分にこなせる山行で、縦走でもしない限り宿泊の必要はありません。

가는 길(대중교통·주차)

운악산は京都・保定市と加平郡の境にまたがる山なので、ソウルから行くなら公共交通よりマイカーのほうが正直ラクです。それでも公共交通で向かう場合は、東ソウル総合ターミナルか清凉里方面から保定・加平行きの市外/市内バスに乗り換え、「雲岳山入口」「雲岳山.懸燈寺入口」「抱川.雲岳山入口」「大院寺」といった停留所のいずれかで降りて、そこから登山口まで歩くのが一般的なルートです。このうち「雲岳山.懸燈寺入口」の停留所は、コース2（懸燈寺入口→屏風岩展望台）にそのままつながっているので覚えておくと便利です。

なお、登録された専用駐車場は特に見当たらないので、懸燈寺や雲岳山休憩所の近くにある民間の小さな駐車スペースを使うことになりそうです。正確なバスの時刻表や乗り換えルート、リアルタイムの道路状況は、KakaoMapかNaver Mapのアプリで「雲岳山」または停留所名を検索して確認するのがおすすめです。外国から来られる方は要注意で、Googleマップは韓国国内の公共交通・登山道の情報が不正確なことが多いので、上記2つのアプリのどちらかを必ず使うようにしてください。

雲岳山に登るなら、やっぱり一番おすすめなのは秋です。紅葉を眺めながらひんやりした気温の中で岩稜を上り下りするのにちょうどいい季節ですね。春はツツジが満開になる時期で一気に人が押し寄せますが、空気が乾燥しがちなので山火事のリスクにはくれぐれも気をつけたいところです。夏は蒸し暑さと湿気に加えて急な夕立に見舞われることも多く、岩稜区間が濡れて滑りやすくなるので、雨具と着替えを一枚多めに持っていくと安心です。そして冬。日陰になる稜線や岩場は凍結しやすく、五つあるコースすべてで難易度がぐっと上がるので、アイゼン・スパッツ・防寒手袋はほぼ必須装備だと思っておいてください。

季節を問わず基本装備として持っていきたいのは、登山靴(岩稜対応でグリップの効くもの)、トレッキングポール、そして十分な飲み水です。山域内に登録された水飲み場がないので、事前にしっかり用意しておくことが大事です。あとは手袋と、下山が遅れた場合に備えたヘッドランプも忘れずに。

外国人旅行者へのヒント

写真: 韓国観光公社観光写真ギャラリー（京畿道抱川市華憲面達仁洞路35）・公共누리

雲岳山は英語表記だと「Mount Unak」または「Unaksan」となり、山頂部は地図アプリで「Unaksan Peak」または「東峰(Dongbong)」と検索すれば場所が確認できます。主な登山口のひとつである懸燈寺入口は「Hyeondeungsa Temple entrance」で検索可能です。国立公園ではなく山林庁管轄の登山道なので、入山許可や入場料は特にかかりませんが、懸燈寺の境内を通る際はお寺見学としてのマナーを守っておいたほうがいいでしょう。トイレはコース2（懸燈寺方面）を中心に公衆トイレ・簡易トイレ・お手洗いなどがいくつか確認できますが、それ以外の稜線区間には使えるトイレがないので、登山口で済ませておくのをおすすめします。給水所は登録地点がなく（飲み水台0か所）、山行前に水を十分に持っていく必要があります。懸燈寺付近の泉が実測ルート上に記載されてはいますが、常時使えるかどうかは現地で確認したほうが安心です。前にも書きましたが、地図アプリはGoogleマップよりKakaoMapやNaver Mapのほうがずっと正確で、登山道のリアルタイム位置確認にも役立ちます。岩稜区間が多いので、雨の日や凍結期は山行を避けるか、アイゼンなどの装備を必ず用意してください。危険地域として表示されている2か所（コース4付近）では、案内標識の警告を必ず守るようにしましょう。携帯電話の電波は稜線部のほとんどで良好ですが、念のため、国立公園ではない山だということを踏まえて、山行計画と下山予定時刻を同行者や宿泊先にあらかじめ伝えておくことをおすすめします。

私、根がのんびり屋なもので、山の話になると「あとちょっとだけ」がすぐ「もうちょっと」に化けてしまうタイプなんですが、今回はその欲張り心をぐっとこらえて筆を置くことにします。ただ、せっかくなので周辺の山もざっくり書き添えておきますね。地図を眺めながら「次はここかな」なんて妄想するのも、登山の楽しみのひとつだったりしますし。
