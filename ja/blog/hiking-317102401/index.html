概要

写真: 韓国観光公社 観光写真ギャラリー (蔚山広域市鬱州郡尚北面) · 公共누리

新仏山（シンブルサン）は、蔚山広域市と慶尚南道の境に位置する山で、ネジ山・干月山（カンウォルサン）・英 축山（ヨンチュクサン）とともに、いわゆる「嶺南アルプス」を形成する代表的な峰です。 標高については、案内板によれば1159m前後とされていますが、GPSの測定値ではコースによって1170〜1220m程度の値が出ることもあります。これは、山頂の石碑の位置や周囲の峰（干月山1086m、ペネボン993mなど）との測定地点の差によるものと思われます。

山全体としては、なだらかな稜線とススキの原、そして所々に現れる岩場が調和しており、「土の山」と「岩の山」の中間のような趣があります。難易度は全体的に「中級」といったところですが、コースによっては片道6時間にも及ぶ長距離もありますので、事前の体力配分が肝心です。 多くのコースは日帰りで楽しめますが、全行程（総延長73.11km）を縦走するとなると、一泊以上の計画を立てたほうが良さそうですね。

山の地形と特徴

写真: 韓国観光公社 観光写真ギャラリー (蔚山広域市 鬱州郡 上北面) · 公共ヌリ

信不山（シンブルサン）は、なだらかな稜線が続く樹林主体の地形と、山頂付近で見られる岩場の表情が共存しているのが特徴です。山頂から西へ続く稜線は、広く「信不山ススキ原」として知られる広大な草原地帯が広がっており、秋になると一面のススキが揺れる絶景スポットとして人気を博します。山頂付近には岩肌が露出した場所もあり、視界が開けている一方で、谷筋や斜面の下部には豊かな森が広がる、典型的な樹林の山としての魅力も持ち合わせています。主なピークとしては、干月山（1086m）、背内峰（993m）、そして信不山頂（約1170〜1220m付近の複数地点）があり、これらが稜線でつながることで、信不山から干月山へと続く縦走コースの骨格を形成しています。

さて、このエリアの登山道は行政区画に基づいて計6つの区間に登録されています。これらはあくまで行政上の区分であり、特定の「登山コース名」ではないので、個別のルートと混同しないよう注意が必要です。以下は、登録されている区間ごとの総延長です。

登山の感想

写真：韓国観光公社 観光写真ギャラリー（蔚山広域市蔚州郡尚北面） ・ 公共누리

申仏山（シンブルルサン）の登山は、入り口の鬱蒼とした森を抜けていくと、次第に視界が開けた稜線へと続いていく流れがとても心地よいんです。

麓の方は渓谷沿いの道が多く、夏場は木陰と川のせせらぎを楽しみながら歩けます。一方で山頂に近づくにつれて、岩場やススキの原が交互に現れ、四方へと広がる絶景が次々と目に飛び込んできます。特に山頂付近の稜線から眺める英南アルプス一帯はまさにパノラマ。かなりの体力を削られますが（笑）、その分、頂上に立った時の達成感はひとしおです。

春は新緑やツツジ、秋はススキと、季節ごとに鮮やかな表情を見せてくれるのがこの山の魅力。ただ、冬は稜線の強風や凍結した道もあるので、初心者の方には少しハードかもしれません。

コース1 — トイレから山頂へ

📏 16.99km ⛰️ 1041m 獲得標高 ⏱️ 5:59 🔁 周回 難易度 ★★★★★ ⤓ GPX

経由地: トイレ → 観光スポット → 水飲み場 → 休憩所 → 案内板 → 駐車場 → 自然景観 → 分かれ道13 → 分かれ道17 → 案内所 → 展望台 → 分かれ道20 → 分かれ道21 → 分かれ道23 → 分かれ道24

登山口へのアクセス: 入り口付近にトイレや駐車場、案内所が揃っているので、しっかり整備された登山口のようです。公共交通機関を利用する場合は、最寄りのバス停から歩いて入り口へ向かい、そこからは案内表示に従って進むのが一般的なルートになります。正確なバス停の名前や運行本数は、事前に地図アプリなどで再確認しておいた方が安心ですね。

特徴・おすすめ: 全行程の中で最も距離がある、約17kmの長丁場コースです。正直、これだけの距離を歩くとなると少し気が遠くなりますが（笑）、道中に水飲み場や休憩所、案内所などが点在しているので、完走に時間がかかったとしても、こまめに体を休める場所を見つけやすいのが救いです。体力に自信があり、「今日は一日がかりで山歩きを楽しむぞ！」という意欲のある方にぴったりです。当日往復するなら、早朝出発を強くおすすめします。

コース 2 — 登山口 → 山頂

📏 12.27km ⛰️ 992m 獲得標高 ⏱️ 4:43 🔁 周回 難易度 ★★★★★ ⤓ GPX

経由: 新仏山（シンブルサン）トンネル（工事終盤区間） → 新仏寺 → 危険箇所 → 分岐点6 → 渓谷 → 分岐点9 → 自然景観 → 安全構造板 → 展望スポット → 分岐点27 → 福雲寺 → 登山口

アクセス: 新仏寺や福雲寺といったお寺を経由するルートなので、ペネゴル方面からの入り口と思われます。車でのアクセスは比較的スムーズな区間ですが、公共交通機関を利用する場合は、近隣の村までバスで行ってから歩いて向かうのが一般的なルートです。

特徴・おすすめの方: 渓谷、お寺、眺望ポイントと、景色がドラマチックに変わる変化に富んだコースです。ただ、ルート上に「危険」という表記がある通り、雨天時や凍結する時期には十分な注意が必要です。お寺の文化や渓谷の情緒を楽しみながら歩きたい方にぴったりです。

📏 12.61km ⛰️ 857m 獲得標高 ⏱️ 4:35 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 有料駐車場 → 自然景観 → トイレ → 観光スポット → カフェ → 食堂 → バス停 → 入口 → 休憩所 → 分かれ道10 → 展望ポイント → 石頂峰山頂

アクセス: 有料駐車場やカフェ、飲食店、バス停がひとつの場所に集まっていることから、おそらく等億（トゥンオク）アルプス温泉団地付近の入り口だと思われます。新仏山（シンブルサン）のコースの中では、車や公共交通機関でのアクセスが非常にスムーズな部類に入ります。

特徴・おすすめ: 観光施設が整った入り口から山道へと移り変わっていく構成です。家族連れの方や、「ちょっと軽く歩いてみたい」という旅行者でも挑戦しやすいでしょう。ただし、展望ポイントを過ぎてから山頂までは本格的な登り坂が続くので、最後まで登り切るにはしっかりとしたペース配分が必要になります。

登山口へのアクセス: 駐車場やカフェが隣接しており、車でのアクセスは非常にスムーズな登山口といえそうです。近くに釣り場があることから、貯水池や渓流沿いの集落へと続く入り口のような場所だと推測されます。

特徴・おすすめ: 片道5.5km、累積標高差899m。距離の割に傾斜がかなり急で、なかなかのハードなコースです。ヘリポートや展望ポイントを通るので眺望は抜群ですが、「立入禁止」の表示がある区間もあるため、現地の通行規制を必ず確認してから進むようにしてくださいね。

コース5 — 休憩所 → 山頂

📏 2.87km ⛰️ 777m 獲得標高 ⏱️ 2:01 ➡️ 片道 難易度 ★★★★☆ ⤓ GPX

経由: 休憩所 → 展望ポイント → 自然景観 → 分岐点19

アクセス: 道しるべがシンプルなので、既存の登山道から山頂へ向かって一気に駆け上がるような、短い接続区間だと思われます。専用のバス停などの情報は見当たらないため、他のコースと組み合わせて歩くのが一般的です。

特徴・おすすめ: 片道2.87km、標高差は777m。全コースの中で最も距離こそ短いですが、傾斜はかなり急な部類に入ります。「短時間でサクッと山頂からの景色を楽しみたい」という方には最適ですが、急斜面が多いので、下山時の膝への負担や滑りやすさにはくれぐれもご注意を。

コース 6 — 安全救助木 → 山頂

📏 7.88km ⛰️ 901m 獲得標高 ⏱️ 3:28 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

登山口へのアクセス: 登山口の名前が「安全救助木」と記録されているため、正式な地名ではなく現地の目印のような扱いかもしれません。実際の入り口が分かりにくい可能性があるため、事前に現地の案内図や地図アプリなどで正確な進路を確認しておくのが安全です。

特徴・おすすめの対象: 道標に「危険」や「険道」といった言葉が繰り返し登場するように、コース全体を通して路面が荒れており、注意が必要な区間が多くあります。のんびりとしたハイキングというよりは、ある程度山歩きの経験がある方向けのコースです。雨上がりや路面が凍結している時期などは、無理をせず避けるのが賢明です。

難易度・路面・注意点

사진: 한국관광공사 관광사진 갤러리 (울산광역시 울주군 상북면) · 공공누리

林野庁の登山道データによれば、信不山（シンブルサン）内のコースにおける最高難易度は「中程度」とされています。ただ、100名山のGPX実測データを確認すると、山頂付近のコースは累積標高差が800mを超えたり、片道で3〜6時間ほどかかる体力勝負な区間も。行政上の「中程度」という表記を鵜呑みにせず、実際はそれよりもハードなコースになることを念頭に置いておいたほうが良さそうです。

施設統計上は、展望スポットや水飲み場、避難小屋、危険箇所などはすべて「0件」として集計されており、公式に登録された利便施設や安全施設は確認できません。しかし、GPXのウェイポイントを見ると、コース2、3、4、6で「展望点」や「危険」といった表記が繰り返し現れます。これは公式のデータベースには反映されていなくても、現地に案内板などが設置されている可能性を示唆しています。 山頂付近の稜線や岩場が露出している区間では、雨天時の滑りやすさや冬場の凍結・強風には特に注意が必要です。「険しい道」や「危険」の表示がある箇所では、決して無理をして進まず、安全第一で行動しましょう。

距離と所要時間

出典：韓国観光公社 観光写真ギャラリー（蔚山広域市鬱州郡尚北面） · 公共누리

エリア内の全登山道を合算した累積所要時間は、登りが1389分、下りが962分となっています。もちろん、これ一回で歩けるわけではなく、登録されている全区間をすべて足し合わせた数値なのですが……。これを距離に換算すると平均ペースは約19分/kmほどになり、信不山（シンブルサン）の登山道がいかに登りメインで、決して楽な道のりではないかがよく分かりますね。

実際のコースごとの所要時間は、先に提示したGPXの実測値を参考にするのが一番確実です。目安として、アクセスが良い「入口 → 石頂峰（ソクジョンボン）山頂」コースは片道で約4時間35分、最も短い「休憩所 → 山頂」コースでも2時間前後かかります。

ほとんどのコースは往復しても日帰りで十分可能ですが、全行程73.11kmをすべて繋いで歩く「縦走」に挑むなら、安全のために1泊以上の行程で計画することをおすすめします。

アクセス（公共交通機関・駐車場）

写真：韓国観光公社 観光写真ギャラリー (蔚山市鬱州郡上北面) · 公共누리

新仏山（シンブルサン）周辺には、通渡寺正門、水精法福、長済会館、白蓮村、太峰村、長済村など、いくつかのバス停があります。ヤンサン（梁山）方面から登り口へ向かう際は、この「通渡寺正門」停留所を一つの目安にすると分かりやすいでしょう。蔚山（ウルサン）市内やヤンサン方面から向かう場合は、まずバスでこれらの停留所まで移動し、そこから各コースの登山口へと歩いて進むのが一般的なルートです。ちなみに、コース3のドングク・アルプス温泉団地方面へ向かうなら、温泉団地行きのバスを利用するのが一番スムーズですよ。

公式なデータ上では駐車場は「0箇所」と集計されていますが、これはあくまで林務庁の登山道情報に基づいたもの。実際には温泉団地や寺院の周辺に、有料駐車場がいくつか運営されています（コース3の案内標識にも「有料駐車場」の表記がありました）。正確なバスの時刻表やリアルタイムの経路を確認するには、KakaoMapかNaver Mapで目的地を検索することをお勧めします。現地の公共交通機関や徒歩ルートの案内に関しては、この2つのアプリがかなり正確で頼りになりますからね。

シーズン・天気・装備

写真：韓国観光公社 観光写真ギャラリー（蔚山広域市 울주군 상북면）

登山に最適なシーズンは、新緑やツツジが楽しめる春（4〜5月）か、稜線のススキが美しい秋（9〜10月）です。 夏場は谷間の湿度や暑さ、そして急な夕立への備えが欠かせません。また、水場が確認できないため、飲み水は十分に持参しましょう。 冬は稜線の強風や路面の凍結に注意が必要です。アイゼン、スパッツ（ゲイター）、防風ジャケットは必須の装備といえます。一部のコースには「険路」や「危険」の表示もあるため、初心者が冬に単独で登るのは少し控えたほうが賢明かもしれません。 基本装備としては、登山靴、着替え、ヘッドライト、非常食、そして地図アプリを準備しておくことをおすすめします。

写真: 韓国観光公社 観光写真ギャラリー (蔚山広域市ウル郡尚北面) · 公共누리

こちらの山は、英語表記では「Sinbulsan」、山頂は「Sinbulsan-jeongsang (Sinbulsan Peak)」、近隣の峰は「Ganwolsan」や「Baenaebong」と記されます。 入山許可や入場料などはかからないようですが、お寺の境内（新仏寺や福雲寺など）を通る際は、参拝のマナーを守って静かに進みましょうね。 設備についてですが、トイレは一部の登山口（コース1と3）にありますが、山頂にはありません。出発前に済ませておくことを強くおすすめします。また、公式な水飲み場はないので、飲み水は必ず各自で持参するようにしてください。 それから、ちょっとしたコツですが、韓国の登山道はGoogleマップだと情報が不正確なことがよくあります。ルートを確認するなら「KakaoMap」か「Naver Map」を使うのが一番確実ですよ。 また、稜線のあたりでは電波が不安定になることもあるので、事前にルートをオフラインで保存しておいたり、同行者や周囲の人に下山予定時刻を伝えておいたりするなど、安全対策は忘れずに。

周辺の山々

  • 石柱が屏風のように連なる光州・無等山 — 直線距離 約186km
  • のこぎりの刃のような八峰の稜線と連珠台の岩峰、ソウル・冠岳山 — 直線距離 約280km
  • 都心のすぐそばにそびえる花崗岩の稜線、ソウル・北漢山 — 直線距離 約297km

📍 ススキの原野と岩稜が織りなす絶景、蔚山・新仏山（シンブルサン） 위치
