概要

写真：韓国観光公社 観光写真ギャラリー（蔚山広域市 蔚州郡 上北面）・公共ヌリ

新佛山（シンブルサン）は、蔚山広域市の蔚州郡と慶尚南道の梁山市の境界にまたがる山で、加智山・肝月山・霊鷲山とともに、いわゆる「嶺南アルプス」の稜線をかたちづくる代表的な峰のひとつです。山頂部の標高は案内板の表記でおおむね1159mほどとされていますが、実測のGPSデータではコースごとに1170〜1220mあたりの値が出ています。これは、山頂の標識石の位置と、近くの峰（肝月山頂1086m、ペネ峰頂993mなど）を測った地点の違いから来ているようです。山全体としては、なだらかな稜線とススキの平原、そしてところどころの岩稜が入りまじった性格を持っていて、土の山と岩の山のちょうど中間くらい、といった評価をされています。難易度は全体としては「中程度」ですが、コースによっては片道6時間におよぶ長丁場の区間も含まれているので、体力の配分には気をつけたいところです。ほとんどの山頂コースは日帰りでこなせますが、管内の登山道すべて（総延長73.11km）を縦走しようとなると、一泊以上の日程を考えておく必要があります。

山並みと地形

写真: 韓国観光公社 観光写真ギャラリー（蔚山広域市 蔚州郡 上北面）· 公共ヌリ

神仏山は、なだらかな尾根が続く土山らしさと、山頂まわりにむき出しの岩稜が現れる、その両方をあわせ持った山です。山頂から西へと延びる尾根は、俗に「神仏山ススキ平原」と呼ばれる広々とした草地が広がる区間として知られていて、秋にはこのススキ群落が代表的な絶景ポイントに数えられます。山頂部には岩盤が顔を出したところが点在していて眺めがぐっと開ける一方、谷筋や斜面の下のほうは樹林がうっそうと茂る、いかにも土山らしい姿を見せてくれます。空間情報に登録された主な峰としては、艮月山頂上（1086m）、梨内峰（993m）、神仏山頂上（およそ1170〜1220m台の複数地点）などがあり、これらの峰が尾根伝いにつながって、神仏山〜艮月山の縦走コースの骨格をなしています。

管内の登山道は行政洞を基準に全6区間として登録されていますが、これは登山道の名前ではなく、登山道が通る行政区域の単位で分けた値なので、個々のコース名と取り違えないよう注意が必要です。以下は、登録された区間ごとの総延長の一覧です。

등산의 느낌

사진: 한국관광공사 관광사진 갤러리 (울산광역시 울주군 상북면) · 공공누리

신불산은 登り始めこそうっそうとした森の道が続きますが、進むにつれて徐々に視界が開け、尾根へとつながっていく——そんな流れが特徴の山です。下のほうは沢沿いを歩く区間が多く、夏場は木陰と水の音を一緒に楽しめます。そして頂上に近づくほど、岩場とススキの原が代わる代わる現れ、四方に開けた眺めを見せてくれます。とくに頂上付近の尾根からは、嶺南アルプス一帯の山々がパノラマのように広がっていて、体力の消耗が大きいぶん頂上での「登ってよかった」という充実感も大きい、というのが一般的な評判です。春は新緑とツツジ、秋はススキと、季節の移ろいがくっきり感じられる山として知られていますが、冬は尾根の刺すような風と凍結区間が現れるので、初心者にはなかなか手ごわいかもしれません。

코스 1 — トイレ → 山頂

📏 16.99km ⛰️ 1041m 獲得標高 ⏱️ 5:59 🔁 周回 難易度 ★★★★★ ⤓ GPX

経由: トイレ → 観光資源 → 水場 → 休憩所 → 案内図 → 駐車場 → 自然景観 → 分岐13 → 分岐17 → 案内所 → 展望 → 分岐20 → 分岐21 → 分岐23 → 分岐24

登山口へのアクセス: 入口にトイレ・駐車場・案内所がまとまって置かれているところを見ると、きちんと整備された進入路のある登山口だろうと思われます。公共交通機関を使う場合は、近くのバス停から歩いてアプローチし、道標に従って入っていくのが一般的なパターンでしょう。正確なバス停名や運行間隔は、出かける前に地図アプリでもう一度確かめておくと安心です。

特徴・おすすめの人: 全コースの中でいちばん長い片道距離（約17km）を誇るロングコースです。水場・休憩所・案内所といった中間の施設が比較的あちこちに配置されているので、完歩に時間がかかっても休みどころを確保しやすいのはありがたいところ。体力に自信があって、まる一日を山歩きに充てられる方にぴったりで、日帰りで往復するなら早朝出発をおすすめします。

コース2 — 登山口 → 山頂

📏 12.27km ⛰️ 992m 獲得標高 ⏱️ 4:43 🔁 周回 難易度 ★★★★★ ⤓ GPX

経由: シンブルサン（新佛山）トンネル（工事の大詰め区間）→ シンブル寺 → 危険 → 分岐6 → 渓谷 → 分岐9 → 自然景観 → 安全救助板 → 眺望点 → 分岐27 → ボグン寺（福雲寺）→ 登山口

登山口へのアクセス: シンブル寺やボグン寺といったお寺を経由することから、ペネ谷（배내골）方面からの進入路ではないかと推測されます。マイカーでのアクセスが比較的しやすい区間として知られていますね。公共交通機関を使う場合は、近くの集落を通るバスに乗ってから、徒歩で向かうルートが一般的です。

特徴・おすすめの人: 渓谷にお寺、眺望点とバランスよく揃っていて、風景の移り変わりがはっきりしているコースです。ただ、道標に「危険」という表示が出てくるくらいなので、雨の日や凍結する時期にはやっぱり注意が必要ですね。お寺の文化と渓谷の風情、その両方をまとめて味わいたいという方にぴったりだと思います。

コース3 — 入口 → 石井峰（ソクチョンボン）山頂

📏 12.61km ⛰️ 857m 獲得標高 ⏱️ 4:35 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 有料駐車場 → 自然景観 → トイレ → 観光資源 → カフェ → 食堂 → バス停 → 入口 → 休憩所 → 分岐点10 → 展望地点 → 石井峰山頂

登山口へのアクセス: 有料駐車場・カフェ・食堂・バス停がひとかたまりに集まっているところを見ると、トゥンオクアルプスリ温泉団地の近くにある進入路ではないかと推測されます。神仏山（シンブルサン）のコースの中でも、公共交通機関・車どちらのアクセスもいちばん良い部類に入りそうです。

特徴・おすすめの人: 観光インフラがしっかり整った登り口を抜けて山道へとつながっていく造りなので、家族連れの方や、山行の入口だけ軽く歩いてみたい旅行者でも気軽に入っていけます。ただし、展望地点を過ぎて石井峰山頂までは本格的な登りが続くので、頂上まで歩き通すつもりなら体力の配分が必要です。

コース4 — 登山口 → 高丈山（コジャンサン）頂上

📏 5.54km ⛰️ 899m 獲得標高 ⏱️ 2:53 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 駐車場 → カフェ → 休憩所 → 自然景観 → 危険箇所 → 展望ポイント → ヘリポート → 分岐40 → 分岐41 → 立入禁止 → 案内図 → 高丈山頂上 → 登山口 → 釣り場

登山口へのアクセス: 駐車場とカフェがすぐそばにあり、車でアクセスしやすい登山口のようです。近くに釣り場も一緒に記されているところを見ると、貯水池や渓流の近くにある集落の入口あたりではないかと思われます。

特徴・おすすめの人: 片道5.5km、標高差は899mと、距離のわりに勾配がきつめのコースです。ヘリポートや展望ポイントを通る区間があるので眺めは楽しめますが、「立入禁止」の表示が出てくる区間もあるので、通行止めになっていないかは現地で必ず確認しておきたいところです。

コース5 — 休憩所 → 山頂

📏 2.87km ⛰️ 777m 獲得標高 ⏱️ 2:01 ➡️ 片道 難易度 ★★★★☆ ⤓ GPX

経由: 休憩所 → 展望地点 → 自然景観 → 分岐19

登山口へのアクセス: 道標の構成がシンプルなことから、既存の登山道の中間地点から山頂へ一気に登り上げる、短い接続区間だと思われます。これといったバス停などの公共交通の情報は見当たらないので、ほかのコースとつなげて使うのが一般的でしょう。

特徴・おすすめの人: 片道2.87km、標高差777mと、全コースのなかでいちばん距離は短いのですが、勾配はかなり急な部類に入ります。短時間で山頂からの眺めを楽しみたい方にはうってつけですが、急な斜面が多いので、下りは膝への負担と滑りにじゅうぶん気をつけてください。

コース6 — 安全구족목 → 山頂

📏 7.88km ⛰️ 901m 獲得標高 ⏱️ 3:28 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由: 安全구족목 → 分岐21 → 危険 → 安全救助木・救急箱 → 自然景観 → 分岐25 → 分岐27 → 眺望点 → 安全救助木 → 険路 → 自然景観

登山口へのアクセス: 登山口の名前が安全施設物（安全구족목）として記録されているだけで、正式な地名が確認できない接続路のようです。実際に足を運ぶ前に、現地の登山道案内図や地図アプリで進入地点をきちんと確認しておくのが安心です。

特徴・おすすめの方: 道標に「危険」「険路」が何度も出てくるだけあって、コース全体を通して路面が荒れていて、気を抜けない区間が多めです。のんびりした散策というより、ある程度の登山経験がある方向けですね。雨上がりの直後や凍結する時期は、避けておくのが無難です。

難易度・路面・注意点

写真: 韓国観光公社 観光写真ギャラリー（蔚山広域市 蔚州郡 上北面）· 公共ヌリ

山林庁の登山道空間情報によると、新不山エリアの登山道の最高難易度は「中級」で、路面の材質については登録がありません。ただ、100名山のGPX実測データを見ると、山頂付近のコースの多くが累積標高差800m以上、片道3〜6時間台という体力の消耗が大きい区間になっていて、行政上の登録では「中級」となっていても、実際に歩いて感じる難易度はそれより高いかもしれない、という点は頭に入れておいたほうがよさそうです。

施設の統計を見ると、展望地点・水飲み場・避難小屋・危険区間がすべて0か所と集計されていて、公式に登録された休憩・安全施設は確認できませんでした。とはいえGPXの道標名を見ると「展望点」や「危険」の表示がコース2、3、4、6に繰り返し出てきます。これは正式な施設データベースには反映されていないものの、現地には案内板の形で存在している可能性を示しています。山頂付近の稜線や岩盤がむき出しになった区間では、雨の日の滑りや、冬場の凍結・強風にとくに気をつけたいところ。「険路」「危険」の表示がある区間では、無理に進もうとしないのが賢明です。

거리와 소요시간

사진: 한국관광공사 관광사진 갤러리 (울산광역시 울주군 상북면) · 공공누리

管内の登山道は、全区間を合計すると上りが1389分、下りが962分と集計されています。これは個々のコースの所要時間ではなく、管内に登録された区間すべてを足し合わせた数字です。これを距離に換算すると、上りの平均ペースはおよそ19分/kmほど。神仏山の登山道は全体的に、なだらかとは言えない登り主体の区間だということが分かります。実際のコース別の所要時間は、先ほど挙げたGPX実測ベースの表を参考にするのが正確でして、たとえばアクセスのよい入口 → 石井峰（석정봉）頂上コースは片道でおよそ4時間35分、いちばん短い休憩所 → 頂上コースは2時間前後といったところです。ほとんどのコースは往復でも日帰り登山が可能ですが、管内の総延長73.11kmをすべてつなぐ縦走登山となると、1泊以上の日程で計画したほうが安全です。

아쿠세스（교통·주차장）

사진: 한국관광공사 관광사진 갤러리 (울산광역시 울주군 상북면) · 공공누리

シンブル山（신불산）の周辺には、通度寺正門（통도사정문）、水晶法服（수정법복）、長堤会館（장제회관）、白蓮村（백련마을）、太峰村（태봉마을）、長堤村（장제마을）など、いくつものバス停が登録されています。このうち通度寺正門のバス停は、梁山（양산）方面から登山口へ向かうときの目印として使えます。ウルサン市内や梁山方面からだと、市外バスや市内バスでこのバス停まで来て、そこから各コースの登山口まで歩いていくのが一般的なパターンですね。トゥンオク・アルプスリ温泉団地（등억알프스리 온천단지）方面（コース3）へ向かう場合は、温泉団地行きのバス路線を使うのが便利だとされています。

公式データ上は登録されている駐車場が0か所となっているのですが、これはあくまで山林庁の登山道空間情報を基準にした数字で、実際には温泉団地やお寺の近くに有料駐車場があるようです（コース3の道標に「有料駐車場」の表記が確認できました）。正確なバスの時刻表やリアルタイムのルートについては、KakaoMapかNaver Mapで目的地を検索して確認するのがおすすめです。この2つのアプリは、どちらも韓国現地の公共交通や徒歩ルートの案内がより正確ですから。

시즌·날씨·장비

写真：韓国観光公社 観光写真ギャラリー（蔚山広域市 蔚州郡 上北面）· 公共ヌリ

ハイキングに向いているシーズンは春（4〜5月）と秋（9〜10月）で、春は新緑とツツジ、秋は稜線を彩るススキが見どころです。夏場は谷沿いの蒸し暑さや、急な夕立に備えておく必要があり、給水場がとくに見当たらないので飲み水はたっぷり持っていきましょう。冬場は稜線の強風や凍結箇所が出ることがあるので、アイゼン・スパッツ・防風ジャケットは必須といっていい装備です。一部のコースには「険路」「危険」の表示もあるので、初心者は冬の単独登山は避けたほうが無難でしょう。基本装備としては、登山靴、着替え、ヘッドランプ、非常食、地図アプリをそろえておくのがおすすめです。

外国人旅行者向けのヒント

写真: 韓国観光公社 観光写真ギャラリー (蔚山広域市 蔚州郡 上北面) · 公共ヌリ

신불산（新佛山）はローマ字では「Sinbulsan」と表記します。山頂は「Sinbulsan-jeongsang（Sinbulsan Peak）」、近くの峰である간월산は「Ganwolsan」、배내봉は「Baenaebong」と併記できます。入山許可や別途の入場料はかからないようですが、寺院の境内（신불사・복운사など）を通り抜ける場合は、参拝のマナーを守るようにしたいところです。トイレは一部の登山口（コース1、3）で確認できますが山頂にはないので、出発前に済ませておくのがおすすめ。給水所も公式には登録されていないため、飲み水は各自で持参するのが必須です。グーグルマップは韓国国内の登山道情報が不正確なことが多いので、現地の実際のルート確認にはKakaoMapかNaver Mapを使うのがおすすめです。稜線には電波が届きにくい区間もあるため、あらかじめルートをオフラインで保存し、同行者や下山予定時刻を周囲に共有しておくと安心です。
