上岩洞からワールドカップ大橋経由で漢江ハートコース76km

自転車 · 公開日 2024/5/11 ·

상암（サンアム）洞から漢江へ出るルートは、大きく分けて2つあります。1つ目は不光川（プルグァンチョン）を通って漢江に合流するルート。たいていの方はこちらで移動されますし、僕もよくこの道を使っていました。でも今は、ワールドカップ大橋を使って漢江に入ると、とても快適に走れるんです。もう工事区間もありません。

ここではMBCを基準にご説明しますね。

ワールドカップ競技場を見ながら右折して、そのままワールドカップ大橋の方へ進みます。すると信号のない横断歩道が1つ出てきます。そこを渡れば、ワールドカップ大橋への進入成功です。ワールドカップ大橋の東側を渡るには信号を2つ通らなければなりませんが、西側は北端に信号のない横断歩道が1つ、そして少し先に信号のある横断歩道が1つあるだけ。南端からはもう横断歩道もなく、エレベーターで漢江のサイクリングロードへそのまま入れます。

僕は朴正煕（パク・チョンヒ）図書館の方から公園の道に沿ってワールドカップ競技場まで進みました。そして、ワールドカップ大橋から眺めた一山（イルサン）方面の江辺北路がこちら。あとはそのまま橋を渡っていくだけです。昨日のように強風が吹く日は、橋を渡るのがなかなか大変。いつでもスピードを落として、路面をよく見ながら渡ってくださいね。

Previous image Next image

ワールドカップ大橋をそのまま進んでいくと、安養（アニャン）、江辺北路（蚕室方面）、内部循環路へ向かう標識に出会います。そこでボタンを押して、もう一度横断歩道を渡ります。すると漢江の南側サイクリングロードまで、もう信号はありません。2枚目の写真は横断歩道から상암洞を眺めたところ、3枚目は橋を渡り切ったあとに상암洞を振り返ったところです。

Previous image Next image

いよいよ安養川の合流地点に到着です。iPhoneで撮った写真なのに、天気がいいおかげでコントラストがなかなか見事。皆さんがよく使う待ち合わせスポットですね。^^ 今日はハートコースを走ります。ブロンプトンTライン（Brompton T Line）を買って、たぶん最長距離になりそうです。水をひと口飲んで、出発します。

コースは、サイクリングロードに沿って漢江の東側へひたすら進みます。良才川（ヤンジェチョン）に出たらそこで良才川へ折れて進み、良才川のサイクリングロードが終わると、そこは果川（クァチョン）。果川で一度外に出て、一般道を走って移動します。仁徳院（インドグォン）駅の交差点で学衣川（ハギチョン）方面へ、普通の街中の道を進み、学衣川に入ってそのまま進むと、双渓（サンゲウル）で安養川と合流します。ここからは安養川なので、休まず進めば、再びスタート地点である安養川の合流地点に戻ってこられます。

良才川から学衣川にかけての区間はGTXをはじめいろいろな土木工事が多く、周りでもマンションをさらに建てているので、あまり快適ではありませんが、30分ほどで通過できるので、欲張らず歩道とうまく組み合わせて進めば、特に問題はありません。

瑞草（ソチョ）区を抜けて果川洞です。向かい風があまりにひどくて、しばらく休憩しました。同行の方が前で引っ張ってくれて、なんとか進めたほど。体力のある方なのに、風がひどいときは時速20キロを切ることもありました。僕なら時速15キロ以下まで落ちていたでしょうね。遠くに冠岳山（クァナクサン）が見えていて、その冠岳山の麓を回り込んでいるところです。

Previous image Next image

初めて訪れた双渓（サンゲウル）の休憩所です。2つの川が1つに合わさる区間で、自転車乗りのための立派な休憩スポットが整備されていました。風は強かったものの、天気は本当に最高でした。

Previous image Next image

普段はガーミンのエンデューロ2（Garmin Enduro 2）を着けています。ガーミンは僕の体力の限界をちゃんと把握しているんですよね。1年以上着けてきたわけですから……。どうか表示された体力以上に残ってくれますように、と願いながら自転車を走らせました。でも、その数値は思った以上に正確でした。残り1%未満と表示されたときは、30分走るごとに5分ずつ休みました。機器を盲信する必要はありませんが、指標があるに越したことはないですね。のんびりと、安養川の合流地点に再び到着しました。ガーミンVaria 715（Garmin Varia 715）のバッテリーは50%以下まで減りましたが、5〜6時間の走行は持ちこたえてくれました。特にモードは気にせず走り、たぶんドライブレコーダーも保存されているはず。日中も一般道を走ったので点滅モードで進み、車が近づくと強くフラッシュする設定にしておきました。

ここから상암洞へ戻るにはワールドカップ大橋を渡らないといけません。いつもどおりだと、エレベーターで上がった分だけまた下りて……結局また上る、という流れになるのですが、今回は出発したときと同じく西側から上がれば、稼いだ高度を損することなく、むしろ少し下り坂で家まで帰ってこられます。

Previous image Next image

無事に帰還しました。상암洞は、洞全体のサイクリングロードが全国どこよりもよく整備されています。今日の走行距離は合計76km、GPSログ上ではハート形になりました。これで初めてハートコースを走破できたわけです。お尻がちょっと痛くなったのと、果川区間は一般道、しかも工事区間がかなりあって少し窮屈でしたが、工事が終わって良才川と学衣川の区間をサイクリングロードでつなぐコースが快適になれば、人気の中距離コースとして多くのサイクリング仲間に使われるようになりそうです。
