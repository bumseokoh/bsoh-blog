概要

写真: 韓国観光公社 観光写真ギャラリー（大邱広域市軍威郡扶渓面） ・ 公共누리

パルコンサン（八公山）は、大邱広域市の東区を中心に、軍威郡・永川市・慶山区にまたがる大邱近郊を代表する名山です。 標高については、実測値では最高地点が約1,204mとされていますが、主峰である毘盧峰（ピロボン）を1,220mとする資料もあり、データの出所によって多少の差が見られます。 地形は、なだらかな土の斜面とゴツゴツとした花崗岩の岩場が組み合わさった変化に富んだ山容が特徴です。山頂を目指す代表的なコースは片道でおよそ4.5〜8.7km、所要時間は2時間15分から3時間20分ほど。大抵の場合は日帰りの登山で十分に楽しめます。 コースによって難易度がかなり変わるのもポイントで、のんびりと歩ける巡礼路のような緩やかな道から、岩場や絶壁が続く本格的な上級コースまで、その日の気分や体力に合わせて幅広く選ぶことができます。

山の地形について

写真: 韓国観光公社 観光写真ギャラリー (大邱広域市 軍威郡 釜渓面) · 公共누리

八公山（パルコンサン）は、土の稜線が続く典型的な山の地形の中に、花崗岩の岩峰や岩場が混ざり合ったダイナミックな地形をしています。 山頂付近の情報によると、主峰である毘盧峰（ビロボン、1,220m）を中心に、東峰（1,193〜1,194m、また八公山東峰として1,189mと表記されることもあります）や西峰が稜線でつながっており、そのほかにも閻彿峰（1,133m）、新林峰（828m）、巨底山（502m）といった数々の峰々が美しい山容を形作っています。 登頂コースのデータを見ても、「ラクダ峰」や「奇岩石」、岩壁地帯といった岩場を指す地名がいくつも登場することから、稜線にはあちこちに岩場が露出していることが分かります。また、山裾には東華寺やガッバウィ（冠峰）といった歴史ある仏教遺跡も点在しており、自然の地形が宗教や歴史的な景観と深く結びついているのも、この山の大きな魅力ですね。

登山の醍醐味

사진: 한국관광공사 관광사진 갤러리 (대구광역시 군위군 부계면) · 공공누리

パルコン山の登山は、選ぶコースによってその表情がガラリと変わります。 浄化寺（トンファサ）などの寺院を抜ける序盤の区間は、松や広葉樹が茂る緩やかな森道で、心地よい木陰が続きます。ところが稜線に近づくにつれて花崗岩の岩場や展望スポットが交互に現れ、パッと視界が開けていくんです。 『元暁求道の道』のように、思索しながら歩くことに重きを置いた区間は、足を洗う場所やあずまや、趣のある案内板などが整備されていて、本格的な登山というよりはどちらかと言えばトレッキングに近い感覚で楽しめます。 その一方で、『カルタックコゲ』のような急斜面や岩場を含むコースは、一気に険しさを増します。落石や断崖への注意看板も目に入り、かなり緊張感のある山行になりますね。 季節の移ろいも楽しみの一つです。春の新緑やツツジ、夏の爽やかな渓流の音、そして大邱（テグ）を代表する紅葉の名所として知られる秋の美しい稜線、そして冬の雪景色や樹氷……。いつ訪れても、また違った表情を見せてくれます。

コース1 — バス停 → カルタックコゲ（急坂）山頂休憩所

📏 8.67km ⛰️ 721m 獲得標高 ⏱️ 3:22 ➡️ 片道 難易度 ★★★★★ ⤓ GPX

経由地: バス停 → 分かれ道 → トイレ → カルタックコゲ山頂休憩所 → 岩場 → 落石注意エリア → 岩の上の展望ポイント → シンリム亭 → ナクタブォン → 落石注意エリア → 展望ポイント → 磨崖薬師如来坐像への入り口 → 八公山（パルコンサン）磨崖薬師如来坐像 → 岩の隙間の木々 → デッキ階段の上の展望ポイント

登山口へのアクセス: 登り口がバス停のすぐ目の前にあるので、公共交通機関をメインにしたコースです。大邱（テグ）市内のバスが八公山方面へいろいろな路線で走っているので、正確な停留所や路線の確認はKakaoMapやNaver Mapをぜひ活用してみてください。

特徴・おすすめの人: ナクタブォンや八公山（パルコンサン）の磨崖薬師如来坐像といった、仏教文化遺産を巡るコースです。落石注意の表示が2回出てくるので、雨上がりなどは特に気をつけないといけません。コース名にある「カルタックコゲ（息が切れるほどの急坂）」の名の通り、延々と傾斜が続くハードな道。体力に自信があるハイカーさん向けのコースといえるでしょう（笑）。

コース 2 — 元暁求道（ウォニョ・クド）の道 入口 → 空の庭展望台

📏 7.08km ⛰️ 621m 獲得標高 ⏱️ 2:48 🔁 周回 難易度 ★★★★☆ ⤓ GPX

経由: トイレ → 元暁求道の道 総合案内 → 分かれ道 4 → 元暁求道の道 入口 → 休憩所 → 看板 → 足洗い場 → 現在地看板 → 碑文 → 青雲亭 → 眺望点 → 仮トイレ → 元暁が歩んだその道の上に立ちて → 分かれ道 17 → 悟道庵（オドアム）入口

アクセス: 「元暁求道（ウォニョ・クド）の道」は、八公山（パルコンサン）の北側に整備された、巡礼路のような趣を持つトレイルです。公共交通機関よりも自家用車でのアクセスが一般的なようです。正確な入り口や駐車場の状況については、事前に地図アプリなどで確認しておくとスムーズですよ。

特徴・おすすめの方: 足洗い場や青雲亭、碑文の案内板など、思索にふけるためのスポットが随所に配置されており、単なる登山というよりは「歩く巡礼」といった雰囲気です。ハイライトは、なんといっても「空の庭展望台」からの眺望！ 整備された土の道が続く緩やかな傾斜なので、ご家族連れやトレッキング初心者の方にもぴったりなコースです。

コース3 — 東華寺入口駐車場 → 山頂

📏 6.54km ⛰️ 693m 獲得標高 ⏱️ 2:47 🔁 周回 難易度 ★★★★☆ ⤓ GPX

経由: 駐車場 → トイレ → 休憩所 → カルタックコゲ（急坂） → 自然景観 → 現在地看板 → 分かれ道 → 東華寺入口駐車場 → 売店・駐車場 → 観光資源 → 東華池

アクセス: パルコンサンの代表的な寺院、東華寺の入り口からスタートします。大邱（テグ）の地下鉄が通っていないエリアなので、市内のバスを利用するのが一般的ですが、入り口に駐車場もしっかり完備されているので、車でのアクセスはかなりスムーズなコースです。

特徴・おすすめ: 序盤に東華池（貯水池）や売店、駐車場などの施設がまとまっていて、歩き出しもスムーズです。中盤の「カルタックコゲ（息が切れるほどの急坂）」で一気に傾斜が厳しくなる、典型的な「お寺から山頂へ」というルート。パルコンサンを初めて訪れる方にも、ぜひおすすめしたいコースです。

特徴・おすすめ: 治山渓谷に沿って登っていく、治山滝や望爆台といった美しい景観が楽しめるコースです。夏場の渓谷歩きとして人気がありますが、雨が降ると急流になったり増水したりするので要注意。渓谷トレッキングが大好きな方にはぴったりのルートです。

特徴・おすすめ: 「登攀事故注意」や「断崖注意」といった看板が次々と現れる、岩場や絶壁のセクションが多いハードなコースです。正直、私のような年齢だと少しばかり足がすくむ場面もありますが（笑）、その分、奇岩や八工山の稜線の絶景という素晴らしいご褒美が待っています。

特徴・おすすめ: 経由地についての情報はそれほど多くありませんが、「ギワ（瓦）」という地名から、かつて建物があった跡をうかがわせます。ビロ峰からソンイン峰へと続く縦走コースの一部として、計画に組み込むのにぴったりの区間です。

難易度・路面・注意点

写真: 韓国観光公社 観光写真ギャラリー (大邱広域市軍威郡部渓面) ・ 公共누리

行政区画ごとの登山道データには、全部で4つの区間が登録されています。データの難易度はすべて「初級」となっていますが、路面情報の登録はありません。

ここでいう「初級」は、あくまで低地にある区間を基準としたもの。山頂を目指す本格的なコースとは、性質が全く違います。実際に100名山のデータをGPXで実測してみたところ、山頂付近のコースは累積標高差が600〜800mほどあり、難易度は星4つから5つ（★★★★☆〜★★★★★）に相当します。体感としては、かなりハードな部類に入りますね。

施設面では、登山道の空間情報に登録されている水飲み場・避難小屋・危険区域・展望スポットなどのPOIは、すべて「0件」と集計されています。つまり、公式な登録施設は実質ゼロ、と考えて間違いありません。

ただし、山頂付近のGPXルート上では、複数のトイレや、落石の危険があるエリア（コース1）、岩壁登攀の危険がある場所や絶壁の注意区間（コース5）など、実際の現場にはリスクを示す標識が確認できます。統計データと現地の看板では、これほど差があるという点に注意が必要です。岩場は凍結すると非常に滑りやすく、雨が降った直後は落石の危険も高まるため、十分な警戒をお願いします。

距離と所要時間

사진: 한국관광공사 관광사진 갤러리 (대구광역시 군위군 부계면) · 공공누리

園内の総延長6.76kmという数字が出てきましたが、これは先ほど紹介した4つの区間をすべて合わせた合計距離です。決して一つのコースがこれほど長いわけではありませんので、そこはご安心を（笑）。この総延長に基づいた登り・下りの所要時間は、登りが120分、下りが84分と記録されています。距離で割ると平均的な登りのペースは1kmあたり約18分となりますが、これも全区間を合算した数値です。「一つのコースでこれだけかかるのか！」と勘違いしてしまわないよう、注意してくださいね。

実際に山頂を目指す際の目安としては、「百名山」のGPXデータに基づく推定時間が参考になります。代表的なコースでは、片道およそ2時間15分から3時間22分の間に収まっています。往復や縦走をしない限りは、ほとんどの場合で日帰りでも十分間に合います。そもそも山小屋などの宿泊施設もありませんので、無理に1泊の計画を立てるよりは、日帰りでの山行をおすすめします。 ただ、複数のコースを繋いで縦走するような場合は、総所要時間が6時間を超えてしまうことも。そんな時は、日の出直後に出発するなど、時間に十分な余裕を持って行動するのが賢明です。

アクセス（公共交通機関・駐車場）

写真：韓国観光公社 観光写真ギャラリー（大邱広域市軍威郡扶渓面） ・ 公共ヌリ

パルゴン山方面は、大邱の地下鉄が直接通っていないため、基本的には市バスを利用することになります。登り口に近い停留所としては「東和寺入口前（동화사입구앞）」がありますので、東和寺方面のコースを計画されている方は、ここを拠点にルートを考えるとスムーズです。他にも、カッバウィ（冠峰）方面やパゲサ・スッテゴル方面など、目的地によってバスの路線がそれぞれ異なります。乗り換えなどは、必ずKakaoMapやNaver Mapでリアルタイムの検索をしてから向かうようにしてくださいね。

駐車場については、登山道の公式な情報では「0箇所」と集計されていますが、実際にはコース3の経由地に「東和寺入口駐車場」があり、コース4の経由地には治山地区探訪支援センター付近に駐車スペースが存在します。つまり、お寺や探訪支援センターの入り口付近には自然発生的な駐車スペースがあるものの、公式な登山道附属の駐車場としては登録されていない、ということになります。週末や繁忙期にはかなりの混雑が予想されるので、その点はあらかじめ覚悟（？）しておいた方が良さそうです。

シーズン・天候・装備

写真: 韓国観光公社 観光写真ギャラリー (慶尚北道 慶山市 ワチョン面) · 公共누리

パルゴンサンへの登山は、気温や眺望のバランスが良い春（4〜5月）と秋（10〜11月）が一番のおすすめです。夏は渓谷コース（チサン渓谷など）での水遊びを兼ねた山歩きが魅力ですが、湿度と雷雨には要注意。冬は稜線の岩場で凍結や霧氷が発生しやすく、体感的な難易度がグンと上がります。

必須装備としては、足首をしっかりサポートしてくれる登山靴はもちろんのこと、水場がほとんどないので、多めの飲み水（1人あたり1〜1.5L以上）は絶対に欠かせません。雨や落石のトラブルに備えて、レインウェアと着替えも用意しておきましょう。冬場ならアイゼン、ゲイター、防寒グローブは必須です。岩場が多いコースでは、手袋とヘッドランプ（万が一下山が遅れた時のために）もあると心強いですね。また、夏のトレッキングには日焼け止めと虫除けがあると快適に過ごせますよ。

外国人旅行者へのヒント

写真: 韓国観光公社 観光写真ギャラリー (慶尚北道慶山市臥村面) · 公共누리

パルコンサンはローマ字で “Palgongsan” と表記され、主峰は「Birobong（ピロボン）」、代表的な寺院は「Donghwasa（トンファサ）」と呼ばれています。山全体への入山許可や登山道の入場料は特にかからないようですが、お寺の境内を訪れる際は現地の案内に従ってくださいね。なお、水飲み場は登録されているものが「0箇所」となっていますので、山に入る前に必ず十分な飲料水を準備しておくようにしましょう。トイレは各コースの入り口や途中の経由地（駐車場や休憩所付近）に設置されています。出発前に済ませておくのがスムーズですよ。

地図アプリについてですが、韓国ではGoogleマップだと徒歩や公共交通機関のルート案内が正確でないことがよくあります。必ず「KakaoMap」か「Naver Map」をインストールして、リアルタイムの経路や停留所情報を確認するようにしてくださいね。安全面では、落石や崖の危険がある区間では必ず標識に従って行動しましょう。また、スマートフォンの充電は十分に確保しておき、万が一の緊急時には迷わず119番へ。特にコース1やコース5のように、落石や崖の危険箇所が続くルートは、ある程度登山経験がある方向けといえます。
