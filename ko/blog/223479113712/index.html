한강 자전거 옷걸이 코스 거리는 여러가지 사이즈가 있다.

자전거 · 발행일 2024. 6. 15. ·

일반적인 옷걸이 코스의 gps궤적은 위 이미지와 같습니다. 거리는 출발지에 따라 다르겠지만 코스자체는 30km미만입니다.

ㅎㅎㅎ 우리가 주행한 옷걸이 코스입니다. 뭔가 옷걸이가 너무 커졌습니다

Previous image Next image

상암동을 출발해서 월드컵대교를 이용해 한강남단으로 이동했습니다. 그리곤 한강 동쪽을 향해 달렸습니다. 기온은31도였고 습도도 어느정도 있어서 매우 무더웠습니다. 바람은 서풍이어서 동쪽으로 가는건 편했습니다. 다만 복귀라이딩이 걱정되었습니다.

알리에서 구입한 라파물병이…져지 등뒤에 빈병으로 끼워진지 몰랐습니다. 빈 물병만 넣고 달렸습니다. 어쩐지 가벼웠습니다. 마치 백패킹 갔는데..텐트만 챙기고 폴대가 없는 기분…

Previous image Next image

반포대교 지나서 나오는 CU에서 얼음과 이온음료를 구입해서 마셨습니다. 혹시몰라 타우린? 타우렌?이 들어있을법한 바카스 젤리도 몇알 먹어주었습니다. 한강동쪽은 양재천을 지나서는 한번도 안가본곳이었습니다. 즉 잠실쪽은 처음이어서 긴장하며 주행했고 잠실대교 이후 잠실철교로 한강을 건너는 부분은 매우 잘 되어 있었습니다.

Previous image Next image

마치 괴물이 나올거 같은 다리를 지나서…

Previous image Next image

잠실철교로 올라갑니다. 지하철2호선이 건너는 다리로 차량은 이동하지 않고 지하철과 자전거 그리고 사람만 건널 수 있게 된 다리였습니다. 쾌적하게 한강을 건넜습니다.

Previous image Next image

쾌적한 잠실철교단과…반대쪽 북단풍경… 사진에서도 더위와 열기가 느껴집니다.

Previous image Next image

괴물이 나올거 같은 다리의 북단…영동대교 북단에서 잠시 휴식을 가집니다. 날도 덥고 서풍입니다. 평균속도를 유지하기 힘들었습니다.

Previous image Next image

자덕들의 성지 용비쉼터입니다. 포카리 한병씩 더 부어줬습니다. 이제 마포구까지 달리면됩니다. 서풍맞으며…

Previous image Next image

월드컵대교 북단입니다. 오르막 오르락 오르락 하면 상암동 진입입니다. 이날 총 거리는 60키로가 조금 안되는 58키로입니다. 날씨가 참 무더웠고 서풍은 야속했습니다. 그래도 평균속력이 21키로 나왔습니다. 16인치 미니벨로로 주행한것 치고는 괜찮은 성적입니다. 아마 로드로 탔으면 25이상 나왔겠습니다.

Previous image Next image

좌는…우리가 옷걸이코스로 돈 결과물…우측은 흔히 사람들이 검색하는 옷걸이 코스의 gps궤적모양…우리는 너무 서쪽에서 출발해서 대형 옷걸이는 사왔습니다.

요약해보면…

옷걸이 코스는 반포대교를 시작으로 잠실철교를 돌아오는 코스로 쾌적한 자전거 라이딩 코스입니다. 차량을 기다리는 신호가 1개도 없으며 무정자 라이딩이 가능하며 잠실철교 올라가는곳은 엘리베이터도 있습니다(나중에 안 사실) 한강근처에서 자전거를 타시는 분들에게는 사랑받아 마땅한 코스입니다.

저처럼 서쪽에서 출발하면 모양은 좀 이상해지지만 반포대교를 지날때 신호1번만 기다리면 60여키로를 쉼없이 주행할 수 있는 자전거 전용 도로로 이루어진 코스로 기분좋게 허벅지를 털어낼 수 있습니다.

처음 다녀와본 코스 기분좋게 주행했습니다.
