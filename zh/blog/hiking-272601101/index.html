概述

照片：韩国观光公社观光照片图库（庆尚南道昌原市马山合浦区东西北10街68号）· 公共누리

无鹤山（Muhaksan）横跨庆尚南道昌原市马山合浦区和马山会原区，居高俯瞰马山市区和马山湾的南侧。山顶海拔通常记作761米左右，不过按这次登山路线的实测数据，顶峰区域的最高点确认为785米（各条路线GPX终点海拔在761到785米之间略有出入，是因为顶峰附近有好几个山峰和观景点，测量点各不相同）。相传山名源自它形似仙鹤展翅起舞的模样，对马山市民来说，这里是守护一方的”镇山”，一年四季都有人来爬，是当地颇具代表性的近郊名山。多数路线往返顶峰大约需要6到7小时，当天来回完全绰绰有余。

山势与地形

照片：韩国观光公社观光照片库（庆尚南道昌原市马山会原区新甘路 13-14）· 公共누리

无鹤山的山势挺有意思，一半是缓坡土山，一半是陡峭岩脊，两种风格并存。越往山顶走，十字岩、书马之基这些岩石地形就越显眼，鹤峰、中峰（482米）、甑峰、烽火山山顶这些大大小小的山峰也沿着稜线一路排开。从西院谷方向上山，山谷和松林比较发达，起步那段基本是好走的土路，可越接近稜线，岩石路段和陡坡就交替出现，走起来挑战性明显上升。山顶一带有好几处视野开阔的地方,连马山湾、市区一直到镇海湾方向都能看见,可以说是一路走一路都有风景可看的山。

登山的感觉

照片：韩国观光公社观光图片图库 · 公共누리

一开始基本都是茂密的林间小路和平缓坡度，走起来跟散步差不多，可过了半山腰，岩石坡面和陡坡就开始密集出现，难度陡然上升。等爬上山脊，视野豁然开朗，马山市区和大海尽收眼底，这样的景致反复出现——所以这座山被认为是”累是累点，但风景绝对值回票价”的类型。春天杜鹃花开、新绿满山，秋天层林尽染,四季分明；夏天湿度高，体感闷热明显；冬天山顶岩石地带容易结冰，整体山行氛围随季节变化十分明显。

路线1 — 头尺登山口 → 舞鹤山山顶

📏 8.27km ⛰️ 666m 爬升 ⏱️ 3:11 ➡️ 单程 难度 ★★★★★ ⤓ GPX

沿途经过：头尺 → 头尺登山口 → 高句丽 → 舞鹤山导览图 → 休息区兼体育公园 → 停车场 → 休息处 → 岔路口 → 洗手间 → 舞鹤山登山道综合导览图 → 西马地基休息处 → 马山港观景点 → 观景点 → 舞鹤山山顶 → 观景点兼休息处

登山口交通：头尺公交站就在登山口附近，坐市内公交就能到。目前没查到正规登记的停车场，所以比起自驾，更建议搭公交前往。

特色·推荐人群：这是一条直通山顶、很有代表性的路线，一路经过体育公园、休息处，再到西马地基，最后登顶，途中好几处都能看到风景，包括眺望马山港的观景点。第一次来舞鹤山的朋友，或者把登顶看风景放在第一位的登山爱好者，走这条路线正合适。

路线 2 — 元溪 → 甑峰

📏 9.48km ⛰️ 721m 爬升 ⏱️ 3:34 ➡️ 单程 难度 ★★★★★ ⤓ GPX

途经：元溪 → 村庄简易停车场兼空地 → 无登山道（非法定区间标示）→ 岔路口 → 内元溪村 → 休息处 → 体育公园兼休息处 → 甑峰 → 歇脚林休息处 → 洗手间 → 相聚广场休息处 → 休息处兼体育公园 → 观景点 → 八角亭休息处 → 烽火山山顶

登山口交通：从元溪公交车站步行即可到达，村里有个简易停车场兼空地，不过地方不大，旺季容易挤满车。

特点·推荐对象：这条路线在全程各段里距离和高差都算是数一数二的，属于一口气连穿甑峰和烽火山山顶的稜线纵走型路段。路牌上标有”无登山道”的区间，第一次走的人尤其得先看好地图再出发，比较适合体力有信心的山友挑战。

路线3 — 马山女中（起点）→ 山顶

📏 8.08km ⛰️ 687m 爬升 ⏱️ 3:10 ➡️ 单程 难度 ★★★★★ ⤓ GPX

沿途：马山女中 → 马山女中（起点）→ 附近无专用停车场 → 休息处 → 洗手间 → 岔路口 → 兼作体育公园的休息处 → 烽火药水泉 → 药水泉 → 无岔路 → 西三支休息处 → 讷川泉药水泉 → 讷川泉休息处 → 无谷塔药水泉 → 无谷塔药水泉休息处

入口交通：马山女中公交站就在起点附近，坐公交过来很方便，不过路牌上也写明了，附近没有专用停车场，开车来的话得自己想办法。

特点·适合人群：六条路线里，这段是药水泉地名出现最多的——烽火药水泉、讷川泉药水泉、无谷塔药水泉一路接连不断，整体走下来更像是紧贴市区生活圈的市民散步健身道，随便走走、喝喝水都挺惬意。不过药水泉是不是真的能取水，得看季节和管理状况，说不准哪天就断水或者暂停开放，为保险起见，还是自己带够水比较踏实。

路线4 — 西院谷入口 → 鹤峰

📏 6.85km ⛰️ 671m 爬升 ⏱️ 2:50 🔁 环线 难度 ★★★★☆ ⤓ GPX

途经：西院谷入口 → 西院谷公营停车场 → 洗手间 → 岔路口 → 鹤峰入口 → 八角亭休息处 → 观景点 → 鹤峰观景点 → 鹤峰 → 孤云台 → 十字岩 → 完月洞岔路口休息处 → 中峰 → 大谷山·舞鹤山岔路口休息处 → 无岔路

登山口交通：这里有西院谷入口公交站，而且六条路线里只有这条在路标上标出了公营停车场（西院谷公营停车场），所以开车过来相对方便一些。

特色·推荐对象：这条路线会连续经过孤云台、十字岩等舞鹤山最具代表性的岩石地貌，鹤峰观景点等观景点也分布得比较密集。距离和用时在六条路线里算是偏短的一条，很适合想用一天时间把舞鹤山的招牌美景浓缩体验一遍的游客。

路线5 — 登山口 → 中间瞭望台

📏 7.1km ⛰️ 662m 爬升 ⏱️ 2:53 ➡️ 单程 难度 ★★★★☆ ⤓ GPX

途经：岔路口 → 休息处 → 白云寺停车场 → 白云寺前洗手间 → 白云寺 → 登山口 → 白云寺入口八角亭 → 山火瞭望哨 → 无学山祈祷院药水泉 → 藤架山庄药水泉 → 中间瞭望台 → 云雾药水泉 → 云雾休息处 → 大谷山山顶 → 大谷山观景点

上山口交通：路牌上标有白云寺停车场，走寺庙方向可以停车；不过没查到附近有专门的公交站，坐公交来的话可能得从最近的站点再走一段路。

特点·推荐人群：这条路线要经过白云寺，沿途还串着山火瞭望哨、药水泉等好几个地名，整体偏向大谷山方向，比起冲顶，更适合以欣赏大谷山和中间瞭望台一带的风景为目的的登山。如果还想顺便逛逛寺庙，这条路线也很合适。

路线6 — 中里站入口 → 山顶

📏 9.8km ⛰️ 671m 爬升 ⏱️ 3:34 ➡️ 单程 难度 ★★★★★ ⤓ GPX

经过路线：中里站 → 中里站停车场 → 内西邮局 → 中里站入口 → 岔路口 → 休息处 → 麻岭 → 麻岭入口简易停车场 → 上玉谷村

起点交通：中里站是庆全线的一般铁路站，可以坐火车过来再步行到起点，中里站停车场和麻岭入口简易停车场这两处都有停车位。

特点·适合人群：全程9.8公里，算是这几条路线里比较长的一条了，中间要翻过麻岭那段山脊，比起单纯的往返登顶，更适合想走完整条山脊、或者想换个下山路线的资深山友。能直接从火车站出发开始爬山，这一点是它跟其他路线最大的不同。

难度·路况·注意事项

照片：韩国观光公社观光图片库（庆尚南道昌原市马山合浦区海岸大路288）· 公共누리

辖区登山道数据上标注的最高难度是”简单”，但这是以行政洞为单位划分的全区段（见下表）为基准得出的结果，实际登顶各条路线的实测难度大多在★★★★☆以上，体感难度要比标注的高，这一点要留意。路面信息虽未单独登记，但从路标上反复出现的书马支基、十字岩、孤云台等地名来看，山顶一带的稜线看起来不是单纯的土路，而是夹杂着岩稜、岩石区段，雨天或落叶季节行走时需注意防滑。

设施POI方面，按官方登山道地点数据统计，观景点、饮水台、庇护所、危险区域均为0个，也就是说未确认有单独指定并管理的庇护所或危险区段标识。不过在各路线的路标实测数据中，出现了马山港观景点、鹤峰观景点、大谷山观景点等观景地名，以及烽火药水泉、訥泉药水泉、无谷塔药水泉、雾药水泉、无鹤山祈祷院药水泉、藤树山庄药水泉等药水泉地名，数量还不少。不过这些只是登山客之间口口相传的地点名称，并非登记在册的官方管理设施，所以饮用水最好还是自己另外准备，这样更保险。厕所方面，在头尺路线、院溪路线、马山女中路线、书院谷路线、白云寺路线的路标上都各自出现过，可以确认在多个登山口附近都能使用。

距离与所需时间

사진: 한국관광공사 관광사진 갤러리 · 공공누리

管内登山道总长为69.93公里，这是无鹤山区域内所有登记路段加总后的数字，并非单一路线的距离。若把所有登记路段全部走一遍，上行累计需1248分钟，下行累计需861分钟，换算成距离后的平均上行配速约为每公里18分钟。不过这个配速只是以管内全部路段的累计数值为基础得出的参考值，并非某条特定路线的实际耗时，规划具体路线时还是要以上面”主要路线”表格里各路线的单程预估时间（约2小时50分~3小时34分）为准。六条路线给出的都是单程时间，如果是原路折返或往返登山，建议按单程时间的1.8~2倍来预估，大部分路线一天之内就能往返完成，无鹤山作为当日往返的登山地是完全够用的。不过如果打算把管内全部路段（69.93公里）串联起来纵走，一天之内很难走完，最好考虑安排1晚以上的行程会比较稳妥。

怎么去（大众交通·停车）

照片：韩国观光公社观光照片图库（庆尚南道昌原市马山会原区斗谷路54，斗尺洞）· 公共누리

马山、昌原一带没有单独的城市地铁，所以市内巴士是主要的交通方式。入口附近的公交站可以确认有斗尺、内西小学、院溪、乐天玛特、马山女中、三鹤寺、书院谷入口等，按各条路线的登山口选相应站下车就行。中里站是京全线经过的一般铁路站，可以坐无穷花号之类的列车到这儿，再步行到入口。停车设施方面，路标上标出的大概只有书院谷公营停车场、白云寺停车场、中里站停车场，以及麻岾岘入口的简易停车场，其余大部分登山口似乎都没有正式停车场，所以还是尽量搭大众交通比较稳妥。想确认实时路线和站点位置的话，用韩文地名识别率高的Kakao地图或Naver地图App查最准。

季节·天气·装备

照片：韩国观光公社观光照片图库（庆尚南道昌原市马山合浦区完月西2街8号（完月洞））· 公共누리

爬山最舒服的时段要数春天（4~5月）和秋天（9~11月），气温宜人，还能顺带欣赏杜鹃花、红叶这类应季景色。夏天又闷又热，体力消耗特别大，建议早点出发，多补水；下雨天岩石路段容易打滑，遇上这种天气还是改天再去比较稳妥。冬天山顶棱线和岩石区域很容易结冰，得备上冰爪、绑腿之类的冬装。通用的必备装备包括登山鞋、登山杖、防风防水外套、头灯、备用衣物，另外山里没有饮水点，水一定要多带一点（建议每人1.5~2升以上）。

外国游客小贴士

照片：韩国观光公社观光照片图库（庆尚南道昌原市马山合浦区鱼市6街18号（南城洞））· 公共누리

无鹤山（Muhaksan）位于庆尚南道昌原市，是当地一座近郊山，罗马字标注中山顶写作”Muhaksan Peak”，主要登山口有 Duchcheok、Wonggye、Masan Yeojung（马山女中）、Seowongok-ipgu（西院谷入口）、Jungni Station（中里站）等。它不是国立公园，而是市民日常爬的近郊山,目前没有查到需要办理入山许可或买门票的规定，可以自由登山。厕所在斗尺、院溪、马山女中、西院谷、白云寺几条路线的登山口附近都能找到，但没有登记在册的饮水点，所以瓶装水一定要提前备好。Google 地图在韩国的登山路线信息经常不准，建议到了当地下载 Kakao Map 或 Naver 地图 App，实时查路线和公交信息更靠谱。安全起见，最好把行程安排在天黑前下山，通讯信号弱的山脊路段建议提前把地图存成离线版备用。
