概述

照片：韩国观光公社旅游照片图库（首尔特别市城北区贞陵洞）· 公共누리

北汉山（Bukhansan）横跨首尔市区与京畿道高阳市，是首都近郊颇具代表性的名山。根据数据，其顶峰海拔约 834 米，最高点是那座广为人知的花岗岩峰——白云台（Baegundae）。它虽坐落于首都圈的正中央，却兼具险峻的岩峰与幽深的溪谷，只靠地铁和市内公交就能抵达登山口（登山起点），交通十分便利。数据里记录的最高难度虽为“中等”，但真正通往顶峰一带的实测路线大多净爬升超过 600 米，对体力其实是不小的考验。将辖区内登山道分成若干条线路合并计算，总长为 17.47 公里；山上没有避难所，因此无论选哪条线路，这都是一座足以当天往返的山。

山势与地形

照片：韩国观光公社观光照片画廊（首尔特别市恩平区佛光洞）· 公共누리

北汉山是典型的花岗岩岩山，最大的特点就是一座座光洁的白色岩峰沿着山脊一路延展。以主峰白云台为中心，仁寿峰、万景台三峰鼎立，自古又被称为三角山（三角山）。翻看公共数据里记录的主要山峰，柴丹峰（Sidanbong）604 米、纱帽岩（Samobawi）548 米、三角点峰（Samgakjeombong）545 米、牛耳峰顶（Uibong）539 米、津宽峰（Jingwanbong）531 米等，星罗棋布地耸立在山脊各处。除此之外，还有一批海拔 388 米到 472 米的矮峰，构成了旁支的支脉。所以这并不是一座只盯着一个山顶往上爬的山，而是把多座山峰和城墙串联起来、越走越有意思的山脊纵走之山。

在地形上，下方谷地那段平缓的林间小路，与顶部裸露的岩板、铁栏杆路段，界限分明。沿着山谷往上走，一旦贴上山脊，视野便豁然开朗，花岗岩峰峦骤然扑面而来——正因为是这样的结构，海拔越高，地形变化越是戏剧化。

登山的感受

照片：韩国观光公社观光照片画廊（首尔特别市恩平区佛光洞）· 公共누리

从登山口到半山腰，一路都是橡树和松树葱郁的林间小道，树荫充足，走起来很舒服。溪谷路线则可以一边听着水声一边往上爬，途中还会遇到不少瀑布和山泉，格外清爽。可一旦接近山顶，气氛就完全变了——出现了需要手脚并用攀爬的岩石陡坡、裸露的岩盘，以及带铁栏杆和台阶的路段。登上山脊后，首尔市区乃至道峰山方向的景色一览无余，能在城市近旁收获这样的视野，正是北汉山登山最大的魅力所在。春天可以欣赏溪谷的新绿和金达莱，秋天有山脊的红叶，而冬天由于岩石和结冰，体感难度会大幅上升。

路线 1 —— 入口 → 山顶

📏 8.09km ⛰️ 745m 爬升 ⏱️ 3:16 🔁 环线 难度 ★★★★★ ⤓ GPX

途经：停车场 → 国师堂 → 洗手间 → 入口 → 岔路口 → 北汉山国立公园信息栏 → 休息区 → 山火及登山事故紧急联络电话，119 → 路标 → 安全登山、可供休憩的休息区 → 山火及登山事故紧急联络电话，119 → 观景点 → 119 急救箱 → 山火扑救用灭火器 → 落石危险

登山口交通：登山口起点配有停车场、国师堂和洗手间，很适合做好准备再出发。搭乘公共交通的话，一般是坐市内公交到登山口附近的站点，再步行进入。具体的上下车位置，建议在地图 App 上确认。

特点与推荐人群：净海拔落差达 745 米，是各条登顶实测路线中爬升幅度最大的一条。前段一路是林间小道和休息区，负担不大，但越往上走越会出现落石危险路段。所以这条路线适合体力有信心、想一鼓作气冲顶的朋友。

登山口接入：沙器幕谷（Sagimakgol）在数据上属于登山口附近确实有公交站点的入口。整体是穿过公园守护站和洗手间、沿溪谷向上攀登的走法，需先经过村里的商店街再进入。

特点与推荐人群：全程 12.02 公里，是登顶线路中最长的一条，适合想沿着溪谷和瀑布慢慢走、尽情游览的人。路程较长，时间分配就显得格外重要，推荐给那些想长时间沉浸在水声与山林间、悠闲徒步的朋友。

路线 3 — 岔路口 → 山顶

📏 8.38km ⛰️ 607m 爬升 ⏱️ 3:06 ➡️ 单程 难度 ★★★★★ ⤓ GPX

登山口接近方式：这是一条连接平昌、兄弟峰方向的路线，一般的走法是从市区一侧的住宅区乘市内公交抵达后，再步行进入。下段设有青水公厕和平昌公园守护站公厕。

特点·推荐人群：这是一条水源丰沛的路线，沿途经过灵泉药水泉、双支流药水泉，并可在兄弟峰观景点饱览风光。不过，途经指示牌上明确标注了”冬季常年结冰路段”，所以冬天走这条路要格外小心。

路线 4 — 北汉山城入口 → 北汉山白云台山顶

📏 9.21km ⛰️ 671m 爬升 ⏱️ 3:25 🔁 环线 难度 ★★★★★ ⤓ GPX

途经：北汉山城入口 → 北汉山城收费停车场 → 咨询处 → 北汉山城公厕 → 岔路 5 → 野花园 → 水口亭公厕 → 西岩寺 → 岔路 9 → 北汉洞历史馆旁公厕 → 菩提寺 → 开涓瀑布 → 岔路 16 → 大东寺山泉 → 大东寺

登山口交通：北汉山城入口（Bukhansanseong Ipgu）是北汉山最具代表性的登山口，咨询处、收费停车场和公厕一应俱全。到入口一带的市内公交线路四通八达，公共交通十分方便。

特色与推荐人群：这是一条直通白云台山顶的经典正统路线，城墙、寺庙、溪谷、山泉沿途轮番登场，很适合第一次来北汉山的朋友。沿途公厕众多，同样值得推荐给家庭出游者或外国游客。

登山口交通：这条线路连通牛耳、道诜寺（Doseonsa）方向，途经牛耳广场卫生间和白云台游览支援中心，交通颇为便利。从市区北部搭公共交通就能轻松抵达。

特色·适合人群：线路沿着通往东将台（Dongjangdae）方向的城墙山脊延展，一路上还有道诜寺、舍利塔等文化看点。休息处分布密集，很适合喜欢边走边多歇脚的人，也适合想饱览山脊城墙风光的朋友。

特点·适合人群：净爬升 602 米，单程 3 小时以内，在登顶级别的路线里算是相对省力的。沿途水源不少，比如清水川泉水处、灵鹫寺甘露水，最后还有一处首尔市区眺望点，适合想俯瞰城区风景的朋友。

距离与耗时

照片：韩国观光公社观光照片画廊（首尔特别市恩平区佛光洞）· 公共누리

北汉山没有避难所，园内总里程为 17.47km，无论走哪条路线，一日登山都绰绰有余。 实测的登顶主线单程从约 2 小时 46 分（国家地点编号 → 罗汉峰）到约 4 小时 7 分（沙器幕谷 → 山顶）不等，把往返和休息算进去，大多数路线也都能在一天之内走完。

顺带一提，数据上园内全程的累计耗时记录为上行 342 分、下行 204 分，不过这是把多个路段全部串联起来的累计值，并不是单条路线的耗时。 由此推算出的平均上行配速约为 20 分/km，你可以用它乘以自己所选路线的距离，大致估算出单程时间。比如一条 8km 上下的路线，光靠配速就能估出上行大约要 2 小时 30 分到 3 小时。实际耗时则会因个人体力和登顶岩脊路段的拥堵而有所不同。

交通指南（公共交通·停车）

照片：韩国观光公社观光照片馆藏（首尔特别市恩平区佛光洞）· 公共누리

北汉山交通十分便利，借助首尔的地铁网和市内公交都能轻松抵达。通常的做法是在市中心的地铁站下车，再换乘开往登山口的市内公交。根据数据，登山口附近实际确认到的公交站点有 孝子2通、沙器幕谷、北汉山动员训练场、孝子治安中心。其中沙器幕谷站可直接连通线路2的登山口。

至于停车，虽然按林业厅的数据，辖区内登记在册的停车场为0个，但线路4的路标上确认到了北汉山城收费停车场。自驾前来时，各登山口的收费停车场情况各不相同，建议出发前先行确认。无论是公共交通还是停车，实际线路和实时路线都建议在 KakaoMap 或 Naver Map 上查询（谷歌地图对韩国国内的步行和公共交通路线可能不够准确）。

季节·天气·装备

照片：韩国观光公社旅游照片图库（首尔特别市恩平区佛光洞）· 公共누리

推荐季节是新绿与杜鹃盛开的春天（4~5月），以及山脊红叶最美的秋天（10~11月）。夏天溪谷凉爽，但要当心酷暑、阵雨和溪流暴涨；冬天山顶岩壁结冰、狂风肆虐，体感难度会陡然上升。第三条线路的路标上明确标注了”冬季常年结冰路段”，冬季登山时尤其要小心。

必备装备首推抓地力好的登山鞋。山顶有不少花岗岩石板和铁栏杆路段，得防着打滑。虽然设有饮水台，但视季节和情况可能没水，所以饮用水要多带一些。冬天需要冰爪和防风保暖衣物，夏天要防紫外线和脱水；这里观景点很多，带上摄影器材或防风衣也很实用。

给外国旅行者的小贴士

照片：韩国观光公社观光照片图库（首尔特别市恩平区佛光洞）· 公共누리

  • 地名罗马字：北汉山 Bukhansan · 主峰白云台 Baegundae（约 834 米）· 主要登山口北汉山城入口 Bukhansanseong Ipgu · 沙器幕谷 Sagimakgol · 牛耳·道诜寺方向 Doseonsa · 贞陵 Jeongneung。
  • 入山许可·门票：无需提前申请任何许可，登山道也不收入山费。不过部分登山口的停车场是收费的（比如线路 4 的北汉山城收费停车场）。
  • 厕所·饮用水：主要登山口和溪谷线路沿途都设有厕所（北汉山城厕所、清水厕所、平昌公园守护所厕所、灵鹫寺厕所等）。山中的饮水点（山泉）确认有 5 处，但不保证随时都能取水，请自备饮用水。
  • 地图 App：谷歌地图在韩国境内的步行和公共交通路线可能不太准确，建议使用 KakaoMap 或 Naver Map。
  • 安全：据数据，危险路段集中在岩壁区域，标注了 5 个危险地点 POI。在主峰岩脊以及有落石、结冰的路段请遵照指示牌行事；山中没有避难所，请把行程安排在天黑前下山。全线共有 36 处观景点，能在山脊各处安全地饱览首尔市区的景色。
