就在城边，尽览花岗岩脊线美景：首尔北汉山

概览

图片来源：韩国观光公社旅游摄影库（首尔特别市城北区贞陵洞） · 公共领域

北汉山（Bukhansan）是横跨首尔市区与京畿道高阳市的代表性近郊名山。其最高点是海拔约834米的花岗岩峰，即广为人知的白云台（Baegundae）。虽然它就坐落在首尔都市圈的核心地带，却兼具险峻的岩壁与幽深的峡谷。最方便的是，只需搭乘地铁或公交车就能直达登山起点，交通极其便利。

虽然数据上标注的难度等级为“中等”，但实际通往顶峰的路径累计爬升高度大多超过了600米，对体力而言可不是件轻松的事。山内登山步道总长约17.47公里，由于山上没有避难所，无论选择哪条线路，都非常适合作为当天往返的一日登山行程。

山势与地形

图片来源：韩国观光公社 旅游摄影画廊 (首尔特别市恩平区佛光洞) · 公共领域

北汉山是典型的花岗岩岩质大山，最显著的特征就是那洁白光滑的花岗岩峰峦，沿着山脊蜿蜒起伏。以顶峰白云台为中心，仁寿峰与万景台构成了一个三角形，因此自古以来也被称为“三角山”。查阅公开数据可以看到，在山脊各处矗立着不少主要峰峦：柴丹峰 (Sidanbong) 604m、Samobawi 548m、三角点峰 (Samgakjeombong) 545m、牛耳峰 (Uibong) 539m 以及进宽峰 (Jingwanbong) 531m 等。此外，还有一些海拔在 388m 到 472m 之间的较低山峰构成了错落的支脉。这使得它不仅仅是一座只看顶峰的登山地，更是一座在多座山峰与城墙之间穿行、极具棱线行走乐趣的山。

这里的地形特征非常鲜明：山下溪谷处是平缓的林间小径，而山顶则是裸露的花岗岩岩面与护栏区域。沿着溪谷向上攀登，一旦踏上山脊，视野会瞬间豁然开朗，直面壮丽的花岗岩峰峦。可以说，随着海拔不断升高，地形的变化非常剧烈。

登山感悟

图片来源：韩国观光公社观光摄影画廊 (首尔特别市恩平区佛光洞) · 公共Nuri

从山脚到半山腰，郁郁葱葱的橡树和松林构成了舒心的林间小道，遮阴充足且走起来很舒服。沿着溪谷路线向上，伴着潺潺水声攀登，途中还会邂逅瀑布和山泉点，格外清爽。不过，一旦接近顶峰，氛围就完全变了——陡峭的岩坡需要手脚并用，裸露的岩床、铁栏杆和台阶层出不穷。登上山脊后，视野豁然开朗，首尔市区一直延伸到道峰山的壮丽景色尽收眼底。能在离市中心这么近的地方领略这般美景，正是北汉山登山最大的魅力所在。春天可以看溪谷的新绿与杜鹃花，秋天则是满山红叶；至于冬天，冰冻的岩石会让登山难度陡然上升。

路线 1 — 入口 → 山顶

📏 8.09km ⛰️ 745m 爬升 ⏱️ 3:16 🔁 环线 难度 ★★★★★ ⤓ GPX

经由：停车场 → 国师堂 → 洗手间 → 入口 → 分岔路 → 北汉山国立公园信息中心 → 休憩处 → 山火/登山事故紧急联系方式 (119) → 指示牌 → 安全登山休憩区 → 山火/登山事故紧急联系方式 (119) → 观景台 → 119 急救箱 → 山火灭火器 → 落石危险区

入场方式：入口处配备了停车场、国师堂和洗手间，出发前在这里做好整备非常方便。如果搭乘公共交通，通常是坐公交到入口附近的车站后步行进入。具体的上下车位置，建议大家还是用地图 App 确认一下比较稳妥。

特点与推荐人群：净海拔爬升达 745 米，是实测路线中坡度较大的一段。前半程主要是林间小路和休息点，走起来压力不算大；但随着海拔升高，会进入落石危险区。因此，这条路更适合体力过硬、喜欢直接挑战顶峰的朋友。

路线 2 — 沙基马克谷 → 山顶

📏 12.02km ⛰️ 670m 爬升 ⏱️ 4:07 ➡️ 单程 难度 ★★★★★ ⤓ GPX

途经: 沙基马克谷 → 帕州屋 → 超市 (新道食品) → 沙基马克公园管理员站 → 洗手间 → 分岔路口 (注：往沙基马克谷方向无步道) → 环山小径里程标识 → 贴岩 (Butim-bawi) → 指路牌 → 山谷 → 国家地理坐标点 → 隐藏山谷下瀑布 → 休息处 → 隐藏瀑布下瀑布

入口说明: 关于沙基马克谷 (Sagimakgol) 入口，通过数据确认，起点附近确实有公交站。这条路线的走法是先经过公园管理员站和洗手间，穿过村里的商店街后，顺着山谷一路向上。

特点与推荐人群: 全长 12.02 公里，是所有登顶路线中长度最长的一段。如果你想在山谷和瀑布间长时间流连，这条线绝对是你的菜。不过因为距离较长，大家出发时务必规划好时间。如果你向往那种能听着潺潺水声、沉浸在森林里的悠闲登山体验，我强烈推荐这一条。

路线 3 — 分岔路口 → 山顶

📏 8.38km ⛰️ 607m 爬升 ⏱️ 3:06 ➡️ 单程 难度 ★★★★★ ⤓ GPX

经由: 分岔路口 → 清水瀑布 → 清水厕所 → 消火器休息站 → 永川药水井 → 双线药水井 → 保国门休息站 → 冬季易结冰路段 → 人豆岩 → 兄弟峰观景台 → 东岭瀑布 → 无名休息站 → 平昌公园护林区洗手间 → 平昌公园护林区 → 甘览山祈祷院

如何到达: 这条路线通往平昌与兄弟峰方向，通常的做法是从市区住宅区搭乘公交车，然后步行进入。山脚下设有清水厕所和“平昌公园护林区”洗手间。

特点与推荐人群: 这是一条水源非常充沛的路线，会经过永川和双线两处药水井；在兄弟峰观景台可以尽享开阔的视野。不过，由于路线上明确标注了“冬季易结冰路段”，冬天行走时务必要格外留心。

路线 4 — 北汉山城入口 → 北汉山白云台顶峰

📏 9.21km ⛰️ 671m 爬升 ⏱️ 3:25 🔁 环线 难度 ★★★★★ ⤓ GPX

经由: 北汉山城入口 → 北汉山城收费停车场 → 服务中心 → 北汉山城洗手间 → 分岔路 5 → 野生花景点 → 水口亭洗手间 → 西岩寺 → 分岔路 9 → 北汉洞历史馆旁洗手间 → 菩提寺 → 开渊瀑布 → 分岔路 16 → 大东寺药水亭 → 大东寺

入山交通: 北汉山城入口 (Bukhansanseong Ipgu) 是北汉山最经典的入山口，这里配套齐全，有服务中心、收费停车场和洗手间。而且公交线路非常发达，坐车就能直接到入口附近，公共交通很方便。

特点与推荐人群: 这是一条直通白云台顶峰的正统路线，沿途能看到城墙、寺庙、溪谷和山泉，景观丰富，对初次挑战北汉山的朋友来说非常友好。另外，因为沿途洗手间很多，无论是家庭出游还是外国游客，都可以放心走这条路。

路线 5 — 仙云山庄 → 东章台

📏 9.66km ⛰️ 611m 爬升 ⏱️ 3:26 ➡️ 单程 难度 ★★★★★ ⤓ GPX

途经: 仙云山庄 → 哈利路亚祈祷院停车场 → 分岔路口 → 分岔路休息区 → 北汉山钟岩 → 贴岩休息区 → 交叉口休息区 → 道仙茶园 → 乌伊广场洗手间 → 白云台登山支援中心 → 道仙寺指示牌 → 道仙寺洗手间 → 龙岩门公园护林站 → 溪谷休息区 → 金尚宫舍利塔（金尚宫庭光华舍利塔）

交通指南: 这条路线连接着乌伊与道仙寺方向，途中会经过乌伊广场洗手间和白云台登山支援中心，可达性很好。从市区北部搭乘公共交通过来也挺方便的。

特点与推荐人群: 路线将引向东章台方向的城墙脊线，沿途还能领略到道仙寺和舍利塔等文化景观。由于休息区分布非常密集，特别适合那些喜欢走一段歇一下，或者想在脊线上尽情欣赏城墙风景的朋友。

距离与耗时

图片来源：韩国观光公社旅游摄影画廊（首尔特别市恩平区佛光洞） · 公共领域

北汉山由于没有避难所，加上全线总长 17.47 公里，无论选哪条路线，作为一日往返的登山行程都绰绰有余。 实测显示，通往顶峰的单程时间大约在 2 小时 46 分钟（从国家基点编号到那汉峰）至约 4 小时 7 分钟（从思岐幕谷到顶峰）之间。即便考虑到往返和休息，绝大多数路线都能在一天之内搞定。

顺便提一下，数据上显示的累计用时为上行 342 分钟、下行 204 分钟，但这是所有路段的累计总和，并不是单条路线的时间。 不过，通过这个数值可以得出平均上行配速约为 20 分钟/公里。大家在规划行程时，可以用这个配速乘以自己选的路段长度，来预估大致的单程时间。比如，如果是 8 公里左右的路段，大约可以预估上行需要 2.5 到 3 小时。当然了，实际用时还得看个人的体力以及山顶岩石地带的拥堵程度。

路线指南（交通与停车）

图片：韩国观光公社旅游照片库 (首尔特别市恩平区佛光洞) · Gonggonuri

北汉山这地方，靠着首尔发达的地铁和公交系统，去起来还算方便。通常的做法是先坐地铁到市区某个站点，然后再换乘公交车前往登山入口。根据查到的数据，入口附近的公交站有：孝子2洞、사기막골 (Sagimakgol)、北汉山训练场、孝子治安中心。特别是 사기막골 站，能直接对接路线2的入口。

关于停车，虽然按林务厅的数据显示该范围内登记的停车场数量为 0，但根据路线4的路标可以确认有“北汉山城收费停车场”。如果你打算开车过来，由于不同入口处的停车场情况各异，出发前请务必先核实一下。另外，无论是查公交还是找路，我建议大家直接用 KakaoMap 或 Naver Map（毕竟在韩国，Google Maps 的步行和公交路径有时不太精准）。

季节 · 天气 · 装备

图片来源：韩国观光公社旅游摄影库（首尔特别市恩平区佛光洞） · 公共领域

推荐季节： 首选春季（4-5月），此时新绿满山，杜鹃花开；或是秋季（10-11月），可以欣赏到美不胜收的山脊红叶。夏天虽然溪谷清凉，但需警惕高温、突发阵雨以及湍急的水流；冬季则由于山顶岩石结冰和强风，体感难度会大幅上升。特别提醒：路线3的路标上明确标注了“冬季易结冰路段”，冬天登山务必倍加小心。

必备装备： 首先，一双抓地力强的登山鞋是基础。因为山顶有很多花岗岩岩面和铁栏杆路段，防滑非常关键。虽然设有直饮水处，但受季节和环境影响可能不出水，请务必备足充足的饮用水。冬天需要准备冰爪、防风及保暖衣物；夏天则要重点注意防晒和预防脱水。此外，由于沿途观景点非常多，带上相机或一件防风衣会很实用。

给外国游客的小贴士

图片来源：韩国观光公社旅游摄影馆 (首尔特别市恩平区佛光洞) · 公共누리

  • 地名罗马音对照：北汉山 Bukhansan · 顶峰白云台 Baegundae (约 834m) · 主要入口北汉山城入口 Bukhansanseong Ipgu · 赦幕谷 Sagimakgol · 友伊·道善寺方向 Doseonsa · 正陵 Jeongneung.
  • 登山许可与门票：无需提前申请许可，进入步道也无需缴纳入山费。不过，部分入口处的停车场是收费的（例如路线 4 的北汉山城收费停车场等）。
  • 厕所与饮水：在主要入口和溪谷沿线均设有厕所（如北汉山城厕所、清水厕所、平昌公园守护站厕所、灵鹫寺厕所等）。虽然确认有 5 处山间饮水点，但不能保证随时都有水，建议还是自带充足的饮用水。
  • 地图应用：由于 Google Maps 在韩国的步行及公共交通路线引导可能不够精准，建议大家使用 KakaoMap 或 Naver Map。
  • 安全须知：根据数据，岩石地带属于危险区域，图中标记了 5 处危险点。在顶峰岩壁、落石及结冰路段，请务必按照指示牌行走。由于山上没有避难所，请合理规划行程，务必在天黑前完成下山。这里共有 36 个观景点，大家可以在山脊各处安全地尽情俯瞰首尔市区的景色。
