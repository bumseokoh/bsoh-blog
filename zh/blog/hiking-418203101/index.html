概述

照片：韩国观光公社观光照片图库（京畿道加平郡上面物骨路120）· 公共누리

云岳山坐落在京畿道抱川市花岘面与加平郡下面的交界处，山顶海拔各资料记载略有出入，大约在935米左右，但按实测GPX数据来看，最高点其实达到948米。自古以来，这座山就与华岳山、冠岳山、紺岳山、松岳山并称”京畿五岳”，又因奇峰怪石与险峻岩脊构成的风景，被人称作”京畿小金刚”。整体而言，山上短而陡的岩石山脊路段颇多，对体力和专注力都有一定要求，不过大部分路线一天之内就能往返或纵走完毕，作为当日往返的登山行程绰绰有余。

山势与地形

照片：韩国观光公社观光照片图库（京畿道抱川市禾岘面禾东路184街39号）· 公共누리

云岳山与其说是坡度平缓的土山，不如说是一座由岩石主导山势的石山。它并非单一浑圆的山头，而是由西峰、东峰（主峰）、毗卢峰等多座岩峰沿山脊相连而成——光是公共数据里登记在册的主要山峰，就有云岳山西峰（962米）、东峰（934米）、云岳山毗卢峰（937.5米，别称929米）、西峰（918米）、儿峰（769米）、峨嵯山（513米）、圆通山（590米）等好几座，各具形态。其中从主峰东峰一带连到西峰的岩石棱线，正是云岳山登山路线的精华所在，眉毛岩、屏风岩、立岩之类各有其名的岩石沿棱线一字排开。南侧山坡上还坐落着相传创建于新罗时代的悬灯寺，山谷、寺庙与岩峰交相辉映，构成了这里独特的地貌景观。

登山的感觉

照片：韩国观光公社观光照片图库（京畿道加平郡明智山路137-49）· 公共누리

云岳山从入口开始就是陡峭的台阶和岩石路段接连不断，一上来难度就摆在那儿，让人不敢掉以轻心。爬上稜线后视野豁然开朗，抱川、加平一带的群山尽收眼底，尤其是屏风岩瞭望台和西峰附近，走在陡峭的岩稜上那种惊险刺激感特别过瘾。溪谷路段（通往玄等寺方向）夏天有树荫和水声相伴，走起来相对凉快，而稜线路段大多直晒太阳，所以季节不同体感差异会很明显。春天杜鹃花和新绿相映，秋天枫叶与岩稜相衬，都是有名的拍照打卡点；到了冬天，岩稜上常结薄冰，整趟山行的气氛也会变得格外严肃认真。

路线1 — 卢峙岭 → 云岳山西峰

📏 10.08km ⛰️ 681m 爬升 ⏱️ 3:39 ➡️ 单程 难度 ★★★★★ ⤓ GPX

沿途经过：卢峙岭 → 休憩处 → 圆通山 → 野猪出没痕迹 → 防滑注意路段 → 岔路口 → 倒木 → 汉北正脉标识 → 天然休息点 → 岩壁 → 枯木与松树 → 立岩 → 婆婆丁草（구절초，一种野菊） → 云岳山西峰 → 山脊眺望点

登山口交通：卢峙岭这个起点，说实话搭大众交通不太好到，一般得先坐往抱川或加平方向的市内公交到附近，再靠双脚走进去。没查到像样的大型停车场，多半只能停在路边或附近的空地上，将就一下。

特点与推荐人群：这条经圆通山通往西峰的路线，是五条线里最长的一条，山脊路段也拉得比较开，比起又短又陡的岩脊，更适合想走那种缓急交替、有点纵走感觉的朋友。沿途实测经过的点里既有野猪出没的痕迹，也有几处要小心防滑的地方，所以更推荐给已经有点登山经验的人。

路线2 — 悬灯寺入口 → 屏风岩瞭望台

📏 6.24km ⛰️ 742m 爬升 ⏱️ 2:48 ➡️ 单程 难度 ★★★★★ ⤓ GPX

经过点：云岳山下板里 → 公共卫生间 → 悬灯寺入口 → 岔路口 → 有情冷面 → 悬灯寺旅游服务中心 → 简易卫生间 → 眉岩 → 路标前休息处 → 屏风岩瞭望台 → 云岳山毗卢峰(937.5m) → 山顶下休息处 → 悬灯寺泉水 → 悬灯寺卫生间 → 悬灯寺简易卫生间

登山口交通：横跨下板里区段的悬灯寺入口，是云岳山各条路线里公共交通最方便的一段。开往抱川或加平方向的巴士都会经过悬灯寺入口附近的站点，下车后沿寺庙进山路步行即可。据说悬灯寺旅游服务中心附近还留有停车位。

特点及推荐对象：这条路线沿途分布着公共卫生间、简易卫生间、悬灯寺泉水和卫生间等设施，是五条路线里配套最密集的一段。途经眉岩和屏风岩瞭望台，能在短短6.24公里内，浓缩体验一把云岳山特有的岩脊景观，特别适合第一次来云岳山的游客，或是专门为拍照而来的山友。不过要注意的是，这条路线相对距离而言累计爬升（742米）在五条路线中是最大的，所以从一开始坡度就会显得比较陡。

路线3 — 登山口 → 阿姬峰

📏 6.62km ⛰️ 687m 爬升 ⏱️ 2:48 🔁 环线 难度 ★★★★☆ ⤓ GPX

沿途：云岳山休息站站 → 云岳山露营地 → 云岳山停车场 → 云岳山休息站 → 登山导览图 → 示范厕所 → 登山口 → 咖啡厅及餐厅 → 岔路口 → 云岳山休养林 → 咖啡店 → 国家地点编号 → 生态厕所 → 云岳寺 → 岩脊路段

登山口交通：顾名思义，途中就有”云岳山休息站站”这一站，所以从公交车站下来就能直接走到登山口。路线中还标出了云岳山停车场作为路标点，对自驾游客来说也是最方便的起点之一。

特点与推荐人群：这条路线要穿过休息站、露营地和餐饮一条街才进入正式的登山道，所以开头一段带着浓浓的商业气息，跟自然景观衔接得挺自然，示范厕所、生态厕所之类的卫生设施也算齐全。在这五条路线里，它是唯一被标为★★★★☆难度的一条，相对轻松一些，很适合带孩子的家庭或刚开始爬云岳山的新手。不过到了后半段的岩脊路段，跟其他路线一样，还是少不了要手脚并用的地方。

路线4 — 阿嗤山→顶峰

📏 8.28km ⛰️ 638m 爬升 ⏱️ 3:08 ➡️ 单程 难度 ★★★★★ ⤓ GPX

经过点：阿嗤山 → 路边停车 → 岔路口 → 山体滑坡危险区 → 败酱草 → 韩北正脉交汇点 → 野韭菜 → 眺望水原山 → 眺望云岳山 → 山罗花 → 独木桥 → 眺望鹫峰 → 危险区 → 眺望龟木峰·恋人山·明智山 → 眺望칼峰山脊

登山口交通：阿嗤山（513米）方向没有专门的大型停车场，实地记录的经过点是”路边停车”，也就是说大家一般都是在路边找块地方随便停车再进山。如果打算坐公交，建议先确认清楚从附近站点下车后要走多远才能接上登山口。

特点·适合人群：这条路线的经过点里特别标出了”山体滑坡危险区”和”危险区”两处，刚下过雨或结冰季节走的话要格外小心。不过好处是能连续看到龟木峰、恋人山、明智山、칼峰山脊，还有水原山、云岳山的眺望点接连出现，对喜欢沿着山脊看风景的人来说很有吸引力。沿途还记录了败酱草、野韭菜、山罗花这些野花，正好可以边走边看看季节花草。

路线5 — 起点 → 阿奇峰

📏 10.41km ⛰️ 658m 爬升 ⏱️ 3:42 ➡️ 单程 难度 ★★★★★ ⤓ GPX

沿途经过：新上桥 → 云岳山 → 文化花园 → 花儿更美（花보다） → 有停车位 → 岔路口 → 停车空地 → 停车位 → 起点 → 停车空地 → 岩石与松树 → 天然休息处 → 白虎稜眺望点 → 支石墓岩 → 来时路

起点交通：这条路线属于新上里区间，以新上桥为起点。沿途反复出现”有停车位""停车空地""停车位”这类标注，看得出这个起点对自驾还算友好。如果打算搭公共交通，建议提前查一下附近站点到新上桥要走多远。

路线特点·推荐对象：这是5条路线里最长的一条，全程10.41公里，沿途经过白虎稜眺望点、支石墓岩这些颇有特色的地形，一路爬到阿奇峰（769米）。累计爬升658米，在五条路线里算中等，但胜在路程长，得留着体力慢慢耗，比较适合能空出一整天、不赶时间的山友。

难度·路面·注意事项

사진: 한국관광공사 관광사진 갤러리 (경기도 포천시 화현면 봉화로 201-65) · 공공누리

在辖区登山路线空间信息中，云岳山的最高难度等级被归为”中等”，但根据实测GPX数据，通往顶峰区域的5条路线大多显示为★★★★☆以上的高难度，实际体感难度可能比标注等级要高不少。虽然没有单独登记各路线的详细路面信息，但从西峰、屏风岩、眉毛岩路段的实测经由地名称就能看出，岩石棱线是这座山的主要地形特征。特别是4号路线（阿其山→山顶）登记了”滑坡危险区域”和”危险区域”两个点位，而整个云岳山登记在案的危险区域一共也只有2处（危险提示、危险警告各1处），也就是说风险几乎都集中在这条路线上。雨后岩石会变滑，冬季背阴的岩棱路段也常常结冰，经过这一段时最好放慢脚步，保持三点支撑。虽然没有单独标注观景点，但从实测经由地来看，屏风岩观景台、稜线眺望、水原山与云岳山远眺等观景点其实不少。

距离与所需时间

照片：韩国观光公社观光图片库（京畿道抱川市禾岘面禾东路548-47）· 公共누리

云岳山辖区内登山道总长12.89公里（是多段路线的合计数值），上山258分钟、下山180分钟这两个数字同样是全辖区路段累加所得，并非某一条具体路线的耗时。用这个累计值除以距离，平均上山配速约为每公里20分钟，比平地步行慢出不少，这也恰好说明了云岳山多岩壁、多陡坡的地形特点。实际上，各代表路线的单程耗时（依据GPX实测）从约2小时48分钟（玄灯寺入口·登山道入口一线）到3小时42分钟（登山口→阿姬峰）不等，来回一趟大致要预留5到7个小时。辖区内没有登记在册的避难小屋，所以这五条路线都完全可以当天往返走完，除非打算走纵走线路，否则不需要过夜住宿。

交通方式（大众交通・停车）

照片：韩国观光公社观光照片图库（京畿道抱川市花岘面峰火路515）· 公共누리

云岳山位于京畿道抱川市与加平郡的交界处，从首尔出发的话，开车比坐大众交通要方便不少。如果一定要坐车，一般是从东首尔综合客运站或清凉里方向换乘开往抱川、加平的市外/市内巴士，在”云岳山入口”、“云岳山·悬灯寺入口”、“抱川·云岳山入口”、“大院寺”等站下车，再步行到登山口。其中”云岳山·悬灯寺入口”站正好衔接路线2（悬灯寺入口→屏风岩观景台）。目前没查到景区内有正规登记的停车场，多半得停在悬灯寺或云岳山休息站附近的私人小停车位。具体的班车时刻、换乘路线和实时路况，建议用KakaoMap或Naver Map搜”云岳山”或站名确认一下。外国游客的话，Google Maps在韩国境内的公交和登山路线信息经常不准，还是老老实实用上面这两个App比较保险。

季节·天气·装备

照片：韩国观光公社观光照片图库（京畿道加平郡朝宗面云岳清溪路702-24）· 公共누리

去云岳山最舒服的季节要数秋天了，枫叶正艳，气温也凉爽，正适合在岩脊上爬上爬下。春天杜鹃花开得最盛的时候人会特别多，这时候天气干燥，山火风险也得留意。夏天暑热潮湿，还常有突如其来的阵雨，岩脊路段一淋湿就变得很滑，所以最好带上雨衣和备用衣物。冬天背阴的稜线和岩石地带经常结冰，五条路线的难度都会明显上升，这时候冰爪、绑腿、防寒手套基本就是必备装备了。不论哪个季节，登山鞋（要选抓地力好、能应付岩脊的）、登山杖、充足的饮用水（这一带没有登记在册的饮水点，务必提前备好）、手套、头灯（以防下山耽误到天黑）这几样基本装备，都建议带齐。

外国游客小贴士

照片：韩国观光公社观光照片图库（京畿道抱川市花岘面达仁洞路35）· 公共누리

云岳山的英文标注为”Mount Unak”或”Unaksan”，山顶部分在地图应用中可以搜索”Unaksan Peak”或”东峰(Dongbong)“来确认位置。主要登山口之一的悬灯寺入口，可以搜索”Hyeondeungsa Temple entrance”找到。这里不属于国立公园，而是由山林厅管辖的一般登山道，因此不需要额外的入山许可或门票，不过路过悬灯寺境内时，还是建议对寺庙参观保持应有的礼貌。厕所方面，2号路线（悬灯寺方向）沿途设有不少公共厕所、简易厕所和寺庙的”解忧所”，但其余山脊路段基本没有合适的厕所，建议在登山口先解决。由于登记在册的饮水点为零，山行前务必自备足够的水；虽然悬灯寺附近的一处泉眼在实测路线图上有标注，但是否随时可用，最好到现场再确认一下比较安全。前面也提到过，地图应用建议用KakaoMap或Naver Map，比谷歌地图准确得多，对确认登山道实时位置也很有帮助。这里岩石陵线路段较多，雨天或结冰季节最好避免登山，或务必携带冰爪等装备；被标注为危险地带的两处地点（4号路线附近）一定要遵守指示牌上的警告。手机信号在大部分山脊区域都还不错，但考虑到这里并非国立公园，还是建议提前把登山计划和预计下山时间告知同行者或住宿处，以防万一。
