概览

图片：韩国旅游发展局摄影画廊（全南光州统合特别市华顺郡） · 公共领域

无等山 (Mudeungsan) 横跨光州广域市东区、北区与全罗南道华顺郡，是湖南地区非常有代表性的国家公园。由于山顶设有军事通信设施，游客无法登顶，大家通常只能走到西锡台 (Seoseokdae) 或立石台 (Ipseokdae) 等柱状节理地带附近。

辖区内的登山步道总长度（是多段路段的总和，并非单条线路的距离）为 13.55 公里。虽然根据林务厅的数据，登记路段中的最高难度也仅被划分为“简单”，但如果按照“百名山”实测的 GPX 路径来看，通往核心区域的主干线单程距离在 5 到 9 公里之间，累计爬升高度也达到了 450 到 780 米。这体力消耗确实不小，不过作为一天的登山行程，完成起来还是绰绰有余的。

山势与地形

图片来源：韩国观光公社观光摄影库 (全南光州统合特别市华顺郡) · Gonggonuri

无等山拥有一种非常独特的地理地貌：在平缓的土质山脊之上，错落有致地分布着规模宏大的岩峰与柱状节理区。顶部的西石台与立石台由如屏风般排列的石柱组成，因其极高的学术与景观价值，被广泛认可并指定为国家地质公园。

根据公开数据记载，山上的主要峰峦包括：万延峰 (628m)、累峰 (1084m)、骆驼峰 (938m)、安养山 (876m)、万延山 (687m)、元晓峰 (583m) 以及中头岭 (617m，也有记录为 613m)。而根据本次行程的 GPX 实测数据，路线所能到达的最高点为 1120m。虽然天王峰 (约 1187m) 通常被视为整座山的海拔最高点，但由于设有通信设施限制出入，大多数登山者往往会将西石台一带视为实际的“登顶体验地”。此外，地空乱石堆（碎石坡区域）以及规峰庵周边的岩石带，也是这座山极具特色的地貌景观。

登山感受

사진: 한국관광공사 관광사진 갤러리 (전남광주통합특별시 화순군) · 공공누리

刚开始走的时候，一路上都在茂密的柏树林与阔叶林间穿行，山路平缓且顺着山脊延伸，走起来还算轻松。但随着海拔不断升高，碎石滩与岩石地带开始交替出现，体感难度也就跟着上来了。在中峰和长不岭附近，大片的芒草群与开阔的山脊视野能让人真切感受到季节的流转；而到了西石台和立石台附近，柱状节理岩峰拔地而起，那种豁然开朗的视觉冲击感非常强。秋天可以赏芒草与红叶，冬天则是冰封的山脊与晶莹的雾凇，这两个季节都是极佳的观赏时节。

路线 1 — 元晓寺入口公交站 → 元晓分所山岳救援队

📏 6.75km ⛰️ 734m 爬升 ⏱️ 2:55 ➡️ 单程 难度 ★★★★★ ⤓ GPX

经由: 元晓寺停车场 → 无等山休息处 → 元晓寺入口公交站 → 元晓寺停车场洗手间 → 松岛山庄 → 无等山走起(Mudeungsan Gaja) → 分岔路口 → 元晓寺分岔路洗手间 → 元晓分所咨询处 → 元晓分所山岳救援队 → 元晓分所饮水处 → 元晓地区餐厅街 → 柏树林 → 꼬막재 → 陡坡路段

如何抵达: 搭乘开往元晓寺方向的市郊公交，在“元晓寺入口”站下车后步行即可进入。虽然附近有停车位，但旺季时可能会挤得不行，建议大家还是尽量搭乘公共交通。

路线特点·推荐人群: 这条线路前半段穿过柏树林和元晓地区餐厅街，基本是平缓的土路；后半段翻过 꼬막재 后就是连续的陡坡了。如果你既想在柏树林里悠闲散步，又想体验一把真正的登山乐趣，这条路线正合适。

路线 2 —— 道园村停车场入口 → 立石台

📏 7.73km ⛰️ 775m 爬升 ⏱️ 3:13 ➡️ 单程 难度 ★★★★★ ⤓ GPX

经由: 道园村游客专用停车场 → 道园村停车场入口 → 道园村指示牌 → 分岔路口 → 道园探访支援中心 → 洗手间 → 道园村-规峰庵路段指示牌 → 休息区 → 国家位置编号 → 规峰庵 → 规峰庵祈祷处 → 规峰庵入口岩石区 → 地空乱石滩 → 石佛庵 → 石佛庵前指示牌

入山方式: 基本上只能通过道园村的游客专用停车场进入，比起搭乘公共交通，开车过来才是最方便的做法。

路线特点·推荐对象: 这条线路会穿过规峰庵和地空乱石滩等岩石地形，让你在领略古刹禅意的同时，也能一睹乱石滩的奇景。由于路程较长且海拔落差大，体力消耗比较可观，因此特别适合那些既想感受静谧山寺氛围，又渴望挑战硬核岩脊徒步的爱好者。

路线 3 —— 分岔路口 → 顶峰

📏 6.79km ⛰️ 610m 爬升 ⏱️ 2:43 ➡️ 单程 难度 ★★★★☆ ⤓ GPX

经由：分岔路口 → 道原村-长不宰指示牌 → 休息处 → 国家坐标号 → 道原村-长不宰区间路标 → 中峰下方观景点 → 道原露营地禁入处 → 宽石龙潭指示牌

如何到达：因为这段路是从其他路线的中途分岔口开始的，并没有专门的车站对应，所以大家通常会顺着通往道原村的路径进入。

特点与推荐人群：这是一段短而高效的路线，在中峰下方的观景点可以尽情欣赏山脊线的壮丽景观。非常适合想要与其他路线串联进行长距离徒步，或是时间有限的登山爱好者。

路线 4 — 元晓寺入口 → 山顶

📏 9.39km ⛰️ 576m 爬升 ⏱️ 3:18 🔁 环线 难度 ★★★★★ ⤓ GPX

经由: 分岔路 → 国家坐标编号 → 休息站 → 旧路第2段指示牌 → 中峰爱情小径 → 芒草拍照点 → 中峰芒草景观区 → 落石注意段 → 结冰路段 → 无等山冰岩 → 路标 → 晚载休息站 → 元晓寺门 → 元晓寺入口

如何抵达: 交通方式与路线 1 一样，搭乘前往元晓寺方向的公交车即可到达。

特点与推荐对象: 这条路线会经过芒草拍照点和中峰观景台，风景很美。但由于包含落石危险区和结冰路段（无等山冰岩），冬季行走时务必要格外留神。如果你想一览秋天那片浪漫的芒草景观，我非常推荐这条路线。

路线 5 — 野菊村入口 水万里3区车站 → 西石台

📏 8.21km ⛰️ 591m 爬升 ⏱️ 3:02 ➡️ 单程 难度 ★★★★☆ ⤓ GPX

途经点: 野菊村游客专用停车场 → 野菊村入口 水万里3区车站 → 野菊村 → 野菊村停车场洗手间 → 分岔路口 → 登山步道入口指示牌 → 野菊村-长佛岭路标 → 国家地理坐标点 → 休息区 → 棱线景观 → 杜鹃花群落 → 落石注意 → 陡坡落石危险区 → 白马棱线小径 → 棱线岩路标

进山方式: 从水万里3区车站下车后即可步行进入，也可以利用附近的游客专用停车场。

特点与建议: 这是一条穿越杜鹃花群落和白马棱线小径的脊线路线，尤其在春季杜鹃花盛放时人气极高。由于路段包含陡坡落石危险区，雨天行走时请务必格外小心。

路线 6 — 无等山增心寺 → 中峰

📏 7.72km ⛰️ 768m 爬升 ⏱️ 3:13 🔁 环线 难度 ★★★★★ ⤓ GPX

经由: 无等山增心寺 → 分岔路口 → 欧邦修养院 → 当山树松风亭休憩处 → 当山树 → 当山树厕所 → ⚠️注意野猪 → 中目里岭 (Jungmeorijae) 登山路休憩处 → 雷电多发区 → 中峰 (Jungbong) → 中峰直升机坪 → KBC 发射台 → 东华寺址休憩处 → 迟才 (Neutjae) 展望台 → 路面损毁路段

入口交通: 增心寺这一带是出了名的方便，从光州市区坐公交就能轻松抵达，附近也备有停车场。

特点与推荐人群: 这条路线穿过增心寺的林区和当山树，一路向上直达中目里岭与中峰，是光州市民最爱走的经典路线。由于路线上标有雷电多发区，夏季要是遇到雷阵雨，建议大家尽早下山。因为它兼具便利性与代表性，非常适合初次登山的游客。

难度、路面与注意事项

图片来源：韩国观光公社旅游摄影库（全南光州统合特别市华顺郡） · 公共 누리

以下是按行政区划划分的登记路段（此处为辖区内总长度区分，而非具体的登山路线名称）：

路段（按行政区划）总长度难度路面状况
无岛里路段 (무도리구간)5.87km简单粘土质
无岛里谷坡 - 无岛里切谷路段 (무도리골안고개-무도리절골구간)5.4km简单粘土质
土桥里路段 (토교리구간)2.28km简单粘土质

虽然从林务厅的数据来看，登记路段的难度被定为“简单”，但那仅仅是针对低海拔区间的总长度而言。要是真按照通往顶峰的 GPX 实测路线来看，由于海拔落差和路程距离的关系，难度绝对是顶级的 ★★★★★。

路面多为粘土质，下雨时非常湿滑，请务必注意防滑。此外，路线上的特定位置也存在实际风险：如落石预警（路线 4、5）、陡坡落石风险（路线 5）、冰冻路段及茂登山冰岩区（路线 4）以及雷电多发区（路线 6）。由于还存在陡坡或登山道损毁路段（路线 6），雨后务必先确认路面状况。

目前路段内仅确认有一处饮水点（元晓分所饮水处）。由于并未登记避难所或专门的危险/观景点 POI，建议大家参考现场指示牌及咨询处的信息。

距离与用时

图片来源：韩国观光公社旅游摄影画廊 (全南光州统合特别市华顺郡) · 公共领域

如果按照山内登山道累计上坡 239 分钟、下坡 170 分钟（注意，这是全段累计数值，并非单条路线的耗时）来换算，平均上坡配速大约是每公里 18 分钟。由于大多数通往山顶的路线单程长度在 5 至 9 公里之间，净爬升高度约为 450 至 780 米，按照这个配速计算，单程用时通常在 2 小时到 3 小时 20 分钟左右。往返一趟可能需要 6 到 8 小时，建议大家还是尽早出发比较稳妥。不过，由于山内没有避难所，且全程总长仅为 13.55 公里左右，作为单日徒步行程还是完全没问题的。

交通指南（公共交通·停车）

사진: 한국관광공사 관광사진 갤러리 (전남광주통합특별시 화순군) · 공공누리

在光州市区，大家通常更习惯乘坐公交车而不是地铁。如果目的地是元孝寺方向，可以坐到“无等山国立公园（元孝寺）”站；如果是去证心寺方向，则可以乘坐往证心寺区的公交线路。要去水万里的话，请在“野菊村入口 水万리3구”站下车。由于具体的公交线路和班次可能会随出发时间而变动，建议大家务必在 KakaoMap 或 Naver Map 上查看实时路线。此外，根据公开数据统计，登记的停车场显示为 0 个（虽然像道园村、水万里等地方也有一些针对游客的小型停车空间，但非常有限），因此在周末或旅游旺季，还是建议大家乘坐公共交通。

季节 · 天气 · 装备

图片来源：韩国观光公社 观光摄影画廊（全南光州统合特别市 和顺郡） · Gonggonuri

春天的杜鹃，秋天的银杏与红叶，是这里最迷人的风景，也是登山的最佳季节。夏季需警惕雷电风险，若预报有午后雷阵雨，建议避开登山或尽早下山以保安全。冬季则会有结冰路段和冰岩，务必准备好冰爪、护腿等防滑保暖装备。由于地面多为黏土质，雨后路面湿滑，建议穿好登山鞋并配备登山杖。此外，无论什么季节，准备充足的水和零食都是必不可少的。

给外国游客的小贴士

图片来源：韩国观光公社旅游摄影画廊（全南光州统合特别市华顺郡） · 公共Nuri

无等山 (Mudeungsan) 的标志性景点包括西石台 (Seosokdae)、立石台 (Ipseokdae) 和中峰 (Jungbong) 等。这里作为国家公园，通常不需要专门的入山许可或门票，但需要注意的是，由于山顶部分区域设有通信设施而限制进入。洗手间分布在元晓寺停车场、元晓寺分岔路口、桃园探访支援中心、寿万里野菊村停车场以及当山树附近等多个地点。关于饮用水，虽然可以参考元晓分所的饮水处（只有一个水龙头），但整体来看补给并不算充足，为了安全起见，建议大家还是自己多备点水。另外，由于 Google Maps 在韩国的登山路线信息往往不够准确，建议大家在当地使用 KakaoMap 或 Naver Map。山里会有落石、结冰路段和雷电多发区的提示牌，请务必听从指示。此外，由于山上没有避难所，一定要合理规划好行程，确保在日落前下山。
