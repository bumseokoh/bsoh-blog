概述

照片：韩国旅游发展局旅游照片图库（全南光州统合特别市和顺郡）· 公共发布

无等山（Mudeungsan）横跨光州广域市东区、北区和全罗南道和顺郡，是代表湖南地区的国立公园。山顶因设有军事通信设施而禁止进入，一般登山者可攀登至瑞石台、立石台等柱状节理地带附近。园区内登山道总长度（多段合计值，并非单一路线的距离）为 13.55 公里，按林业厅数据，已登记路段的最高难度被划为”简单”。不过，若以百大名山实测 GPX 为准，攀登至峰顶一带的几条代表性路线，单程达 5～9 公里、净高差 450～780 米，体力消耗相当可观，当天往返即可轻松走完全程。

산세와 지형

照片：韩国观光公社观光照片图库（全南光州统合特别市和顺郡）· 公共누리

无等山地形独特：在平缓土山的山脊之上，叠着颇具规模的岩峰和柱状节理地带。山顶的瑞石台、立石台是石柱如屏风般排列的柱状节理，被指定为国家地质公园，学术价值与景观价值都很高，广为人知。据公共数据登记的主要山峰有万渊峰（628m）、蚕峰（1084m）、骆驼峰（938m）、安养山（876m）、万渊山（687m）、元晓峰（583m）、中머리재（617m，视测点不同也有记为613m的），而以实测GPX为准，这几条路线所到达的最高点确认为1120m。一般来说，全山最高峰是天王峰（约1187m），但因设有通信设施而限制出入，登山客往往把瑞石台一带当作实际意义上的登顶体验点。지공너덜（碎石堆积成的乱石坡路段）和圭峰庵周边的岩石地带，也是这座山颇具代表性的地形。

登山的感受

照片：韩国观光公社观光照片画廊（全南光州统合特别市和顺郡）· 公共누리

登山初段是柏树林与阔叶林郁郁葱葱的土路，加上坡度平缓的山脊路，走起来负担不大；但随着海拔升高，碎石坡与岩脊路段夹杂出现，体感难度也随之提升。走到中峰、장불재一带，芒草群落和山脊视野豁然开朗，季节感格外分明；而到了瑞石台、立石台附近，柱状节理的岩峰在眼前铺陈开来，视野开阔通透。秋天有芒草和红叶，冬天则是山脊的结冰与雾凇，这些往往被视为最令人难忘的看点。

线路 1 — 元晓寺入口公交站 → 元晓分所山岳救援队

📏 6.75km ⛰️ 734m 爬升 ⏱️ 2:55 ➡️ 单程 难度 ★★★★★

途经：元晓寺停车场 → 无等山休息区 → 元晓寺入口公交站 → 元晓寺停车场卫生间 → 松岛山庄 → 무등산가자 → 岔路口 → 元晓寺岔路口卫生间 → 元晓分所服务处 → 元晓分所山岳救援队 → 元晓分所饮水点 → 元晓地区餐饮街 → 扁柏林 → 蚶岭 → 陡坡路段

登山口交通：有开往元晓寺方向的市内公交，在元晓寺入口站下车后步行进入即可。附近设有停车位，但旺季可能会比较拥堵，推荐乘坐公共交通前往。

特色与推荐人群：起步是一段穿过扁柏林和元晓地区餐饮街的平缓土路，后半段翻越蚶岭时则是连续的陡坡。适合既想体验扁柏林漫步、又想来一场正经登山的朋友。

线路 2 — 桃源村停车场入口 → 立石台

📏 7.73km ⛰️ 775m 爬升 ⏱️ 3:13 ➡️ 单程 难度 ★★★★★

途经点：桃源村访客专用停车场 → 桃源村停车场入口 → 桃源村导览牌 → 岔路口 → 桃源探访支援中心 → 洗手间 → 桃源村—圭峰庵路段指示牌 → 休息区 → 国家地点编号 → 圭峰庵 → 圭峰庵祈祷处 → 圭峰庵入口岩石 → 指空乱石坡 → 石佛庵 → 石佛庵前指示牌

登山口交通：几乎只能利用桃源村访客专用停车场进入，比起公共交通，自驾前往更为普遍。

特色·适合人群：这条线路会经过圭峰庵、指空乱石坡等岩石地形，寺庙与乱石坡尽收眼底。距离和高差都不小，体力消耗偏大，适合既想感受静谧山寺氛围、又想来一场正儿八经的岩脊徒步的人。

线路 3 — 岔路口 → 山顶

📏 6.79km ⛰️ 610m 爬升 ⏱️ 2:43 ➡️ 单程 难度 ★★★★☆

途经：岔路口 → 桃源村獐佛岾指示牌 → 休息区 → 国家地点编号 → 桃源村—獐佛岾路段路标 → 中峰下方观景点 → 桃源营地禁止通行 → 宽石龙沼指示牌

登山口接近方式：这一段是从其他线路中途的岔路口出发的，没有明确的独立站点，因此大多会结合桃源村方向的接近路进入。

特色与推荐人群：这是一段短小而高效的路段，可以在中峰下方的观景点欣赏山脊全景。适合与其他线路衔接做纵走，或时间有限的登山者。

路线 4 — 元晓寺入口 → 山顶

📏 9.39km ⛰️ 576m 爬升 ⏱️ 3:18 🔁 环线 难度 ★★★★★

途经：岔路口 → 国家地点编号 → 休息区 → 古道第 2 段路标 → 中峰爱之路 → 芒草路摄影点 → 中峰芒草路观景处 → 落石注意 → 结冰路段 → 无等山冰岩 → 路标 → 晚岭休息区 → 元晓寺出入口 → 元晓寺入口

登山口接驳：可乘坐与路线 1 相同的开往元晓寺方向的公交车抵达。

特点·推荐人群：这条路线会经过芒草路摄影点和中峰观景处，不过途中有落石注意路段和结冰路段（无等山冰岩），冬季尤其要格外小心。推荐给想欣赏秋日芒草风光的登山者。

线路 5 — 野菊花村入口 水万里3区站 → 瑞石台

📏 8.21km ⛰️ 591m 爬升 ⏱️ 3:02 ➡️ 单程 难度 ★★★★☆

途经：水万里野菊花村访客专用停车场 → 野菊花村入口 水万里3区站 → 水万里野菊花村 → 水万里野菊花村停车场卫生间 → 岔路口 → 登山道入口指示牌 → 野菊花村-场佛岭路标 → 国家地点编号 → 休息处 → 山脊眺望 → 杜鹃花群落地 → 注意落石 → 陡坡落石危险 → 白马山脊路 → 能仙岩路标

登山口接驳：可在水万里3区站下车后步行进入，也可利用附近的访客专用停车场。

特色·推荐人群：这是一条穿越杜鹃花群落地和白马山脊路的山脊型线路，尤其在春季杜鹃花开花期人气很旺。途中有陡坡落石危险路段，雨天需格外小心。

코스 6 — 무등산 증심사 → 중봉

📏 7.72km ⛰️ 768m 爬升 ⏱️ 3:13 🔁 环线 难度 ★★★★★

沿途经过：无等山证心寺 → 岔路口 → 五方修炼院 → 堂山树松风亭休息区 → 堂山树 → 堂山树公厕 → 野猪出没注意 → 中头岭上坡休息区 → 雷击多发区 → 中峰 → 中峰直升机坪 → kbc 发射台 → 桐华寺遗址休息区 → 晚岭观景台 → 山路损毁路段

登山口交通：证心寺片区是从光州市区乘市内公交最方便抵达的登山口之一，广为人知，附近还设有停车场。

特点·推荐人群：这条路线穿过证心寺的寺院林和堂山树，一路登上中头岭、中峰，是光州市民最爱走的招牌路线。沿途标注有雷击多发区，夏季雷雨时建议下撤。兼具交通便利与代表性，很适合推荐给初次来访的人。

难度·路面·注意事项

照片：韩国观光公社观光照片馆（全南光州统合特别市和顺郡）· 公共누리

以行政洞为准登记的路段（并非登山路线名，而是辖区内总里程的划分）如下。

路段（行政洞为准）总里程难度路面
舞道里路段5.87km容易黏土质
舞道里谷安岭—舞道里绝谷路段5.4km容易黏土质
土桥里路段2.28km容易黏土质

按林业厅的数据，登记路段本身的难度是”容易”，但这只是以低海拔路段的总里程为准。实际爬到接近山顶的 GPX 实测路线，因净海拔差和距离的关系，最高会达到 ★★★★★ 的难度。路面多为黏土质，下雨时尤其要当心打滑；各条路线的途经点也确实立有危险警示牌，比如落石注意（路线 4、5）、陡坡落石危险（路线 5）、结冰路段·无等山冰岩（路线 4）、雷击多发区（路线 6）等。还有陡坡和登山道被冲毁的路段（路线 6），所以雨后一定要确认路面状况。路线内确认有一处饮水点（元晓分所饮水处），但除避难所以外，并没有单独登记危险区域和观景点的 POI，最好参考路上的指示牌和咨询处的信息。

距离与耗时

照片：韩国观光公社观光照片画廊（全南光州统合特别市和顺郡）· 公共누리

以区域内登山路上行累计239分钟、下行累计170分钟（这是全部路段的合计数值，并非单条线路的耗时）为基准换算，平均上行配速约为18分钟/公里。顶峰一带的线路大多单程59公里、净海拔差450780米，套用这一配速，单程通常在2小时上下到3小时20分钟左右。往返可能需要6~8小时上下，因此建议尽早出发；不过这里没有避难所，区域内总里程也就13.55公里，规模不大，作为当日往返的登山完全可以走完全程。

前往方式（公共交通·停车）

照片：韩国旅游发展局观光照片图库（全南光州统合特别市和顺郡）· 公共누리

在光州广域市的市中心，比起地铁，坐市内公交更为普遍。去往元晓寺方向，可以坐到”无等山国立公园（元晓寺）“站；而增心寺方向，通常是搭乘开往增心寺片区的公交前往。要去水满里方向，则在野菊花村入口的水满里三区站下车。具体的公交线路和发车间隔会随出发时间而变，所以出发前最好用 KakaoMap 或 Naver Map 确认一下实时路线。公共数据里登记的停车场数量为 0 个（虽然有桃源村、水满里访客专用停车场之类的本地小型车位，但相当有限），因此周末和旺季还是建议乘坐公共交通。

시즌·날씨·장비

사진: 한국관광공사 관광사진 갤러리 (전남광주통합특별시 화순군) · 공공누리

春天的杜鹃、秋天的芒草和红叶，是这里最具代表性的景致，因此春秋两季被公认为最适合登山的时节。夏天这一带被确认为落雷多发区，所以遇到午后雷雨预报时，最好不要上山，或者提早下山，这样才安全。冬天则确实存在结冰路段和冰岩地带，需要备好冰爪、雪套等防寒装备。由于路面是黏土质，雨后地面很滑，最好带上登山鞋和登山杖防滑。另外无论哪个季节，都要备足饮水和干粮。

给外国旅行者的小贴士

照片：韩国观光公社观光照片图库（全南光州统合特别市和顺郡）· 公共누리

无等山的罗马字标记为”Mudeungsan”，代表性景点有 Seosokdae（瑞石台）、Ipseokdae（立石台）、Jungbong（中峰）等。这里属于国立公园范围，据了解无需另行申请入山许可，也不收门票，不过顶峰的部分区域因设有通信设施而限制出入。厕所分布在多个地点，包括元晓寺停车场、元晓寺岔路口、桃源探访支援中心、水满里野菊花村停车场以及堂山树附近等；饮用水方面可参考元晓分所的饮水点（一处饮水台），但整体上供水设施不多，最稳妥的做法还是自己备足饮用水。谷歌地图对韩国境内登山路线的信息往往不够准确，建议在当地改用 KakaoMap 或 Naver Map 应用。山上确实设有”注意落石""结冰路段""雷击多发区”等警示牌，请务必遵照牌上的提示行动；由于没有避难所，重要的是提前规划好在日落前下山。
