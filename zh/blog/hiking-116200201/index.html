概要

冠岳山（Gwanaksan，관악산）横跨首尔特别市冠岳区、衿川区与京畿道果川市、安养市，是一座海拔约632米的岩石山，按实测路线数据，山顶一带最高海拔为657米。山顶莲主台（Yeonjudae）上有一座建在奇岩绝壁之上的小庵（应真殿），在首尔近郊的群山中，这里的山顶景观堪称最具戏剧性。登山路线数据显示最高难度为”中等”，区内登山道总长达43.69公里（这是多条路线的合计，并非单条路线的距离）。得益于从地铁站出来就能直达登山口的便利交通，这座山完全可以当天往返。

山势与地形

冠岳山是一座典型的花岗岩石山。棱线上随处可见裸露的花岗岩石峰和坡面（slab，倾斜的岩面），路面数据中主要路面也被记录为”岩石”。代表性山峰是山顶的恋主台（연주대），以及旁边那座顶着气象观测雷达圆顶的山峰；往西是八座石峰如锯齿般连绵的八峰棱线（Palbong Ridge），西北方向则伸展着鹤岩棱线和紫云庵棱线。公共数据里登记了鸟岩、场院岩、龟岩等观景岩石点位，实地路标上还接连出现河马岩、蘑菇岩、拇指岩、兔子岩等一个个有名有姓的岩石——奇岩之多可见一斑。溪谷方面，最具代表性的是果川一侧的文原瀑布、二段瀑布溪谷，以及首尔大学一侧的溪谷。

关于登山的感受

登山口附近是一段平缓的橡树林小道，但过了半山腰之后，岩石路段的比重便骤然增大。登上棱线，首尔市区和汉江尽收眼底，往南还能远眺清溪山和果川市区，视野极其开阔——光是公共数据里登记在册的观景点就有9处之多，一路上视野频频打开。需要手脚并用攀爬的岩石路段和绳索路段交替出现，所以就距离而言，体感强度算是相当高的。春天有冠岳山公园入口的樱花和杜鹃，秋天的红叶与岩棱相映成趣，冬天岩石上覆着白雪的雪景也很美，只是结冰的风险会大大增加。

  • 登山口交通：一般从地铁 2 号线落星垡站或 4 号线舍堂站乘社区小巴或步行前往仁宪洞方向。登山口位于住宅区，没有专用停车场，建议乘坐公共交通。

路线 2 —— 教授会馆入口 → 山顶

📏 7.71km ⛰️ 567m 爬升 ⏱️ 2:52 ➡️ 单程 难度 ★★★★☆ ⤓ GPX
  • 途经：首尔大学 → 公共卫生间 → 岔路口 → 桌椅休息区 → 平台、长椅 → 平台、长椅 → 桌椅休息区、平台 → 凉亭 → 简易卫生间 → 观景台 → 位置指示牌 蘑菇岩 → 位置指示牌 斗笠僧群 → 观景点 → 右转 → 位置指示牌、冠岳门（上）
  • 登山口交通：最常见的走法是从2号线首尔大入口站换乘开往首尔大学方向的公交（5511、5513路等），直接坐进校园深处。首尔大学校园内停车收费，而且经常很挤。
  • 特点与推荐人群：前半段是缓坡林间路，长椅和卫生间一应俱全，走到蘑菇岩、冠岳门一带则变成攀岩式的山路，节奏张弛分明。设施多，适合想走走歇歇、慢慢爬上去的人。

路线 3 —— 登山口 → 八峰顶

📏 8.92km ⛰️ 579m 爬升 ⏱️ 3:12 🔁 环线 难度 ★★★★★ ⤓ GPX
  • 途经：登山口 → 普光寺 → 岔路口 → 黄杨木 → 场岩方向休息点 → 二段瀑布 → 文原瀑布 → 白山路 → 八峰方向 → 六峰方向 → 八峰顶 → 汇合点附近 → 恋主台方向主棱线 → KBS 基站 → 十字路口前
  • 登山口交通：一般从 4 号线政府果川厅舍站或果川站往文原洞方向走过去，也可以利用果川一侧的公共停车场。
  • 特点与推荐人群：穿过二段瀑布、文原瀑布的溪谷，再依次翻越八座岩峰，是冠岳山最具代表性的岩棱路线。不少路段需要手脚并用，适合想体验刺激攀岩式登山的老手；新手或雨天最好避开这条路线。

路线 4 — 郊游林洗手间 → 鹤岩国旗峰

📏 7.43km ⛰️ 563m 爬升 ⏱️ 2:48 ➡️ 单程 难度 ★★★★☆ ⤓ GPX
  • 途经：郊游林洗手间 → 岔路口 → 休息处 → 位置指示牌 → 山泉水取水点 → 观景台 → 绳索路段 → 造型漂亮的岩石 → 鹤岩国旗峰 → 直升机停机坪 → 兔子岩 → 沉默之脸岩 → 三张桌椅 → 下山口 → 公交车站
  • 入山口交通：一般从 2 号线新林站或首尔大入口站乘坐开往首尔大学方向的公交车前往，下山口正好连着公交车站，公共交通衔接很方便。
  • 特色与推荐人群：这条路线先在山泉水取水点补充饮水，再穿过绳索路段，最后迎来鹤岩国旗峰上飘扬的太极旗和开阔的视野。适合不执着于顶峰恋主台、更想以清静的山脊峰头为目标的人，也适合喜欢一路寻访各种有名号的奇岩、乐在其中的人。

路线 5 —— 落星垈站 → 紫云庵九技峰

📏 6.78km ⛰️ 579m 爬升 ⏱️ 2:40 ➡️ 单程 难度 ★★★★☆ ⤓ GPX
  • 途经: 落星垈站 → 岔路口3 → 休息处 → 岔路口11 → 岔路口12 → 岔路口13 → 岔路口14 → 拇指岩 → 紫云庵九技峰 → 松鼠岩 → 紫云庵棱线路 → 拳头岩 → 岔路口20 → 用餐休息处 → 岔路口22
  • 登山口交通: 从2号线落星垈站（Nakseongdae Station）出来直接步行开走，是地铁接驳最直观的一条路线。车站周边不好停车，建议坐地铁来。
  • 特点·适合人群: 这条紫云庵棱线路线出站就直接上山，拇指岩、拳头岩等花岗岩石板上的展望是全程精华。岔路口特别多，务必随时打开地图 App 确认位置。想来一场短而带劲的岩石山行的首都圈上班族，走这条正合适。

路线 6 —— 入口餐馆 → 山顶

  • 登山口交通：一般从4号线果川站步行前往果川乡校（Gwacheon Hyanggyo）方向即可，也可以利用果川市区的公共停车场。
  • 特色·推荐人群：小卖部、厕所、泉水处、凉亭一路依次相连，是配套设施最密集的传统热门路线。全程6.37公里，也是表中最短的一条，最适合首次登冠岳山的朋友和外国游客，值得优先推荐。

从数据上看，最高难度只是”中等”，但主要路面以岩石为主，实际走起来的体感难度很可能比数字要高。尤其是八峰棱线、紫云庵棱线以及恋主台前的最后一段台阶，冬天结冰时不带冰爪非常危险；下雨天花岗岩坡面湿滑，事故频发。公共数据中登记了 1 处危险区域，经过相应标识附近时请走绕行路线。设施方面，登记在册的有 9 处观景点（鸟岩、场院岩、龟岩、监视塔·T.P基准石、掩体·直升机坪、监视哨所、带休息区的观景点等）、7 处饮水点（含泉眼）、1 处避难所（该处还设有山火宣传板、广播设备和森林解说牌）以及 1 处停车场。

距离与所需时间

冠岳山是一座当天往返绰绰有余的山。以代表性路线计算，单程约2小时30分至3小时40分，往返一般预留5~7小时即可。数据显示，山内全部路段的累计耗时为上行779分钟、下行542分钟，但这是多个路段加总的结果，千万别误以为是单条路线的所需时间。不过由此算出的平均上行配速约18分钟/公里，倒是制定个人计划时一个不错的参考基准。只有打算把全长43.69公里分成几条山脊纵走时，才需要考虑住上一晚以上（山内有1处避难所）；一般的登顶往返，一天足矣。

前往方式（公共交通·停车）

  • 地铁：2号线落星垈站、首尔大入口站、新林站（首尔方向），4号线舍堂站、果川站、政府果川厅舍站（果川方向）是主要门户。从首尔大入口站可换乘5511、5513路等公交，抵达首尔大学校园一侧的登山口。
  • 公交：各登山口附近的站点都有市内公交和社区小巴经过，不过据数据显示，登山道旁并没有登记在册的公交站，所以还是建议以从地铁站步行前往为基本方案。
  • 停车：根据公共数据，路线范围内的停车场只有1个。果川乡校、首尔大学方向虽有收费停车场，但周末非常拥挤，强烈建议乘坐公共交通。
  • 实际路线和公交车次，最准确的办法还是在 KakaoMap 或 Naver Map 上输入出发地查询。

季节·天气·装备

  • 推荐季节：春季（45月）和秋季（1011月）最合适。春天可以看冠岳山公园入口的樱花，秋天岩棱线上的红叶很美。
  • 夏季：花岗岩晒得滚烫，没有遮阴的石板坡路段又多，建议带足水（1.5L 以上），尽量一大早出发。
  • 冬季：岩石结冰严重，冰爪（链式防滑钉）和登山杖基本是必备品，另外记得准备防风衣物。
  • 通用装备：抓地力好的登山鞋（不建议穿运动鞋——路面多为岩石）、手套（攀岩路段用）、充足的水、零食、充电宝。下雨天最好避开岩棱路线，改走溪谷一侧的平缓山路更安全。

外国游客小贴士

  • 主要地名罗马字：冠岳山 Gwanaksan、山顶恋主台 Yeonjudae、落星垈站 Nakseongdae Station、舍堂站 Sadang Station、果川站 Gwacheon Station、首尔大学 Seoul National University(SNU)、八峰棱线 Palbong Ridge。
  • 许可与门票：进山无需许可，也不收门票。完全免费，不用预约。不过要遵守夜间登山限制，以及封闭的小路（有官方标识）不要走。
  • 卫生间与饮用水：登山口和前半段有多处公共卫生间和简易卫生间，路线上还登记有 7 处饮水点（泉眼）。但棱线上没有任何设施，请在山下提前把水灌满。果川乡校路线上还有一家小卖部（冠岳山庄）。
  • 地图应用：Google Maps 对韩国的登山路线和步行路径很不准确。请务必安装 KakaoMap 或 Naver Map（支持英语）。这座山岔路很多，建议养成随时确认 GPS 的习惯。
  • 安全：记住登山路沿途的位置标识牌（多用途位置编号），紧急情况下拨打 119 报警时能派上用场。路线上部分观景点旁设有紧急联络方式的标牌。岩石路段请走指定的绳索和台阶区间，雨天或结冰时请避开岩棱路线。
