骑布朗普顿T Line走世界杯大桥西侧，畅骑76公里首尔心形路线

骑行 · 发布日期 2024/5/11 ·

从上岩洞进入汉江大致有两种走法。第一种是沿着佛光川（불광천）前进汇入汉江，大多数人都走这条路，我以前也常这么走。不过现在改用世界杯大桥进入汉江，一路非常畅快，再也没有施工路段了。

我以 MBC 为起点来说明。

看到世界杯体育场后右转，朝世界杯大桥方向一直走。接着会遇到一个没有红绿灯的人行横道，过了它就成功进入世界杯大桥了。世界杯大桥的东侧要过桥得等两个红绿灯，但西侧只需在北端过一个没有红绿灯的横道，稍后再过一个有红绿灯的横道，到了南端就不必再过横道，可以直接搭电梯进入汉江自行车道。

Previous image Next image

我从朴正熙图书馆那边沿着公园道路一直骑到世界杯体育场。这张是在世界杯大桥上眺望一山方向的江边北路。就这样一直过桥就行。像昨天这样刮大风的日子，过桥可真不容易。请务必放慢速度、盯紧路面通过。

Previous image Next image

从世界杯大桥继续往前，会遇到通往安养、江边北路（蚕室方向）、内部循环路的路标。在那里按按钮再过一次人行横道，之后一直到汉江南侧自行车道都不会再有红绿灯。第二张是在横道上眺望上岩洞的样子，第三张是完全过桥后回望上岩洞的样子。

Previous image Next image

终于到了安养川汇流口。虽然是用 iPhone 拍的，但天气好，对比度相当漂亮。这里就是大家常约见面的地方。^^ 今天要骑的是心形路线。这大概会成为我买了 Brompton T Line 之后骑过的最长距离。喝口水就出发。

路线是沿着自行车道朝汉江东侧一直骑。遇到良才川就在那里拐进良才川继续前行，良才川自行车道到头就是果川。在果川短暂驶出，上普通公路骑一段。在仁德院站十字路口拐向鹤义川方向走一段普通的社区小路，接上鹤义川一路前行，到双溪川（쌍개울）就与安养川汇合了。从这里开始就是安养川，一口气骑下去，便又回到出发点——安养川汇流口。

良才川到鹤义川这一段有 GTX 和各种土木工程，周边还在盖更多公寓，环境不算舒适，但大约三十分钟就能通过，别贪快，人行道和公路适当穿插着骑，也不会太吃力。

Previous image Next image

过了瑞草区就是果川洞。逆风太猛，稍作休息。同行的伙伴在前面一路领骑帮我破风，才勉强能前进。他明明体力很好，可风大的时候，时速偶尔也会掉到 20 公里以下。换作是我，恐怕会掉到 15 公里以下。远处能看到冠岳山，我们正绕着冠岳山下方兜过去。

Previous image Next image

这是第一次来的双溪川休息区。两条川在这里汇成一条，给骑行的人打造了一处很棒的歇脚地。虽说风大，但天气实在太好了。

Previous image Next image

平时我戴的是 Garmin Enduro 2。Garmin 很清楚我的体力极限，毕竟已经戴了一年多……我一边骑，一边盼着实际体力能比它标出来的多留一点。可那个数值比想象中要准。当它显示剩余不到 1% 时，我每骑 30 分钟就休息 5 分钟。虽然不必太迷信设备，但有个指标参考总归不坏。就这样走走停停，又回到了安养川汇流口。Garmin Varia 715 的电量虽然剩不到 50%，但撑过了 5、6 个小时的骑行。我没特别调模式就骑了，行车记录应该也存下来了。白天也走了公路，所以开着闪烁模式，并设成有车辆靠近时灯光会强烈闪动。

要从这里回上岩洞得过世界杯大桥。按平常的走法，搭电梯上去多少就得再下来多少……结果又得重新爬升。但现在按出发时那样从西侧上去，就不会白白损失已经爬到的高度，反而能带点小下坡回到家。

Previous image Next image

平安归来。上岩洞整个区的自行车道，做得比全国任何地方都好。今天总骑行距离 76 公里，GPS 轨迹正好呈心形。这样我就第一次骑完了心形路线。屁股有点痛，果川那段是普通公路，还有不少施工路段，稍微有点憋闷，但等施工结束、把良才川和鹤义川两段用自行车道连起来变得畅快之后，估计会成为很受欢迎的中距离路线，吸引不少车友来骑。

谢谢大家。
