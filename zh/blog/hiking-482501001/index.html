概览

照片：韩国观光公社观光照片库（京畿道杨州市）· 公共传源

道峰山（Dobongsan，道峰山）横跨首尔特别市道峰区、京畿道议政府市与杨州市，与北汉山一同构成北汉山国立公园的两大主体之一。它的最高峰通常被认为是紫云峰，海拔约740米上下；而真正有探访路径相连、登山者可以攀登的顶峰一带的神仙台高755米，顶峰区域的最高海拔则达到759米。整座山以突出的花岗岩石峰构成典型的”骨山”山势，在首尔近郊的名山中，岩脊之美尤为突出。根据辖区内登山道的空间信息，最高难度仅为”轻松”，短距离路段即便是新手也能轻松走完；以神仙台、女性峰、五峰等顶峰一带为目标的大多数线路，一日往返足矣。

山势与地形

照片：韩国观光公社观光图片馆（京畿道杨州市）· 公共누리

道峰山是典型的岩山地形，花岗岩石峰与山脊密密相连、绵延不绝。代表性的山峰包括位于顶峰一带的神仙台（755米，别称道峰山神仙台）、与之相邻的656峰（660米）、炮台山脊上的炮台山火瞭望哨（666米）、西侧山脊的女性峰峰顶（505米），以及堪称道峰山象征的奇岩群五峰（671米）。下山方向还分布着双子瞭望台（152米）这类相对低矮的观景点。仙人峰、万丈峰等近乎垂直的巨大花岗岩石柱在山谷各处拔地而起，因此这里也以攀岩胜地闻名；山脊沿线更接连着庭院岩、脸庞岩等一座座有名的岩石。谷地（元道峰溪谷、道峰溪谷、松楸溪谷等）坡度平缓，夏天常有戏水的游客光顾，但越靠近山脊，陡峭的岩盘地带便反复出现，山势的起伏格外分明。

登山的感受

照片：韩国观光公社观光照片图库（京畿道杨州市）· 公共누리

入山之初，多是沿着阔叶林与溪谷平缓延伸；越接近山脊，花岗岩板岩和岩石台阶接连出现，体感难度骤然上升——这正是道峰山特有的节奏。以仙人峰、万丈峰为背景的观景点和拍照点散布在沿途各处，为徒步增添乐趣；一旦登临峰顶一带，首尔市中心乃至北汉山的山脊尽收眼底，那开阔的视野宛如给你的奖赏般铺展开来。春天，溪谷入口处的新绿与杜鹃格外动人；秋天，与岩脊相映的红叶尤为醒目；冬天，背阴的岩石路段仍残留着结冰。同一条路线，随季节不同，体感难度差别很大。

路线 1 — 岔路口 → 道峰山神仙台

📏 3.75km ⛰️ 629m 爬升 ⏱️ 1:59 ➡️ 单程 难度 ★★★☆☆

途经：岔路口 → 休息区 → 神仙台观景处 → 拍照点 → 观景点 → 道峰山神仙台 → 仙人安全休息区 → 落石危险区 → 注意野猪出没 → 马当岩休息区 → 马当岩 → 简易休息区 → 简易厕所

登山口交通：普遍认为，从道峰山站（地铁1号线）方向的道峰山探访支援中心一侧可以通往登山口。公共交通比较便利，因此被视为道峰山的代表性路线。

特点与推荐人群：一路上神仙台观景处和拍照点接连分布，攀登途中处处都是取景的好地方，而临近山顶的马当岩路段视野豁然开朗。不过沿途设有落石危险和注意野猪出没的警示牌，单独登山时务必遵守安全守则。这条路线适合第一次来道峰山的游客，也适合想用较短时间领略山顶一带风光的人。

路线 2 — 黄米糕岩 → Y溪谷顶

📏 5.13km ⛰️ 646m 爬升 ⏱️ 2:22 ➡️ 单程 难度 ★★★★☆

途经：黄米糕岩 → 岔路口 → 仙人峰观景点 → 以仙人峰·万丈峰为背景的拍照点 → 满月庵 2 → 满月庵 1 → 满月庵山神阁 → 仙万子拍照点 → 台阶休息处 → 坠落危险区 → Y溪谷顶 → 冬季常年结冰路段 → 石窟庵 → 青泉 → 三岔路安全休息处

登山口进入方式：据说是从道峰山站附近朝满月庵方向进入的路线，是一段香客与登山者共用的路段。

特点·推荐人群：这条路线能近距离眺望巍峨的仙人峰与万丈峰岩壁，是岩脊景观最为震撼的一条。不过 Y溪谷路段标有坠落危险警示，也注明了冬季常年结冰路段，遇雨或结冰时需要绕行，比较适合有岩脊登山经验的人。

线路 3 — 道峰山大圆寺 → 元道峰1号安全休息站

📏 9.3km ⛰️ 669m 爬升 ⏱️ 3:26 ➡️ 单程 难度 ★★★★★

途经：道峰山大圆寺 → 北汉山国立公园元道峰事务所志愿者服务中心 → 药山庭园 → 元道峰第3停车场 → 元道峰卫生间 → 元道峰第1停车场 → 元道峰休息站 → 岔路口 → 元道峰第2停车场 → 双龙寺 → 地藏庵下方卫生间 → 元孝寺 → 妙法莲华经 → 内村泉 → 内村安全休息站

入山口交通：从地铁1号线望月寺站、回龙站附近的元道峰方向进入。从元道峰第1到第3停车场依次相连来看，这一段的自驾可达性也相对不错。

特点与推荐人群：大圆寺、双龙寺、元孝寺等寺庙接连分布，前半段坡度平缓，可以一边沿溪散步一边逛寺庙。全程9.3公里，是几条线路中最长的一条，卫生间和休息站也布置得很密集，适合想留出充裕时间、慢慢走走看看的人。

路线 4 — 寺庙专用停车场 → 山顶

📏 5.86km ⛰️ 648m 爬升 ⏱️ 2:33 ➡️ 单程 难度 ★★★★☆

途经：寺庙专用停车场 → 岔路口 → 仙人峰拍照点 → 仙人峰观景休息处 → 天竺寺下方休息处 → 天竺寺瀑布 → 天竺寺一柱门休息处 → 九节草·黄波斯菊花丛 → 天竺寺泉眼 → 天竺寺 → 场岩泉眼 → 急救箱休息处 → 原木休息处 → 观音瀑布 → 脸岩

登山口交通：这条路线从道峰山站方向、天竺寺进山路上的寺庙专用停车场出发。据说是既有不少人徒步接近、也有人利用小规模停车位的一段路。

特色·推荐人群：这条路线一路上有天竺寺瀑布、一柱门，还有九节草、波斯菊花丛等寺院内四季分明的看点，相当丰富；沿途还标注了两处泉眼（天竺寺泉眼、场岩泉眼），补充饮用水比较方便。适合想同时饱览寺院风光与岩脊景致的人。

路线 5 — 登山口 → 五峰顶眺望

📏 12.65km ⛰️ 580m 爬升 ⏱️ 4:08 🔁 环线 难度 ★★★★★

途经：静谧的溪谷 → 登山路与溪谷 → 壮观的岩石 → 小支流 → 松楸瀑布 → 松楸瀑布中段 → 松楸瀑布上段 → 休息区 → 溪谷眺望 → 岩石休息区 → 天然休息区 → 岔路口 → 五峰顶眺望 → 五峰眺望 → 注意坠落

入口接近方式：这条路线从松楸溪谷方向的入口出发。相比道峰山站，一般更常见的做法是搭乘开往松楸方向的市内公交前往。

特点·推荐人群：全程 12.65 公里，是 8 条路线中最长的一条。前半段沿着松楸瀑布延伸，是一路溪谷相伴的路，夏天听着水声散步格外惬意。正面眺望五峰的地点是全程的亮点，不过后半段有需要注意坠落的路段，因此更适合体力充沛、习惯纵走的登山者。

线路 6 — 王盲鳗 → 山顶

📏 7.68km ⛰️ 611m 爬升 ⏱️ 2:56 ➡️ 单程 难度 ★★★★☆

途经：王盲鳗 → 山谷大麦饭 → 湖院派出所猫头鹰哨所 → 保护树 → 银杏树山泉 → 回龙探访支援中心 → 回龙探访支援中心洗手间 → 路边停车场 → 体能锻炼场休息区 → 岔路口 → 回龙泉 → 回龙瀑布 → 回龙寺3 → 回龙寺2 → 回龙寺1

入山口交通：从地铁1号线回龙站附近的湖院洞方向进入，回龙探访支援中心配有洗手间和路边停车场，是初段配套设施相对齐全的一段。

特点·推荐人群：这条线路标注了银杏树山泉、回龙泉、回龙瀑布等多处与水相关的景点，比其他线路更有溪谷的意趣，沿途还会经过回龙寺一带的三座寺院，带着几分文化气息。从回龙站可直接抵达，对于主要依靠公共交通登山的人来说很方便。

난이도·노면·주의

사진: 한국관광공사 관광사진 갤러리 (경기도 양주시) · 공공누리

根据辖区登山道空间信息，登记在册的路段共有两处。井着溪谷小蓄水池—大监面事务所旁路段（3.08km，难度轻松，路面为岩石／砂土）标注有”陡坡及岩石地带”的危险路段警示；墨坊里路段（0.63km，难度轻松，路面为岩石／砂土）则没有另外的危险标识。两段路面都由岩石和砂土构成，即便在天气干燥时，走到岩盘路段也要小心打滑，而下雨天或落叶堆积的秋季，实际的湿滑感会更加明显。从辖区登山道数据来看，单独登记的饮水点、避难所、观景点、危险区域各为0处，起点和终点则确认有3处。不过在实测的 GPX 路线上，落石危险、注意野猪出没、坠落危险、冬季常年结冰路段（Y溪谷）都以地点名的形式标注了出来，因此经过这些路段时务必查看指示牌，尤其是冬季结冰路段，切勿不穿冰爪就贸然进入。

距离与耗时

照片：韩国观光公社观光照片画廊（京畿道杨州市）· 公共누리

辖区内登山道的总长度（把定着溪谷小蓄水池—大甘面事务所旁路段和默方里路段加在一起）为 3.71km，这只是各路段相加的总量，并不是单条线路的距离。辖区全线的累计耗时统计为上行 67 分钟、下行 46 分钟，换算成距离后，平均上行配速约为 18 分钟/km。这个配速只能作为估算各条线路预计耗时的参考数值，切不可把上行 67 分钟、下行 46 分钟本身误当成某一条线路的耗时。以实测 GPX 为准，登顶线路的单程耗时差别很大，短的约 2 小时上下（神仙台、女性峰），长的可达 4 小时以上（五峰顶眺望、大源寺线路），往返或纵走时甚至可能需要 6~8 小时以上。不过无论哪条线路，按往返来算都足够当天来回，加之这里没有单独设避难所，所以最好把行程安排成在太阳落山前完成下山，这样更安全。

交通指南（公共交通·停车）

照片：韩国观光公社观光照片库（京畿道杨州市）· 公共누리

道峰山有地铁1号线的道峰山站、望月寺站、回龙站环绕山脚而过，从首尔市区乘公共交通前往十分方便。从道峰山站可步行进入神仙台、天竺寺、Y溪谷方向的路线；从望月寺站、回龙站则可分别进入元道峰、回龙寺方向的路线。松楸方向（观赏五峰顶的路线）需另乘市内公交，出发前请先确认。区域内登山道的空间信息中没有单独登记的公交站或停车场，因此比起自驾，优先考虑地铁与公交的组合更为划算；若确实要开车，建议提前在 KakaoMap（카카오맵）或 Naver Map（네이버지도）上结合实际路线，查看各条路线是否有元道峰第1~3停车场或寺庙专用停车场之类的小型停车空间。这两款 App 都提供公共交通换乘信息和实时登山道指引，不仅方便国内用户，对外国游客也很实用。

照片：韩国观光公社观光照片库（首尔特别市道峰区道峰洞）· 公共누리

道峰山的罗马字写作 Dobongsan，山顶一带的神仙台标为 Sinseondae，五峰为 Obong，女性峰为 Yeoseongbong。道峰山隶属于北汉山国立公园，无需另外申请入山许可或购买门票，可以自由探访。沿途设有多处洗手间，包括简易洗手间、元道峰洗手间、地藏庵下方洗手间、回龙探访支援中心洗手间等；不过按照园区登山道数据来看，山中并没有正式登记的饮水点，因此与其指望银杏药水泉、天竺寺泉眼、庭石泉、回龙泉这类天然泉水，不如提前备好至少 1～1.5 升以上的饮用水，更为稳妥。地图应用方面，谷歌地图往往无法准确显示韩国国内的登山道信息，建议安装 KakaoMap 或 Naver Map，以确认真实路线和公共交通信息。安全方面，在设有 Y 溪谷、坠落危险、落石危险等警示牌的岩脊路段，遇到下雨或结冰时请绕行；同时最好记住，遇到紧急情况可拨打韩国的综合救援报警电话（119）。
